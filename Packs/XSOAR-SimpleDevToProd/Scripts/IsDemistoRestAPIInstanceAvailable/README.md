Checks if the provided Demisto REST API instance is available for the XSOAR Simple Dev to Prod workflow.

## Script Data
---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | Condition, infra |

## Inputs
---

| **Argument Name** | **Description** |
| --- | --- |
| instanceName | The name of the Demisto REST API instance to check. |

## Outputs
---

| **Path** | **Description** | **Type** |
| --- | --- | --- |
| yes | The instance is available. | Unknown |
| no | The instance is not available. | Unknown |
