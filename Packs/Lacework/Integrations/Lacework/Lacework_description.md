# How to generate a Lacework API key
1. Log in to your Lacework account: https://***account***.lacework.net 
2. Once logged in, click on the ***Settings*** button.
3. Select ***API Keys*** from the left-hand menu, then click the ***Create New*** button.
4. Give your API Key a name and description, then click ***Save***.
5. Download your API Key and API Secret for use in the XSOAR integration.
