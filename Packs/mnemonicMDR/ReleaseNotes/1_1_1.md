
#### Integrations
##### mnemonic MDR - Argus Managed Defence 
- Added 1 command
  - ***argus-add-attachment*** 
  Upload a War Room entry file to an Argus Case. File should be an archive and max 50 MB. 
- Added 2 integration parameters
  - *close_argus_case* configure if XSOAR incident is closed, close the mirrored Argus Case as well.
  - *close_incident* configure if Argus Case is closed, close the mirrored XSOAR Incident as well.
