
#### Integrations

##### Strata Logging Service XSOAR Connector

- Added a new authentication method that uses a *Client Secret* together with the *Registration ID* and *Encryption Key*.
- Deprecated the *Authentication Token* authentication method. Existing instances continue to work until the token expires, after which they should be reconfigured using the *Client Secret*.
- Updated the Docker image to: *demisto/python_pancloud_v2:1.0.0.10133006*.
