Magnet Automate is a workflow orchestration tool by Magnet Forensics designed to run forensic tools automatically. It enables digital forensic teams to manage cases, submit evidence sources for processing, and orchestrate forensic workflows across distributed processing nodes.

## What does this pack do?

This content pack enables you to:
- Create, list, cancel, and delete forensic cases with support for organization-defined custom fields.
- Start workflow runs to process evidence sources and associate them with existing cases.
- List, cancel, and delete workflow runs, and monitor their processing status.
- Start merge workflow runs to consolidate multiple evidence runs into a single analysis.
- List, export, and delete forensic workflows available in Magnet Automate.
- Create, list, update, and delete processing nodes used to execute forensic workflows.
- Retrieve organization-defined custom fields to enrich cases and evidence with structured metadata.
