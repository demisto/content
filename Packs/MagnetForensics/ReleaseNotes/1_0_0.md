
#### Integrations

##### Magnet Automate

New: Magnet Automate integration for Cortex XSOAR.

Magnet Automate is a workflow orchestration platform by Magnet Forensics that automates digital forensic investigations. This integration enables managing cases, workflows, workflow runs, and processing nodes through the Magnet Automate REST API.

The following commands are available:

- ***ma-forensics-custom-fields-list*** - Retrieves a list of custom fields for cases and evidence sources.
- ***ma-forensic-case-create*** - Creates a new case in Magnet Automate.
- ***ma-forensics-cases-list*** - Retrieves a list of all cases or information about a specific case.
- ***ma-forensics-case-delete*** - Deletes a specific case from Magnet Automate.
- ***ma-forensics-case-cancel*** - Cancels an ongoing case in Magnet Automate.
- ***ma-forensics-workflow-list*** - Retrieves a list of all available workflows.
- ***ma-forensics-workflow-get*** - Retrieves detailed information and export data for a specific workflow.
- ***ma-forensics-workflow-delete*** - Deletes a specific workflow from Magnet Automate.
- ***ma-forensics-merge-workflow-run-start*** - Starts a merge workflow run for multiple existing workflow runs.
- ***ma-forensics-workflow-run-start*** - Starts a new workflow run and associates it with a specific case.
- ***ma-forensics-workflow-run-list*** - Retrieves a list of all workflow runs for a specific case or details of a specific run.
- ***ma-forensics-workflow-run-delete*** - Deletes a specific workflow run from a case.
- ***ma-forensics-workflow-run-cancel*** - Cancels a specific workflow run.
- ***ma-forensics-node-create*** - Creates a new node (agent) in Magnet Automate.
- ***ma-forensics-nodes-list*** - Retrieves a list of all available nodes (agents).
- ***ma-forensics-node-update*** - Updates the configuration of an existing node.
- ***ma-forensics-node-delete*** - Deletes a specific node from Magnet Automate.
