Armis is an agentless, enterprise-class security platform to address the threat landscape of unmanned and IoT devices. We discover every managed, unmanned, and IoT device on and off of your network, analyze device behavior to identify risks or attacks, and protect your critical business information and systems. This Pack contains Armis incident fields and provides the ability to interface with the Armis system

##### What does this pack do?

Fetch and digest different types of events from an Armis instance into an XSIAM tenant using the Armis Event Collector.
