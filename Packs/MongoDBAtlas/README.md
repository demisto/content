MongoDB Atlas is a fully managed cloud database service that simplifies deploying, managing, and scaling MongoDB clusters across cloud providers.

MongoDB Atlas provides advanced database security features, automated backups, performance monitoring, and detailed analysis for optimizing data management.
It also integrates with popular tools and frameworks, offering real-time data insights and improving application performance.

# What does this pack do?
This pack enables users to fetch events and alerts from MongoDB Atlas into Cortex XSIAM.
