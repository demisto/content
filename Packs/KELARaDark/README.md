This content pack was created to allow KELA's customers an integration with RaDark.

#### What does this pack do?
* Fetches incidents from RaDark into your Cortex XSOAR.
* Supports enrichment for existing data from RaDark.
* Manages incidents in RaDark from your Cortex XSOAR.

The content pack consists of several content items including mappers, classifiers, incident types, fields and layouts and provides a playbook for RaDark items. These create a complete pack that supports a full integration with KELA's RaDark platform.
