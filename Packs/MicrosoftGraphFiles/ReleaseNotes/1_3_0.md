
#### Integrations

##### O365 File Management (Onedrive/Sharepoint/Teams)

- Added the ***msgraph-driveitem-update*** command. Use this command to rename a driveItem, move it within or across drives, or update its metadata.
- Added the ***msgraph-driveitem-copy*** command. Use this command to copy a driveItem within or across drives. A monitor URL is returned so the operation can be tracked to completion.
- Added the ***msgraph-driveitem-permissions-list*** command. Use this command to list the sharing permissions on a driveItem, including permissions granted to internal users, external guest users, groups, and sharing links.
- Added the ***msgraph-driveitem-permission-delete*** command. Use this command to revoke a single sharing permission from a driveItem.
