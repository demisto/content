
#### Integrations

##### O365 File Management (Onedrive/Sharepoint/Teams)

- Added the ***msgraph-list-sensitivity-labels*** command to retrieve sensitivity labels available in the tenant from Microsoft Information Protection.
- Added the ***msgraph-extract-sensitivity-label*** command to retrieve the sensitivity label currently assigned to a drive item.
- Added the ***msgraph-apply-sensitivity-label*** command to assign a sensitivity label to a drive item.
