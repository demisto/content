#### Integrations

##### Generic Export Indicators Service

- Fixed an issue where the EDL output contained fewer entries than expected when the list included wildcard Domain indicators (e.g., `*.example.com`). Each wildcard indicator writes two lines to an intermediate file (the bare domain and the wildcard form), causing the raw-line limit check to truncate the list before all unique indicators were processed. The limit is now enforced against unique output entries rather than raw intermediate lines.
