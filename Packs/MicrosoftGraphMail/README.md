Microsoft Graph Mail gives you authorized access to a user's Outlook mail data enabling you to perform cybersecurity investigations, leading to rapid incident responses and threat protections against ransomware and spearfishing.

## What does this pack do?
Using the commands in this integration, you can:

- Access the properties of emails. 
- Delete an email or an email folder.
- Access the attachments of an email.
- Access mail folder lists.
- Create a new mail folders.
- Update the properties a folder.
- Retrieve an email message
- Create draft messages.
