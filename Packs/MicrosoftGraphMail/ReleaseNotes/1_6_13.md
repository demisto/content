
#### Integrations

##### Microsoft Graph Mail Single User

- Added a Use Legacy Name configuration parameter in order to use the original attachment name.
- Updated the attachment name to be in the following format: <the attachment ID\>-attachmentName-<the attachment name\> only if the attachment is inline and **Use Legacy Name** is disabled.

##### O365 Outlook Mail (Using Graph API)

- Added a Use Legacy Name configuration parameter in order to use the original attachment name.
- Updated the attachment name to be in the following format: <the attachment ID\>-attachmentName-<the attachment name\> only if the attachment is inline and **Use Legacy Name** is disabled.
