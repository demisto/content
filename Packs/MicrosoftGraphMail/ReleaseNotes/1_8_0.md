#### Integrations

##### O365 Outlook Mail (Using Graph API)

- Added the ***msgraph-mail-create-rule*** command to create a new inbox rule for a user's mailbox.
- Added the ***msgraph-mail-update-rule*** command to update an existing inbox rule for a user's mailbox.
- Added the ***msgraph-mail-get-settings*** command to retrieve a user's mailbox settings (timezone, language, automatic replies, working hours, etc.).
- Added the ***msgraph-mail-get-mailtips*** command to retrieve mail tips (out-of-office, mailbox full status, delivery restrictions, etc.) for a recipient email address.
