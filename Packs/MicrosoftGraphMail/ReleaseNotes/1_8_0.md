
#### Integrations

##### Microsoft Graph Mail

- Added the ***msgraph-mail-create-rule*** command, which creates a new mailbox rule in the user's Inbox folder.
- Added the ***msgraph-mail-update-rule*** command, which updates an existing mailbox rule by ID.
- Added the ***msgraph-mail-get-settings*** command, which retrieves the user's mailbox settings.
- Added the ***msgraph-mail-get-mailtips*** command, which retrieves Mail Tips for the specified email address.
