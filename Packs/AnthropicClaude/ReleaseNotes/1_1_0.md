
#### Integrations

##### Anthropic Claude

- Added event collection from the Anthropic Compliance API **Activity Feed** into Cortex XSIAM (vendor: *anthropic*, product: *claude*).
- Added the ***claude-get-events*** command to manually retrieve Activity Feed events.
- Added the following read-only Compliance API commands: ***claude-list-organizations***, ***claude-list-organization-users***, ***claude-list-roles***, ***claude-list-role-permissions***, ***claude-list-groups***, ***claude-list-group-members***, ***claude-list-chats***, ***claude-list-chat-messages***, ***claude-list-projects***, ***claude-list-project-attachments***, and ***claude-get-project-document***.
- Added the *Compliance Access Key* configuration parameter, used for event collection and the Compliance API commands.
- Added the *Organization UUID* configuration parameter, used as the default organization for compliance commands when their *org_uuid* argument is not provided.
- Updated the *API Key* configuration parameter to be optional, so the integration can be configured with only the *Compliance Access Key*.
- Updated the *Activity types* parameter to accept a comma-separated list of Activity Feed types.
- Updated event collection to default to collecting the last minute of activity on the first fetch and to continue from the last collected event on subsequent fetches.
- Updated the pack to be supported by Cortex and available on the Cortex XSIAM and Cortex Platform marketplaces.
