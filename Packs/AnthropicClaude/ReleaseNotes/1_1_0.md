
#### Integrations

##### Anthropic Claude

- Added event collection from the Anthropic Compliance API **Activity Feed** into Cortex XSIAM (vendor: *anthropic*, product: *claude*).
- Added the ***claude-get-events*** command to manually retrieve Activity Feed events.
- Added the following read-only Compliance API commands: ***claude-list-organizations***, ***claude-list-organization-users***, ***claude-list-roles***, ***claude-list-role-permissions***, ***claude-list-groups***, ***claude-list-group-members***, ***claude-list-chats***, ***claude-list-chat-messages***, ***claude-list-projects***, ***claude-list-project-attachments***, and ***claude-get-project-document***.
- Added a **Compliance Access Key** configuration parameter, used for event collection and the Compliance API commands.
- Added an **Organization UUID** configuration parameter, used as the default organization for compliance commands when their *org_uuid* argument is not provided.
- The **API Key** configuration parameter is no longer required, so the integration can be configured with only the Compliance Access Key.
- The **Activity types** parameter now accepts a comma-separated list of Activity Feed types.
- Event collection now defaults to collecting the last minute of activity on the first fetch and continues from the last collected event.
- The pack is now supported by Cortex and is available on the Cortex XSIAM and Cortex Platform marketplaces.
