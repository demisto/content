#### Integrations

##### Okta v2

- Added the revoke_sessions argument to **okta-expire-password**. When true, this will revoke the user's existing sessions.
- Added the hide_password argument to **okta-expire-password**. When true, this will prevent the password from being saved in the War Room.
- Updated the **okta-expire-password** Marketplace documentation to include the two new arguments.
