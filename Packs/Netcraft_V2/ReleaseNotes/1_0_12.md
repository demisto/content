
#### Integrations

##### Netcraft

- Fixed an issue where the **netcraft-takedown-note-list** command returned only a partial list of notes for a given takedown ID. The command now paginates through all API results to ensure all notes are retrieved.
- Added the **limit** argument to the **netcraft-takedown-note-list** command to control the maximum number of notes returned when **all_results** is set to *false*. Default is 50.
