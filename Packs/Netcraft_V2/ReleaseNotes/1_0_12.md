
#### Integrations

##### Netcraft

- Added support for **Advanced: Time in minutes to look back when fetching incidents** parameter that use this parameter to determine how long backward in time to look in the search for incidents that were created before the last run time and did not match the query when they were created.
- Added support for **netcraft-get-incidents** command that retrieve incidents from netcraft using the same api request as fetchincidents, filtered by id.
- Updated the Docker image to: *demisto/python3:3.12.13.10404775*.
