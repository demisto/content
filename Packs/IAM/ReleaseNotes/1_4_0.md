#### Playbooks

##### IAM - Hire User
- Hire events from Workday will only create users in Active Directory and Okta, instead of creating them in all IAM integrations.

##### IAM - Rehire User
- Rehire events from Workday will only enable users in Active Directory and Okta, instead of enabling them in all IAM integrations.

##### IAM - Update User
- Update events from Workday will only update users in Active Directory and Okta, instead of updating them in all IAM integrations.

##### IAM - Termminate User
- Termination events from Workday will only disable users in Active Directory and Okta, instead of disabling them in all IAM integrations.
