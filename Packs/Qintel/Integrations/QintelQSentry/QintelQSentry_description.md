## QintelQSentry Help

To configure an instance of this integration in Cortex XSOAR, you will need to supply your API token. Refer to the [Integrations Getting Started Page](https://docs.qintel.com/integrations/overview) for details on how to retrieve your token.

Specify the following to configure this instance:

1. Enter your **token** for **Qintel Token**
2. (Optional) Enter a custom QSentry API URL
