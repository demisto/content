
#### Integrations

##### Absolute

- Added support for the ***absolute-wipe-request-list*** command that get information about the wipe requests.
- Added support for the ***absolute-wipe-actions-list*** command that get the status of actions by devices or by request.
- Added support for the ***absolute-wipe-request-create*** command that create a wipe request for the devices.
- Added support for the ***absolute-wipe-request-cancel*** command that attempts to cancel all wipe actions with an action status of pending for the request specified by the *request_uid* path parameter.
