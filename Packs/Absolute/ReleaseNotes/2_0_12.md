
#### Integrations

##### Absolute

- Added support for **absolute-wipe-request-list** command that get information about the wipe requests.
- Added support for **absolute-wipe-actions-list** command that get the status of actions by devices or by request.
- Added support for **absolute-wipe-request-create** command that create a wipe request for the devices.
- Added support for **absolute-wipe-request-cancel** command that attempt to cancel all wipe actions with an actionstatus of pending for the request specified by the requestuid path parameter.
- Updated the Docker image to: *demisto/auth-utils:1.0.0.3639076*.
