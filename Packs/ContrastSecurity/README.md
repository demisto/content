The Contrast Security Pack helps security teams receive and respond to application security incidents and issues detected by Contrast Security. It enables real-time ingestion of both security incidents and vulnerability issues into Cortex XSOAR via webhooks, allowing teams to take quick actions such as blocking IP addresses, updating ADR (Application Detection and Response) policies, managing issue status, and investigating attack observations directly from XSOAR.

##### What does this pack do?

* Receive Contrast Security Incidents and Issues into XSOAR in real time via webhook.
* List and retrieve Contrast Security Issues with filtering and sorting capabilities.
* List Incident Observations with cursor-based pagination for efficient data retrieval.
* Retrieve detailed information about specific Observations.
* Add comments to Contrast Security Incidents and Issues.
* Update the status of Contrast Security Incidents and Issues.
* Block IP addresses associated with a Contrast Security Incident.
* Configure ADR (Application Detection and Response) policy rules per environment (Development, QA, Production).
* Support bi-directional incident mirroring to keep XSOAR and Contrast Security in sync.
* Support outgoing issue mirroring from XSOAR to Contrast Security.
