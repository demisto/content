
#### Integrations

##### Chronicle

- Added the **gcb-reference-list-append-content** command that appends lines into an existing reference list.
- Added the **gcb-reference-list-remove-content** command that removes lines from an existing reference list.
- Added the *entry_id* argument to the **gcb-create-reference-list** and **gcb-update-reference-list** commands to allow users to create or update reference lists from a file.
- Added the *use_delimiter_for_file* argument to the **gcb-create-reference-list** and **gcb-update-reference-list** commands to determine whether the file content is split by a delimiter or by new lines.
- Updated the Docker image to: *demisto/googleapi-python3:1.0.0.4114363*.
