
#### Integrations

##### Google SecOps Streaming API

- Added support for **Default Severity of Incident** parameter to select the default severity for incident when detection does not specify a severity. Available options are 'Critical', 'High', 'Medium', 'Low', 'Informational' and 'Unspecified' \(If not selected, defaults to Unspecified\).
- Fixed an issue when the detection doesn't specify any severity.
