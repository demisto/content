
#### Integrations

##### Github Feed

- Added support for the *Incremental Feed* integration parameters and *When removed from the feed* expiration policy.
- Updated the **fetch-indicators** logic. When the feed is configured as non-incremental, each fetch retrieves all new and modified files since the configured *First fetch time*, allowing indicators removed from the source repository to be expired.
- Updated the Docker image to: *demisto/taxii2:1.0.0.9117276*.
