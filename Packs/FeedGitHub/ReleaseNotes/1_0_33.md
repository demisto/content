
#### Integrations

##### Github Feed

- Added support for the *feedIncremental*, *feedExpirationPolicy* and *feedExpirationInterval* integration parameters.
- Updated the fetch-indicators logic so that when the feed is configured as non-incremental, each fetch retrieves all new and modified files since the configured *First fetch time*, allowing indicators removed from the source repository to be expired.
- Updated the Docker image to: *demisto/taxii2:1.0.0.9117276*.
