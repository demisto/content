
#### Integrations

##### Google Drive

- Added the *drive_id* and the *corpora* arguments to the ***google-drive-files-list*** command.
- Added the *supports_all_drives* argument to the ***google-drive-file-get*** command.
- Added the ***google-drive-drive-delete*** command to delete shared drive from Google Drive.
- Deprecated the ***google-drive-create*** command. Use the ***google-drive-drive-create*** command instead.
- Updated the Docker image to: *demisto/googleapi-python3:1.0.0.73655*.
