Prerequisites for OAuth 2.0 Authentication

* Create an API Credential such as Client ID and Client Secret. More information [here](https://docs.cribl.io/cribl-as-code/api-auth/#api-auth-cloud)  
* In the Organization Permissions drop-down menu, select a Permission to apply for the Credential tokens. Define the desired Permissions. To use the API Credential to grant read-only access on individual Cribl Search resources select either the User or Editor Permission on Cribl Search  
* Enter the received Client ID and Client Secret into the appropriate integration parameter fields.
