Cribl Search is a search solution that enables querying, retrieving, and managing search jobs, datasets, and saved searches across your Cribl Cloud deployment.

## What does this pack do?

This content pack enables you to:

- Run synchronous search queries against datasets and retrieve results inline.
- Create long-running search jobs to process larger workloads asynchronously.
- List, monitor the status of, update, and delete search jobs.
- Retrieve the events produced by completed search jobs, with pagination support.
- List the datasets available for searching across your Cribl Cloud deployment.
- List saved searches defined in your Cribl Cloud deployment.
