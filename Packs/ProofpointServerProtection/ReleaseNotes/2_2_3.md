#### Integrations

##### Proofpoint Protection Server v2

- Updated the ***proofpoint-pps-quarantine-messages-list*** command to support querying multiple quarantine folders in a single execution. When a comma-separated list or JSON array of folder names is provided via the `folder_name` argument, each folder is now queried sequentially and results are aggregated into a single response. This prevents intermittent "timeout while waiting for engine to answer" errors that occurred when the XSOAR engine attempted to fan out multi-folder queries as parallel sub-tasks.
