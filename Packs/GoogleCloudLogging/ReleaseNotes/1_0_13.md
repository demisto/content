
#### Modeling Rules

##### New: Google Cloud Logging Modeling Rule

<~XSIAM> 
Added an XDM mapping for the following GCP logs (Available from Cortex XSIAM 2.3):
- Audit Logs:
  - Admin Activity audit logs.
  - Data Access audit logs.
  - System Event audit logs.
  - Policy Denied audit logs.
- DNS Queries.  
</~XSIAM>

#### Parsing Rules

##### New: Google Cloud Logging Parsing Rule

<~XSIAM> 
Added an assignment of the ingested Audit and DNS logs timestamp field to the *_time* field (Available from Cortex XSIAM 2.3).
</~XSIAM>
