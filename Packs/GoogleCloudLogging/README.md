Google Cloud Logging is a service provided by Google Cloud Platform (GCP) that allows you to store, search, analyze, monitor, and alert on log data and events from your applications and services running on GCP.
The Google Cloud Logging Cortex XSOAR pack helps users to centralize all their logs in a single location, making it easier to troubleshoot issues and gain insights from their data.
## What does this pack do
This integration enables you to:

- Retrieve log entries that originated from a project/folder/organization/billing account.
