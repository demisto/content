#### Incident Types

##### New: Splunk Investigation

- Added a new **Splunk Investigation** incident type to model investigations fetched from the Splunk Mission Control v2 API.

#### Incident Fields

- Added new SplunkPy incident fields for the **Splunk Investigation** type: Investigation GUID, Investigation ID, Investigation Name, Incident Origin, Investigation Type, Incident IDs, Excluded Finding IDs, Consolidated Findings, Implicit Finding IDs, Intermediate Finding IDs, Disposition Name, Event Type, Risk Object.
- Extended the `associatedTypes` of existing SplunkPy incident fields (Splunk Urgency, Splunk Status, Splunk Disposition, Splunk Sensitivity, Splunk Notes) to include the new **Splunk Investigation** type.

#### Layouts

##### New: Splunk Investigation

- Added a new **Splunk Investigation** layout with tabs: Incident Info, Investigation Summary, Findings (including Consolidated Findings), Network/User, and Raw Data.
