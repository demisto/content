## Splunk

#### Integrations

##### SplunkPy v2

- Fixed an issue where drill down searches resolved a field to the wrong value when the field name was a substring of another field (for example, `src_ip` resolving to the value of `src`).
- Fixed an issue where backslashes in drill down search field values were not escaped, causing Splunk to mangle values such as Windows paths and return no results.
