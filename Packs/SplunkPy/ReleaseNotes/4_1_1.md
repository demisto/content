
#### Integrations

##### SplunkPy v2


- Added the ***splunk-investigation-create*** command, which creates a new investigation in Splunk Enterprise Security.
- Added the ***splunk-investigation-list*** command, which lists investigations from Splunk Enterprise Security with support for filtering by IDs, status, owner, urgency, sensitivity, disposition, and create/update time ranges.
- Updated the Docker image to: *demisto/splunksdk-py3:1.0.0.8936013*.
