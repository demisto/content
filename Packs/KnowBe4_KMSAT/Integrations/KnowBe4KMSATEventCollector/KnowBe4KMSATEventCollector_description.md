KnowBe4 KMSAT allows you to push and pull your external data to and from the KnowBe4 console.

## Create the API Key 
1. Go to your Knowbe4 account settings.
2. Click the API Key tab.
3. Create an API Key.
4. In the Name field, enter a unique name for your API key.
5. Click the +Create API Key button.
