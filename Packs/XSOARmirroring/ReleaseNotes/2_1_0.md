
#### Integrations

##### XSOAR Mirroring

- Added the **get-modified-remote-data** command, which implements incremental mirroring by fetching only the incident IDs modified since the last sync. Note: This feature is supported only for Cortex XSOAR version 8.14.0 and later.
- Updated the Docker image to: *demisto/python3:3.12.13.7444307*.
