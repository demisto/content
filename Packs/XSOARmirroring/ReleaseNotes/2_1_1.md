
#### Integrations

##### XSOAR Mirroring

- Improved the **get-modified-remote-data** command to avoid repeated failures when the remote XSOAR machine does not support the `/public/v1/incidents/modified` endpoint. When a 303 or 404 response is received, the integration now stores a daily flag in the integration context and skips the endpoint for the rest of that UTC day, retrying automatically the next day (or immediately if the remote machine is upgraded and the call succeeds).
