
#### Integrations
##### Prisma Cloud (RedLock)
- Added command ***redlock-list-scans*** which returns a list of IaC scans that meet the given conditions.
- Added command ***redlock-get-scan-status*** which returns the status of the asynchronous IaC scan job that has the specified scan ID.
- Added command ***redlock-get-scan-results*** which returns scan result details for the completed scan that has the specified scan ID.
- Updated the Docker image to: *demisto/python3:3.9.7.24076*.
