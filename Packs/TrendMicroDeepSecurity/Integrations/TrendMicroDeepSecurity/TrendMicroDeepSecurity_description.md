## TrendMicro Deep Security Help

### Server URL
The URL of the TrendMicro console. 
The default value of the server URL is the cloud TrendMicro console: *https://app.deepsecurity.trendmicro.com*.

## API Secret
The secret token that is used for accessing the API inside Cortex XSOAR.

To create a new API secret: 
1. Go to the Administration tab inside the web console.
2. Click the **API Keys** tab, and click **New**. A popup window appears.
3. Fill all properties inside that window, and click **Next**.
The API secret value will appear. Copy it into the API secret parameter.

**Note**: You will not be able to view the API secret key after closing the popup window!
