This transformer convert the Epoch or UTC timestamp to desired stamp

## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | transformer, date |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| value | Time in UTC Format, epoch Format |
| format | format of the time to convert\(Ex:"%Y-%m-%d%H:%M:%S"\) |
| utcoffset | timeoffset to be converted\(i.e. \+2:30,\+5:30\) |
| timezone | Timezone to be converted \(i.e 'Asia/Kolkata','Pacific/Auckland','Australia/Adelaide'\).Get the timezone list https://timezonedb.com/time-zones |

## Outputs

---
There are no outputs for this script.
