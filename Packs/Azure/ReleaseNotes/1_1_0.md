
#### Integrations

##### Azure

- Added the following commands:
  - **azure-nsg-resource-group-list** command that lists all resource groups for a subscription.
  - **azure-nsg-subscriptions-list** command that lists all subscriptions for a tenant.
  - **azure-nsg-network-interfaces-list** command that lists network interfaces in a resource group.
  - **azure-nsg-public-ip-addresses-list** command that lists public ip addresses in a resource group.
  - **azure-nsg-security-rule-get** command that gets a specific rule.
  - **azure-nsg-security-rule-delete** command that deletes a security rule.
  - **azure-nsg-security-rule-create** command that creates a security rule.
  - **azure-nsg-security-groups-list** command that lists all network security groups.
  - **azure-storage-blob-service-properties-get** command that gets a blob service properties.
  - **azure-storage-blob-containers-update** command that updates a blob container.
  - **azure-vm-instance-power-off** command that Powers off a given virtual machine.
  - **azure-vm-instance-start** command that powers on a given virtual machine.
  - **azure-vm-instance-details-get** command that gets the properties of a given virtual machine.
  - **azure-vm-network-interface-details-get** command that gets the properties of a given network interface.
  - **azure-vm-public-ip-details-get** command that gets the properties of a given public IP address.
  - **azure-billing-forecast-list** command that returns the cost forecast for a subscription over a given time range.
  - **azure-billing-usage-list** command that returns actual usage and cost details for a given time period, optionally filtered by service name.
  - **azure-billing-budgets-list** command that lists configured budgets at the subscription or resource group scope.

- Added support for the following quick actions:
  - **Disable Storage Account Cross Tenant Replication**, which disables cross-tenant object replication for the storage account, preventing data from being copied to other Azure AD tenants.
  - **Enable Azure App Service Auth**, which enables App Service Authentication for the web app.
  - **Set Webapp HTTP Version to 2.0**, which enables http 2.0 for the web app.
  - **Disable Storage Account Public Access**, which disables public access to blobs for the storage account, blocking anonymous/public read access to containers and blobs.
  - **Allow Access by Trusted Services**, which configures the storage account's network rules to allow traffic from trusted azure services.
  - **Disable Remote Debugging on Azure App**, which disables remote debugging for the web app to reduce the risk of unauthorized code access.
  - **Assign Managed Identity to Web App**, which assigns a system-assigned managed identity to the web app, enabling secure access to other azure services without managing credentials.
  - **Revoke NSG Rule to Stop Traffic**, which updates an NSG rule's access settings.
  - **Set Function App HTTP Version to 2.0**, which enables HTTP/2.0 for the Function App.
  - **Delete NSG Rule**, which deletes an NSG rule.
  - **Set Disk Data Access to AD**, which sets the managed diskâ€™s data access authentication mode to Azure Active Directory.
  - **Assign Managed Identity to Function App**, which assigns a system-assigned managed identity to the Function App. This allows the app to securely access other Azure services without the need to manage sensitive credentials, reducing the risk of exposure.
  - **Disable Public Access and Exports on ACR**, which disables public network access and content exports for the Azure Container Registry (ACR) to enhance security.
  - **Disable ACR Authentication as ARM**, which disables the legacy 'Authentication as ARM' feature for the Azure Container Registry (ACR).
  - **Disable ACR Anonymous Pull**, which disables anonymous pull access for the Azure Container Registry (ACR), requiring clients to authenticate before pulling images.
  - **Enable Soft Delete on Blob**, which enables blob soft delete for the storage account, retaining deleted blobs for 30 days.
  - **Disable Public & Private Access to VM Disk**, which disables all public and private network access for the managed disk.
