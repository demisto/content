#### Integrations

##### Azure
- Added the following new commands:
  - ***azure-storage-blob-service-property-get***
  - ***azure-storage-blob-service-property-set***
- Deprecated the following matching commands:
  - ***azure-storage-blob-service-properties-get***
  - ***azure-storage-blob-service-properties-set***
- Added the quick action ***Enable Blob Soft Deletion***, and deprecated the matching quick action ***Enable Soft Delete on Blob***.

- Updated the context output path of the command ***azure-storage-container-property-get*** to start with the prefix *Azure.Storage.Container*.
- Updated the context output path of the command ***azure-monitor-log-profile-update*** to start with the prefix *Azure.Monitor.LogProfiles*.
- Updated the context output path of the quick action ***Update Azure Monitor Log Retention Period*** to start with the prefix *Azure.Monitor.LogProfiles*.
- Updated the context output path of the command ***azure-policy-assignment-create*** to start with the prefix *Azure.Policy.PolicyAssignments*.
- Updated the context output path of the quick action ***Apply MS Defender Policy*** to start with the prefix *Azure.Policy.PolicyAssignments*.
- Updated the context output path of the command ***azure-storage-account-update*** to start with the prefix *Azure.Storage.StorageAccounts*.
- Updated the context output path of the quick action ***Allow Access by Trusted Services*** to start with the prefix *Azure.Storage.StorageAccounts*.
- Updated the context output path of the quick action ***Disable Storage Account Cross Tenant Replication*** to start with the prefix *Azure.Storage.StorageAccounts*.
- Updated the context output path of the quick action ***Disable Storage Account Public Access*** to start with the prefix *Azure.Storage.StorageAccounts*.
- Updated the context output path of the quick action ***Disable Network Access to Storage Account*** to start with the prefix *Azure.Storage.StorageAccounts*.
- Updated the context output path of the quick action ***Set Storage Account to HTTPS only*** to start with the prefix *Azure.Storage.StorageAccounts*.
