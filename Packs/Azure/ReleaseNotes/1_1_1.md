
#### Integrations

##### Azure

- Added support for **azure-storage-container-blob-tag-set** command that updates the tags for the specified blob. the command can replace the entire tags of the blob and can be used to remove tags.
- Added support for **azure-storage-container-create** command that create a new container under the specified account.
- Added support for **azure-storage-container-blob-tag-get** command that retrieve the tags of the specified blob.
- Added support for **azure-storage-container-property-get** command that retrieve properties for the specified container.
- Added support for **azure-storage-container-blob-property-get** command that retrieve blob properties.
- Added support for **azure-storage-container-public-access-block** command that block public access to a container.
- Added support for **azure-storage-container-delete** command that marks the specified container for deletion. the container and any blobs contained within it, will be deleted during garbage collection.
- Added support for **azure-storage-container-blob-get** command that retrieve blob from container.
- Added support for **azure-storage-container-blob-create** command that retrieve blob from container.
- Added support for **azure-storage-container-blob-property-set** command that set blob properties.
