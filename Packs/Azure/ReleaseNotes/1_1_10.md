#### Integrations

##### Azure

- Added support for the ***azure-vn-security-group-create*** command that creates or updates a network security group.
- Added support for the ***azure-vn-network-interface-update*** command that updates a network interface.
- Added support for the ***azure-compute-vm-list*** command that lists all the virtual machines in the specified subscription.
- Added support for the following quick actions:
  - **Update Threat Policy for SQL Database**, which enables email notifications to account administrators for SQL database threat detection alerts.
  - **Enable TDE on SQL**, which enables Transparent Data Encryption (TDE) for the SQL database to encrypt data at rest.
  - **Enable KeyVault Purge & Soft-delete features**, which enables soft delete and purge protection for the Key Vault to prevent the permanent loss of keys and secrets.
  - **Set App Service to HTTPS only**, which enforces HTTPS-only traffic for the WebApp.
  - **Disable Key Based Metadata Write on CosmosDB**, which disables key-based write access for Cosmos DB metadata to enhance security.
  - **Enable Client Certs on Azure App**, which enables client certificate authentication for the Web App to secure incoming requests.
