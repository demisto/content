
#### Integrations

##### Azure

- Updated the **Test** button to validate the connection for the Client Credentials, Authorization Code, and Azure Managed Identities authentication types. For the Authorization Code flow, provide the *Authorization code* parameter before clicking **Test**.  
    The **Test** button is not supported for the Device Code authentication type. In this case, validate the connection by running the ***azure-auth-test*** command after executing the ***azure-auth-start*** and ***azure-auth-complete*** commands.

- For the Device Code authentication type, validate the connection by running the *azure-auth-test* command after completing the *azure-auth-start* and *azure-auth-complete* steps. The **Test** button is not supported for this flow.
- Documentation and metadata improvements.
