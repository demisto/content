
#### Integrations

##### Azure

- The **Test** button now validates the connection for all authentication types. For the Device Code and Authorization Code flows, complete the authentication first (Device Code: run the *azure-auth-start* and *azure-auth-complete* commands; Authorization Code: provide the Authorization code parameter), then click **Test** to verify the connection. Relevant only for Cortex XSOAR and Cortex XSIAM.
- Documentation and metadata improvements.
