
#### Integrations

##### Azure

- Added support for running the integration on Cortex XSOAR and Cortex XSIAM, with four authentication types: Client Credentials, Device Code, Authorization Code, and Azure Managed Identities.
- The **Test** button now validates the connection for the Client Credentials, Authorization Code, and Azure Managed Identities authentication types. For the Authorization Code flow, provide the Authorization code parameter before clicking **Test**.
- For the Azure Managed Identities authentication type, the **Test** button validates the connection using a lightweight resource groups request.
- For the Device Code authentication type, validate the connection by running the *azure-auth-test* command after completing the *azure-auth-start* and *azure-auth-complete* steps. The **Test** button is not supported for this flow.
- Added validation of the required parameters per authentication type before running a command.
- Documentation and metadata improvements.
