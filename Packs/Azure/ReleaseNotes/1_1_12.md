#### Integrations

##### Azure
- Added the ***azure-sqldb-security-alert-policy-update*** command. Use this command instead of ***azure-sql-db-threat-policy-update***, which will be deprecated.
- Added the ***azure-keyvault-vault-update*** command. Use this command instead of ***azure-key-vault-update***, which will be deprecated.
- Added the ***azure-appservice-webapp-update*** command. Use this command instead of ***azure-webapp-update***, which will be deprecated.
- Added the ***azure-cosmosdb-db-account-update*** command. Use this command instead of ***azure-cosmos-db-update***, which will be deprecated.
- Updated the context path of **Assign Managed Identity to Web App** to *Azure.AppService.WebApp*. 
- Deprecated the **Assign Managed Identity to Function App** quick action.
