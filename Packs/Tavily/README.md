Tavily is an AI-powered search and content extraction service designed for developers and LLM-based applications. It offers two main APIs:

Search API for retrieving relevant, real-time web results with concise summaries.

Extract API for cleanly extracting structured content from web pages, with options for advanced depth and image inclusion.

Tavily is ideal for tools like AI agents, research assistants, and chatbots that need reliable access to current web content.
