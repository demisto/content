The Shadow IT Pack provides additional capabilities for handling Shadow IT incidents in Cortex XSOAR. This includes a new **Shadow IT** Incident Type and a Playbook.

##### What does this pack do?
- Provides an additional **Shadow IT** Incident Type
- Provides a **Shadow IT** Indicator Layout
- Provides a playbook to Handle Shadow IT incidents (**Handle Shadow IT Incident**)
