## Test Connection After inputting the **Base URL** and **API Key**, click **Test** to validate the connection.

## Configuring the **Base URL**
In most parts of the world the default `base_url` should suffice.
Different base URIs can be taken from [here](https://developer.cisco.com/meraki/api-v1/getting-started/#base-uri).

## Obtaining your Meraki API key
In order to interact with the Dashboard API, you must first obtain an API key.

- Open your Meraki dashboard: https://dashboard.meraki.com.
- Once logged in, navigate to the **Organization** > **Settings** page.
- Ensure that the API Access is set to “Enable access to the Cisco Meraki Dashboard API”.
- Go to your profile by clicking on your account email address (on the upper-right) > **My profile** to generate the API key.
