## ProofpointEmailSecurityEventCollector

### Authentication

#### Cluster ID

The Cluster ID is displayed in the upper-right corner of the management interface next to the release number.

#### API Key

Proofpoint Email Security uses API keys for authentication. To create an API key:

1. Log in to your Proofpoint Email Security console and navigate to **Settings** > **API Key Management**.
2. Click the `Create New` button to create a new API key.
3. Copy the API key and paste it into the API Key field in the integration configuration.
