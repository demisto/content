
#### Integrations

##### Have I Been Pwned? v2

- Added support for ***pwned-breach-get*** command that gets a single breached site by breach name.
- Added support for ***pwned-subscribed-domains-list*** command that gets the list of subscribed domains.
- Added support for ***pwned-breaches-for-domain-list*** command that gets all breached email addresses for a domain.
- Added support for ***pwned-latest-breach-get*** command that gets the most recently added breach.

- Updated the Docker image to: *demisto/python3:3.12.13.8160132*.
