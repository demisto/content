The MISP content pack enables you to access MISP's threat sharing platform toshare information about cybersecurity indicators and threats. 

## What does this pack do?
- Searches for events in MISP.
- Checks the reputation a file or IP address.
- Checks if a  URL is in MISP events.
- Creates a new MISP event.
- Adds an attribute to an existing MISP event.
- Downloads/uploads a file sample from/to MISP.
- Deletes an event according to the event ID.
- Adds information to MISP events.
