
#### Integrations

##### Exabeam Security Operations Platform

- Deleted the **Fetch events** parameter.
- Added support for **exabeam-platform-create-case-note** command that add a new note to the specified case.
- Added support for **exabeam-update-case-details** command that update details for a specific case.
- Added support for **exabeam-platform-list-case-notes** command that retrieve a list of notes associated with the specified caseid.
- Added support for **exabeam-get-threat-summary** command that retrieve the copilot threat summary for an individual alert.
