This is the Hello World event collector integration for Cortex XSIAM.
This integration was integrated and tested with version xx of SymantecCloudWorkloadProtectionEventCollector.

## Configure SymantecCloudWorkloadProtectionEventCollector on Cortex XSOAR

1. Navigate to **Settings** > **Integrations** > **Servers & Services**.
2. Search for SymantecCloudWorkloadProtectionEventCollector.
3. Click **Add instance** to create and configure a new integration instance.

    | **Parameter** | **Required** |
    | --- | --- |
    | Server URL | True |
    | Client ID | True |
    | Client Secret | True |
    | Customer ID | True |
    | Domain ID | True |
    | Fetch alerts with status (ACTIVE, CLOSED) | False |
    | Max number of events per fetch | False |
    | Trust any certificate (not secure) | False |
    | Use system proxy settings | False |

4. Click **Test** to validate the URLs, token, and connection.
