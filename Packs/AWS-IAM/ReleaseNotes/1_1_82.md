#### Integrations

##### AWS - Identity and Access Management

- Added support for the ***Deactivate User IAM Access Key*** quick action that immediately disables a specific IAM access key.
- Added support for the ***Block All User Actions*** quick action that attaches an inline "deny all" policy for the specified IAM user.
- Added support for the ***Remove User Console Access*** quick action that permanently deletes the IAM user's password (login profile).
