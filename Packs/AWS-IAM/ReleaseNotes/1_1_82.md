
#### Integrations

##### AWS - Identity and Access Management

- Added support for ***Deactivate User IAM Access Key*** Quick Action, that instantly disables a specific IAM access key.
- Added support for ***Block All User Actions*** Quick Action, that attaches an inline "deny all" policy for the specified IAM user.
- Added support for ***Remove User Console Access*** Quick Action, that permanently deletes the IAM user's password (login profile).
