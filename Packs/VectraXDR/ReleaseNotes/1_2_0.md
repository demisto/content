
#### Integrations

##### Vectra XDR

- Added support for **vectra-detection-tag-list** command to returns a list of tags for a specified detection.
- Added support for **vectra-detections-mark-asopen** command to open detections with provided detection IDs in the argument.
- Added support for **vectra-detection-tag-remove** command to remove tags from the detection.
- Added support for **vectra-entity-detections-mark-asclosed** command to mark the detections of the entity as closed with the provided entity ID in the argument.
- Added support for **vectra-detection-tag-add** command to add tags to a detection.
- Added support for **vectra-detections-mark-asclosed** command to mark detections as close with provided detection IDs in the argument.
- Updated the Docker image to: *demisto/python3:3.12.12.5490952*.
