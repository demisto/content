
#### Integrations

##### Vectra XDR

- Added support for **vectra-detections-note-remove** command that remove a note from the detection.
- Added support for **vectra-detections-note-list** command that returns a list of notes for a specified detection.
- Added support for **vectra-detections-note-update** command that update a note in the detection.
- Added support for **vectra-detections-note-add** command that add a note to the detection.
- Added support for **vectra-entity-reset-fetch** command that reset the given entity to refetch incidents.
