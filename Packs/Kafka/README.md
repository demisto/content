Kafka is an open source distributed streaming platform. Use the Kafka integration to manage messages and partitions and to fetch Kafka messages to create incidents in Cortex XSOAR.
## What does this pack do? 
- Fetches Kafka messages to create Cortex XSOAR incidents.
- Publishes messages to Kafka.
- Prints all topics and their partitions.
- Consumes Kafka messages.
- Fetches partitions for a topic.
