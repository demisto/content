
#### Integrations

##### JAMF v2

- Migrated deprecated Classic API Computer Inventory endpoints to the Pro API.
- Deprecated the ***jamf-get-computer-peripherals-subset*** command.
- Deprecated the ***jamf-get-computer-iphones-subset*** command.
- Added support for the ***jamf-get-computer-subset*** command, which returns the requested section for a specific computer.
- **Breaking Change**: Classic Authentication is no longer supported by the Computer Inventory endpoints.
- **Breaking Change**: Updated the **jamf-get-computer-by-match** command to use the *filter* argument instead of the *match* argument.
- **Breaking Change**: The following commands now use the Jamf Pro API, which results in different context paths:
  - ***jamf-get-computers***
  - ***jamf-get-computers-basic-subset***
  - ***jamf-get-computer-by-id***
  - ***jamf-get-computer-by-match***
  - ***jamf-get-computer-general-subset***
  - ***jamf-get-computer-location-subset***
  - ***jamf-get-computer-purchasing-subset***
  - ***jamf-get-computer-hardware-subset***
  - ***jamf-get-computer-certificates-subset***
  - ***jamf-get-computer-security-subset***
  - ***jamf-get-computer-software-subset***
  - ***jamf-get-computer-extension-attributes-subset***
  - ***jamf-get-computer-groups-accounts-subset***
  - ***jamf-get-computer-configuration-profiles-subset***
