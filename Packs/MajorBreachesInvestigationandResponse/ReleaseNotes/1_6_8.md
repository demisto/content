##### New: CVE-2021-1675 - PrintNightmare
- CVE-2021-1675 is a vulnerability dubbed “PrintNightmare” which allows remote code execution on Windows Print Spooler. Microsoft patched the vulnerability in June but an exploit POC and complete technical analysis was made publicly available online. 
This playbook includes the following tasks:
- Manual actions to mitigate the exploit
- Search Vulnerable Devices using the CVE
- Search Vulnerable Devices using the SIEM
** Note: This is a beta playbook, which lets you implement and test pre-release software. Since the playbook is beta, it might contain bugs.
