Example Field Change trigger script which can be added to the DBot Status field, and will trigger if an Incident is reopened.

## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | field-change-triggered, training |
| Cortex XSOAR Version | 6.8.0 |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| old | The old value of the changed field |
| new | The new value of the changed field |

## Outputs

---
There are no outputs for this script.
