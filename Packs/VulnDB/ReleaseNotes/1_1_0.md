
#### Integrations

##### VulnDB

- Added the ***vulndb-generic-api-request*** command to allow you to run any API request against the VulnDB API by specifying route, request type, and body.
- Added the ***vulndb-get-cpe-by-vuln-id*** command to allow you to get the full CPE list by vuln_id.
- Added the ***vulndb-get-vuln-report-by-vuln-id*** to get the PDF vulnerability report from VulnDB and return it to the war room.
- Improved the implementation of several existing commands to return more details about vulnerabilities including timeline information and technical descriptions if available.
