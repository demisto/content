
#### Integrations

##### BMC Helix ITSM

- Updated the Docker image to: *demisto/python3:3.12.13.7444307*.
- Added the "Related to" value to *association_type* argument of the *bmc-itsm-ticket-create-relationship* command.
- Added support for the *Client ID* parameter. Required when 'Use Oauth' is selected.
- Added support for the *Authorization Code* parameter, which is obtained by running the ***bmc-itsm-generate-login-url*** command and following the URL. Required when 'Use Oauth' is selected.
- Added support for the *Redirect URI* parameter. Required when 'Use Oauth' is selected.
- Added support for the *OpenID Issuer URL* parameter to set the RSSO server URL. Required when 'Use Oauth' is selected.
- Added support for the *Use OAuth* parameter to use an Oauth 2.0 authorization code for authentication instead of username/password.
- Added support for the ***bmc-itsm-add-attachment*** command that adds attachments to an existing BMC Helix ITSM ticket.
- Added support for the  ***bmc-itsm-generate-login-url*** command that generates the login URL used for the Oauth 2.0 authorization code.
