#### Playbooks
##### New: Download Close File - Cybereason
- This playbook aborts a file download operation which is in progress based on the Malop ID and username provided. (Available from Cortex XSOAR 6.5.0).
##### New: Download File - Cybereason
- This playbook starts the file download operation and finally give an option to download the file if possible based on the Malop ID and username provided. (Available from Cortex XSOAR 6.5.0).
##### Isolate Endpoint - Cybereason
- This playbook isolates an endpoint based on the hostname provided. (Available from Cortex XSOAR 5.0.0).
##### Unisolate Endpoint - Cybereason
- This playbook unisolates endpoints according to the hostname that is provided by the playbook input. (Available from Cortex XSOAR 5.5.0).
##### Block File - Cybereason
- This playbook accepts an MD5 hash and blocks the file using the Cybereason integration. (Available from Cortex XSOAR 5.0.0).
##### Search Endpoints By Hash - Cybereason
- Hunt for endpoint activity involving hash, using Cybereason. (Available from Cortex XSOAR 5.0.0).
