
#### Integrations

##### Threatmon

- Added the **Request Takedown** command (*threatmon_request_takedown*) to submit takedown requests for eligible Threatmon findings such as Phishing Domain, Rogue Mobile App, and Fake Social Media Account detections.
- Added a **Fetch Interval (minutes)** configuration parameter to control how often the integration polls ThreatMon for new incidents. The interval is fixed at up to 100 pages per run to prevent timeouts.
