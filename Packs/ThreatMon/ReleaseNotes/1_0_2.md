
#### Integrations

##### Threatmon

- Added support for the ***threatmon_request_takedown*** command that submits takedown requests for eligible Threatmon findings such as Phishing Domain, Rogue Mobile App, and Fake Social Media Account detections.

- Added support for the **Fetch Interval (minutes)** configuration parameter to control how often the integration polls ThreatMon for new incidents. The interval is fixed at up to 100 pages per run to prevent timeouts.

- Updated the Docker image to: *demisto/python3:3.12.13.7444307*.
