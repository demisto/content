# Creating an API Key

Navigate to the Settings Page: Access the Settings page from the top navigation bar.Open the API Access Tab: In the Settings page, select the API Access tab then Click "Create New API Key": Click the Create new API key button. Access Your API Key: Within a few seconds, a new API key will appear in the table. Click the Copy button next to the key to copy it securely.
