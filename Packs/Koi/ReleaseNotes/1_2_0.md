
#### Integrations

##### KOI

- Added support for the Cortex XSOAR marketplace, and hid the event collection (Collect) configuration parameters when running on Cortex XSOAR.
- Updated the ***koi-get-events*** command so that the *should_push_events* argument gracefully degrades on non-Cortex XSIAM platforms: events are returned without being pushed instead of failing.
- Updated the Docker image to: *demisto/fastapi:0.125.0.9094740*.
