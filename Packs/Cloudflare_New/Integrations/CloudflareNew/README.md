## Commands

You can execute these commands from the CLI, as part of an automation, or in a playbook.
After you successfully execute a command, a DBot message appears in the War Room with the command details.

### get-firewall-events

***
Retrieve HTTP firewall events of a specific zone in Cloudflare

#### Base Command

`get-firewall-events`

#### Input

| **Argument Name** | **Description** | **Required** |
| --- | --- | --- |
| source_ip | The source IP to fetch firewall logs for. | Required | 
| start_time | The start time bound in ISO8601 date and time format (Example: 2025-03-15T08:01:09Z). | Required | 
| end_time | The end time bound in ISO8601 date and time format (Example: 2025-03-17T08:01:09Z). | Required | 
| zone_ids | Enter the website ID(s) of the associated website. Possible values are: 6eea44e08204729c6a8e86e708b1aedd. | Optional | 
| ZoneID | No description provided. | Optional | 

#### Context Output

There is no context output for this command.
