## Lansweeper

- Register for an account on [Lansweeper](https://app.lansweeper.com/).

#### Steps to generate Application Identity Code
- Log in to the Lansweeper account.
- Once logged in, all available sites for your account are shown.
- Please select the sites for which you need to fetch the asset details.
- Go to "Developers tools" by clicking the user's profile logo.
- Click on "All applications".
- Navigate to "Add new application", select "Personal Application" and enter the "Application name".
- Click on "Add Application".
- Choose an application that has been created. 
- Click "Authorize" button that will redirect to the permissions screen.
- Select the sites you wish to grant permission to, and click "Allow".
- The granting permission procedure will redirect to the Application identity code.
