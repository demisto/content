
#### Integrations

##### National Vulnerability Database Feed v2

- Fixed an issue where the CVSS Severity Filter was not working because the API parameter name used incorrect casing (`cvssv4severity` instead of `cvssV4Severity`). The filter now correctly applies across all CVSS versions (V2, V3, and V4).
- Added the **Max Indicators Per Fetch** parameter to prevent timeouts during the initial sync with large lookback windows. The feed will now process indicators in batches and resume from where it left off in the next fetch interval.
