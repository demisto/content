#### Integrations

##### PANOS-HA

- Enhanced: panos-ha-get-config now displays comprehensive HA configuration details with organized markdown sections
- Added: Overview section showing HA enabled status, group ID, mode, and sync settings
- Added: Peer Configuration section displaying primary and backup peer IPs
- Added: Election Settings section showing device priority, preemptive mode, heartbeat backup, and timers
- Added: Detailed interface configuration for HA1, HA1-backup, HA2, and HA2-backup with all properties
- Added: Monitoring section showing link and path monitoring status
- Added: Link Monitoring Groups with status indicators, failure conditions, and monitored interfaces
- Improved: Conditional display for backup interfaces (shows "Not configured" if not set up)
- Improved: Visual enhancements with emoji section headers for better readability
