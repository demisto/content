
#### Integrations

##### CyberArk EPM SOC Response

- Deleted the **Application ID** parameter.
- Added support for **Identity URL** parameter that the cyberark identity fqdn for oauth2 authentication (e.g., https://abc1234.id.cyberark.cloud).
- Added support for **Client ID** parameter that service username (configured as oauth confidential client).
- Added support for **Web App ID** parameter that the application id of the oauth2 server web app configured in identity administration.
