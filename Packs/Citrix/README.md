Citrix Cloud is a unified platform that provides management, monitoring, and delivery services for Citrix products and resources across cloud and on-premises environments. It centralizes administration, enhances visibility, and simplifies operational workflows.

For more information about Citrix Cloud, visit:
https://www.citrix.com/products/citrix-cloud/

## What does this pack do?

This pack includes the Citrix Cloud SystemLog integration that:

- Collects system log records from the Citrix Cloud platform using the Citrix Cloud SystemLog API
- Monitors operational, administrative, and security-related activities across Citrix Cloud services
- Tracks events such as resource changes, configuration updates, and user-initiated actions
- Provides centralized visibility into Citrix Cloud activity for auditing, security monitoring, and compliance
