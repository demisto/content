#### New: Twilio SendGrid Pack v1.0.0
This is the first version of the Twilio SendGrid pack. This pack is designed to collect and process email activity events from Twilio SendGrid, providing visibility into email deliveries, opens, clicks, bounces, and spam reports.

##### Integrations
* **Twilio SendGrid Event Collector**
  * Collects email activity events from the Twilio SendGrid API.
  * Handles incremental fetching and deduplication.
  * Provides a debug command for testing and troubleshooting.
