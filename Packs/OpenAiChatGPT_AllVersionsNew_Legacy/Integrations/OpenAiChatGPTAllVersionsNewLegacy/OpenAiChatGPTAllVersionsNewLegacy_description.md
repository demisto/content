## OpenAi ChatGPT
### This section explains how to configure the instance of OpenAi ChatGPT in Cortex XSOAR.
- Login to OpenAi API
- Click on Create New Secret Key
- Copy the key to the left parameter
- URL is default "To Change it Check OpenAI API Guide"
- Click Test
