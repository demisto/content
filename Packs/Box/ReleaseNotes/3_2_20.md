
#### Integrations

##### Box Event Collector

- Fixed an issue where the *fetch-events* command timed out on high-volume tenants because a single fetch attempted to drain the entire event backlog. Added a new **Maximum number of events per fetch** parameter that caps the total number of events collected per fetch cycle, and ensured the event stream position is persisted incrementally so the backlog drains gradually across successive fetches.
- Clarified the **Maximum number of events per page** parameter description to indicate it controls the per-request page size only.
