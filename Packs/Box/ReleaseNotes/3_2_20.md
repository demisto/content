
#### Integrations

##### Box Event Collector

- Fixed an issue where the ***fetch-events*** command timed out on high-volume tenants because a single fetch attempted to drain the entire event backlog. Switched from limiting the number of events per API request to limiting the total number of events per fetch cycle via the new *Maximum number of events per fetch* parameter, and ensured the event stream position is persisted incrementally so the backlog drains gradually across successive fetches.
