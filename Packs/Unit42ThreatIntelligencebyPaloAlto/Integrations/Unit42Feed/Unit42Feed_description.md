## Unit 42 Feed

Use the Unit 42 Feed integration to fetch indicators and threat objects from Palo Alto Networks Unit 42's comprehensive threat feed. This integration provides continuous access to high-quality indicators of compromise (IOCs) and contextual threat data.

### Authentication

This integration requires a Unit 42 license and API access. Contact your Palo Alto Networks representative to obtain the necessary credentials and API key.

### Configuration Parameters

- **Feed Types**: Select which types of data to fetch (Indicators, Threat Objects, or both)
- **Indicator Types**: Choose specific indicator types to collect (IP, Domain, URL, File)
- **Source Reliability**: Configure the reliability level for feed data (default: A++ - Reputation script)
