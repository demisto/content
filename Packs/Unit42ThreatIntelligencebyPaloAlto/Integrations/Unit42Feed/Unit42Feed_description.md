## Unit 42 Feed

Use the **Unit 42 Feed** integration to fetch indicators and threat objects from Palo Alto Networks Unit 42's comprehensive threat feed. This integration provides continuous access to high-quality indicators of compromise (IOCs) and contextual threat intelligence.

### Prerequisite

This integration requires a Cortex Threat Intelligence Management (TIM) license.

### Configuration Parameters

- **Feed Types**: Select which types of objects to fetch (Indicators, Threat Objects, or both)
- **Indicator Types**: Choose specific indicator types to collect (IP, Domain, URL, File)
- **Source Reliability**: Configure the reliability level for feed context (default: A++ - Reputation script)
