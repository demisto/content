##### Gurucul-GRA

- **Breaking Change**: New drop-down ***Fetch Incident By Case Status*** is added in ***Add instance***. The valid options are:
  - ***ALL***: Incident is fetched for ALL Case Status. This is default value set.
  - ***OPEN***: Incident is fetched for OPEN Case Status.
  - ***REOPENED***: Incident is fetched for REOPENED Case Status.
  - ***CLOSED***: Incident is fetched for CLOSED Case Status.
  - ***RISK ACCEPTED***: Incident is fetched for RISK ACCEPTED Case Status.
