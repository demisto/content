Vercara UltraDNS is a cloud-based DNS management platform that provides DNS services and configuration management capabilities.

For more information about Vercara UltraDNS, visit: https://vercara.digicert.com/resources/ultradns

## What does this pack do?

This pack includes the Vercara UltraDNS integration that:

- Collects DNS configuration audit logs from Vercara UltraDNS platform
- Monitors DNS record changes including creation, modification, and deletion events
- Tracks user activities and configuration changes across DNS infrastructure
- Provides visibility into DNS management activities for security and compliance purposes
