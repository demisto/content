# Vercara UltraDNS Event Collector

This integration collects DNS configuration audit logs from the Vercara UltraDNS platform.

## Configuration

### Authentication

- **Username**: Your Vercara UltraDNS account username
- **Password**: Your Vercara UltraDNS account password

The integration uses **OAuth 2.0 password grant flow** for authentication. Access tokens are automatically refreshed when they expire, ensuring continuous operation.
