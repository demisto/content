Stellar Cyber enables a security team to protect their cloud, on-premises, and IT/OT environments from a single platform without changing their existing security stack. This Pack contains Stellar Cyber incident fields, classifier/mappings, a layout, as well as the integration.

##### What does this pack do?

The Integration in this pack provides the following functionality:

- Fetch and mirror Cases (including alerts) as Incidents from the Stellar Cyber Platform.
- Retrieve an Alert from Stellar Cyber.
- Update Stellar Cyber Case severity, status (including setting Case as resolved), assignee, or add/remove tags.
