
#### Integrations

##### ServiceNow v2

- Added support for **Mirrored ServiceNow Ticket Close Codes** parameter that specify how to close mirrored tickets in servicenow using a close code. provide a comma separated list of valid close_code values (e.g., no resolution provided,duplicate). note that a corresponding list of custom close reasons must be defined under server configuration in cortex xsoar to ensure proper mapping. if no close code is provided, no close code will be sent when closing tickets.
