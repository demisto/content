
#### Integrations

##### ServiceNow v2

- Added support for **Mark attachment as note** parameter that mark attachments as note. Set this to yes if you prefer to mark file attachments from servicenow as notes.
- Added support for **Interpret comments in a specific format** parameter that define the format of the servicenow comments. Per default the format is supplied by servicenow during the fetch as part of the comments data (source). This can be overwritten to a specific format for all comments. The available options are source, text, and html. Use html if a html editor is used within servicenow to add comments. defaults to source.
