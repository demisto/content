
#### Integrations

##### Team Cymru Scout

- Added support for **Use Foundation API for IP enrichment** parameter. Enable this parameter to allow the use of Foundation API for IP enrichment. If not selected, then "ip" command will utilize the Scout details API endpoint.

- Added support for **force_use_scout_details_api** argument in the **ip** command. If set to yes, it will forcefully use the Scout Details API endpoint, overriding the **Use Foundation API for IP enrichment** parameter in the integration configuration.

- Updated the Docker image to: *demisto/python3:3.12.12.5490952*.
