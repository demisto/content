Each indicator is ranked from 0 to 100. Indicators are being collected from multiple sources and are cross-verified using multiple criteria. 
Please check indicator tags and malware family field. An indicator may describe you a known malware or a scanning host. Threfore, different actions may be required based on the context.

Usually, to exlude False Positives use the following thresholds:
 - for IPs you start with score=50
 - for Domains you start with score=40
 - for URLs you start with score=30
