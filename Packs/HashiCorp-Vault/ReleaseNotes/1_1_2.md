
#### Integrations
##### HashiCorp Vault
- Added a new parameter to the ***hashicorp-configure-engine*** command called *aws_roles_list* that will let the user control for which roles it wants to generate credentials for. 
- Added a new parameter to the ***hashicorp-configure-engine*** command called *aws_method* that will let the user control the type of request to generate credentials by. 
- Removed ttl parameter from ***hashicorp-configure-engine*** command called *aws_method*.
