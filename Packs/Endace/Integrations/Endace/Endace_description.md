The integration uses Endace APIs to Search, Archive and Download PCAP file.

To setup an instance, follow the following steps:
1. Create a user account on EndaceProbe with RBAC roles of "vision_user" and "vision_download". 
2. Add this user account to XSOAR.
3. Navigate to Settings -> Integrations -> Servers and Services.
4. Search for "Endace".
5. Click Add Instance to create and configure a new integration instance. 
6. Add the Server URL of EndaceProbe.
7. Click Test to verify connectivity and user's access to EndaceProbe.

The integration works for EndaceProbe and Endace InvestigationManager.
For more information on how to configure EndaceProbe, visit the www.endace.com or email support@endace.com
