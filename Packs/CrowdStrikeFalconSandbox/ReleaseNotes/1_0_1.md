
#### Integrations
##### CrowdStrike Falcon Sandbox
Added the *dontThrowErrorOnFileDetonation* to the ***crowdstrike-submit-url*** command, which gives you the option not to throw an error when sending a URL that points to a file.

#### Playbooks
##### Detonate URL - CrowdStrike
The playbook will not fail when it detonates a URL that points to a file.
