## Microsoft Teams

- Added certificate-based authentication (client assertion JWT) support as an alternative to Bot Password for authenticating against Azure AD.
- Certificate authentication works with both the Bot Framework token acquisition and the Microsoft Graph token acquisition.
- Certificate authentication supports both the Client Credentials and Authorization Code flows.
- Added two new advanced integration parameters: **Certificate Thumbprint** (SHA-1 hex thumbprint of the certificate registered in Azure AD) and **Private Key** (PEM-encoded RSA private key). When both are provided, certificate-based authentication is used automatically instead of Bot Password.
