#### Palo Alto Networks PAN-OS

##### Playbooks

- New playbook to analyze Firewall software version and Global Protect client package version for vulnerability to a Palo Alto Networks Security Advisory:
  - ***PAN-OS - Security Advisories Vulnerability Check***
- This playbook depends on the Panorama integration ***pan-os-platform-get-system-info*** command
- It also uses two new custom scripts
  - ***PAN-OS_Security_Advisories_Enrichment***
  - ***CheckFirewallAndGPForCVEs***

##### Scripts

- PAN-OS_Security_Advisories_Enrichment
  - This script enriches CVEs published by Palo Alto Networks with detailed vulnerability information from the official Palo Alto Networks Security Advisories website <https://security.paloaltonetworks.com>.

- CheckFirewallAndGPForCVEs
  - This script checks if PAN-OS firewall and GlobalProtect versions are affected by specific CVEs (Common Vulnerabilities and Exposures).
