
#### Integrations

##### Palo Alto Networks PAN-OS

- Added the `ip_address` argument to the `pan-os-platform-get-device-state` command which allows the IP address of a Panorama managed firewall to be specified. This can be used in situations where a service route is enabled on a firewall.
- Updated the Docker image to: *demisto/pan-os-python:1.0.0.105214*.
