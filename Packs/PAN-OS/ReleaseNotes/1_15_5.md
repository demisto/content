
#### Playbooks
##### New: PAN-OS -Block Domains By Adding EDL To Anti-Spyware
This playbook blocks domains using Panorama Anti-Spyware. It assigns External Dynamic List URLs that contains domains to block to Panorama Anti-Spyware. You can create External Dynamic List(EDL) and add domains to it using the Cortex XSOAR pack called "Generic Export Indicators Service". (Available from Cortex XSOAR 6.5.0).
