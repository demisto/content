
#### Playbooks
##### New: PAN-OS -Block Domains By Adding EDL To Anti-Spyware
- This playbook goal is to block domains using Panorama Anti-Spyware. it assigning External Dynamic List URL that contains domains to block to Panorama Anti-Spyware. You can create External Dynamic List(EDL) and add domains to it using XSOAR pack called "Generic Export Indicators Service".  (Available from Cortex XSOAR 6.5.0).
