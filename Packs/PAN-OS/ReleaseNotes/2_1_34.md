
#### Integrations

##### Palo Alto Networks PAN-OS

- Added a add command for adding an exception to a Vulnerability Protection Profile or for a Anti Spyware Profile.
- Added an edit command for editing an exception in a Vulnerability Protection Profile or in a Anti Spyware Profile.
- Added a delete command for deleting an exception in a Vulnerability Protection Profile or in a Anti Spyware Profile.
- Added a list command for listing all exceptions in a Vulnerability Protection Profile or in a Anti Spyware Profile.
