
#### Playbooks

##### New: PAN-OS - DNS Sinkhole

- New: Added a playbook that creates a DNS sinkhole within PAN-OS Next Generation Firewalls. The playbook also outputs logs of the infected systems that were revealed in the process. (Available from Cortex XSOAR 6.8.0).

##### New: PAN-OS - Add Anti-Spyware Security Profile To Rule - Safely

- New: Added a playbook that adds an Anti-Spyware Security Profile to a security rule in PAN-OS Next Generation Firewalls in a safe manner. (Available from Cortex XSOAR 6.8.0).
