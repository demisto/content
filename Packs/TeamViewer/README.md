This pack contains an integration to collect events automatically from Team.
You can also use the ***teamviewer-get-events*** command to manually collect events.

1. User access token scope: Event logging - Allow requesting all event logs.
2. A Tensor license is required.

[For on instructions how to generate your script token, click here](https://community.teamviewer.com/English/kb/articles/109647-use-the-teamviewer-api).
