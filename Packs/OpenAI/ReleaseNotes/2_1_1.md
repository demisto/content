
#### Integrations

##### OpenAI GPT

- Added the ***gpt-create-response*** command to send messages via the OpenAI Responses API. Supports multi-turn conversations, reasoning effort control for o-series and gpt-5 models, and background execution with polling.
- Added the ***gpt-analyze-email-header*** command to analyze email headers for potential security issues using the OpenAI Responses API.
- Added the ***gpt-analyze-email-body*** command to analyze email body content for potential security risks using the OpenAI Responses API.
- Added the ***gpt-draft-soc-email*** command to draft SOC email templates using the OpenAI Responses API with prior conversation context.
- Added the ***gpt-list-models*** command to list all models available to the configured API key.
- Added the ***gpt-create-moderation*** command to run text or images through the OpenAI Moderations API and return per-category flagging results.
