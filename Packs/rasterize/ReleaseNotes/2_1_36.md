## Rasterize

- Added a new **Rasterize** integration variant for platform (version 8.15.0 and above) with the following changes:
  - The *url* argument of the **rasterize** command now accepts a single value (`isArray: false`).
  - Added a new hidden *lightweight* parameter (default: `true`).
- Added a new hidden *lightweight* parameter (default: `false`) to the existing integration for non-platform marketplaces.
- The existing integration is now limited to versions up to 8.14.99.
