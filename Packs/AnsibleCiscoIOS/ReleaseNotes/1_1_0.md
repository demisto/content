
#### Integrations

##### Ansible Cisco IOS

- Added support for **None** parameter that  this field determines how long (in seconds) the system waits to establish the initial handshake with the cisco device before timing out.
- Added support for **None** parameter that this field determines the timeout (in seconds) for the underlying communication socket.
- Added support for **Command timeout** parameter that his field controls how long  (in seconds) xsoar will wait for a specific command (like a full configuration backup) to finish before timing out.
