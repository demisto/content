
#### Integrations

##### CTIX v3

- Added support for **Timeout** parameter to configure connection timeout in seconds (default is 15).
- Deleted the **ctix-get-indicator-relations** command.
- Deleted the **ctix-delete-tag** command.
- Added support for **ctix-disable-or-enable-tags** command which disables or enables a tag with given tag_name.
- Added support for **ctix-get-object-relations** command that get object relations.
- Updated the **ctix-saved-result-set** command to not use the *query* argument.
- Added support for *version* argument in the **ctix-saved-result-set** command.
- Updated the Docker image to: *demisto/python3:3.12.12.6204436*.
