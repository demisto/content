#### Integrations

##### Cyware Intel Exchange

- Renamed the CTIX v3 integration to Cyware Intel Exchange.
- Updated the default timeout value from 15 seconds to 300 seconds to allow for longer response times from the Cyware Intel Exchange application.
- Added support for **Feed Fetch Interval** parameter that how often the platform polls ctix for new indicators (e.g. "30 minutes", "1 hour", "12 hours"). controls the xsoar feed scheduler cadence.
- Added support for **Incident type** parameter.
- Added support for **Indicator Reputation** parameter that indicators from this integration instance will be marked with this reputation.
- Added support for **Bypass exclusion list** parameter that when selected, the exclusion list is ignored for indicators from this feed. this means that if an indicator from this feed is on the exclusion list, the indicator might still be added to the system.
- Added support for **Saved Result Set Version** parameter that the version of saved result set to use.
- Added support for **Saved Result Set Label** parameter that the label name of the saved result set to pull indicators from.
- Added support for **Retrieve Enriched Data** parameter that if enabled, indicators will be enriched via bulk ioc lookup (with relations and enrichment data) before ingestion.
- Added support for **Fetch indicators** parameter.
- Added support for **Incident Fetch CQL Query** parameter that custom cql query used when fetching incidents. updates the default cql query.for example, type = \"report\" and severity = \"high\".
- Added support for **Tags** parameter that supports csv values.
- Added support for **Source Reliability** parameter that reliability of the source providing the intelligence data.
- Added support for **First fetch time** parameter that the date and time from which to start fetching on the first run. if left empty, defaults to 3 days ago. for example, 10 days or p10d.
- Added support for **Fetch incidents** parameter.
- Added support for **Traffic Light Protocol Color** parameter that the traffic light protocol (tlp) designation to apply to indicators fetched from the feed.
- Added support for **ctix-bulk-ioc-lookup-advanced** command that performs a bulk lookup for threat data objects in the ctix platform and retrieves details such as basic info, enriched data, and relations.
- Added support for *q* argument in the **ctix-get-conversion-feed-source** command.
- Added support for *ioc_type* argument in the **ctix-get-lookup-threat-data** command.
