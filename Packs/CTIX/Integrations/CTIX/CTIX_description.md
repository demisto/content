1. Open Administration > Integrations Management > CTIX Integrators.
2. Click Add New to create and configure a new integration instance and download the key files.
3. Sign to the Cortex XSOAR application and open the App Configuration page.
4. Fill in the following details:
    a. Name: Enter a name for the integration instance.
    b. Endpoint URL: Enter the endpoint URL of your CTIX Instance. The endpoint URL can be generated by a licensed user from the CTIX application. For example, https://<domain_name>/ctixapi/.
    c. Access Key: Enter the Access Key from the CTIX application. Only a licensed user can generate the Access Key from CTIX. For example, “74xxxxx7-xxxb-4xxa-xxxx-0xxxxxxxxxx2“.
    d. Secret Key: Enter the Secret Key from the CTIX application. Only a licensed user can generate the Secret Key from CTIX. For example, “0xxxxxx8-xxxx-4xx4-xxx6-5xxxxxxxxxxc”.
5.  Click Test to validate the URL, token, and connection.
