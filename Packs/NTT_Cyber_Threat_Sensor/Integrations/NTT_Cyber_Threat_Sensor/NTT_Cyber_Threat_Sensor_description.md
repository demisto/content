Integrate directly with the NTT TD service via Cyber Threat Sensor

## Requirements
For this integration you need one or more NTT Cyber Threat Sensors that are configured for SOCaaS service. 

During the registration of each device the following information will be provided:
* API key
* Tenant identification
* Tenant unique key

This information has to be configured in the integration instans of NTT Cyber Threat Sensor for XSOAR
