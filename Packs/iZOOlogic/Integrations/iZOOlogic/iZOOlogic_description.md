## iZOOlogic

### Obtaining API Credentials

To configure this integration, you need an **API Key** and a **Secret Key** from the iZOOlogic platform.

1. Log in to the [iZOOlogic platform](https://www.izoolabs.com) with an account that has **admin-level access**.
2. Navigate to your **user profile** settings.
3. Locate the **API Key** and **Secret Key** in your profile.
4. Copy both values and use them to configure the integration instance.
