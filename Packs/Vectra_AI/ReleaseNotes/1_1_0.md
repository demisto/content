
#### Integrations
##### Vectra Detect (Beta)
  - Updated the Docker image to: *demisto/python3:3.10.7.33922*.

  - Added support for Vectra Account entity, allowing you to create incidents based on Vectra Accounts and to interact with Vectra Account objects
  - New commands
    - vectra-search-accounts
    - vectra-account-describe
    - vectra-account-add-tags
    - vectra-account-del-tags

  - Improved Detection commands output
    - Fixed "Description"
    - Added "DestinationIPs" and "DestinationPorts"

- **Vectra Threat Score**
