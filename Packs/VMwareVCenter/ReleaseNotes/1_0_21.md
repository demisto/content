
#### Modeling Rules

##### VMware Vcenter 2.9 Modeling Rule

- Updated the **VMware Vcenter 2.9 Modeling Rule** to support virtual machine and data center names that contain spaces in VM Operations events (for example, `vim.event.VmRenamedEvent`).
- Updated the **VMware Vcenter 2.9 Modeling Rule** to map the target virtual machine hostname for `vim.event.VmReconfiguredEvent` events.
