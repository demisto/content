Use the ServiceNow MCP integration to securely connect to a ServiceNow Model Context Protocol (MCP) server and access ServiceNow tools in real time.

## What does this pack do?

This pack is intended for Cortex Agentic AI.

Connects to a ServiceNow MCP server to automatically discover tools and expose them as agentic system actions that users can enable for agents.

## Tools

Tool availability depends on the permissions granted to the user and the MCP server configuration on the ServiceNow instance.
