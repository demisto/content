# Datadog Cloud SIEM

Datadog is an observability service for cloud-scale applications, providing monitoring of servers, databases, tools, and services, through a SaaS-based data analytics platform.

The SaaS platform integrates and automates infrastructure monitoring, application performance monitoring and log management to provide unified, real-time observability of our customers' entire technology stack.

## How to get a Datadog Cloud SIEM Credentials 
### Create Datadog API Key and APP Key.

1. To Create **API Key**: Navigate to **Account** > **Organization Settings** > **API Keys** and click **New Key**.  

2. In the New API Key popup enter **Name** and click on **Create Key**. 

3. To Create **APP Key**: Navigate to **Account** > **Organization Settings** > **Application Keys** and click **New Key**.

4. In the New Key popup enter **Name** and click on **Create Key**. 


For more information please see: [log-into-datadog](https://app.datadoghq.com/).
