Use the CrowdStrike Falcon MCP integration to connect securely with a CrowdStrike Falcon Model Context Protocol (MCP) server and access its tools in real time.

## What does this pack do

This integration provides tools primarily intended for use with Agentix Actions. It enables you to:

- Retrieve a list of available tools in the CrowdStrike Falcon MCP server.
- Call a specific tool on the CrowdStrike Falcon MCP server with optional input parameters.
