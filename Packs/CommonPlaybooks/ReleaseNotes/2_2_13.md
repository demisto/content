#### Playbooks

##### New: Block Email - Generic v2

- Added the Block Email - Generic v2 playbook which will block emails at your mail relay integration. (Available from Cortex XSOAR 6.5.0).

##### New: Block Account - Generic v2

- Added the Block Account - Generic v2 playbook which blocks malicious usernames using all integrations that you have enabled.

Added support for Supported integrations for this playbook:

* Active Directory.
* PAN-OS - This requires PAN-OS 9.1 or higher. (Available from Cortex XSOAR 6.5.0).

##### New: Block Domain - Generic v2

- Added the Block Domain - Generic v2 playbook which blocks malicious Domains using all integrations that are enabled.

Added support for Supported integrations for this playbook:

* Zscaler.
* Symantec Messaging Gateway.
* FireEye EX.
* Trend Micro Apex One.
* Proofpoint Threat Response.
* Cisco Stealthwatch Cloud. (Available from Cortex XSOAR 6.5.0).

##### New: Block URL - Generic v2

- Added the Block URL - Generic v2 playbook which blocks malicious urls using all integrations that are enabled.

Added support for Supported integrations for this playbook:

* Palo Alto Networks PAN-OS
* Zscaler
* Sophos
* Forcepoint
* Checkpoint
* Netcraft (Available from Cortex XSOAR 6.5.0).

##### New: Block Indicators - Generic v3

- Added the Block Indicators - Generic v3 playbook which blocks malicious Indicators using all integrations that are enabled, using the following sub-playbooks:

* Block URL - Generic.
* Block Account - Generic.
* Block IP - Generic v3.
* Block File - Generic v2.
* Block Email - Generic.
* Block Domain - Generic. (Available from Cortex XSOAR 6.5.0).
