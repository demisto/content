This pack enables you to manage VMware vCenter and ESXi using Ansible modules. The Ansible engine is self-contained and pre-configured as part of this pack onto your XSOAR server. The Ansible modules have been exposed as XSOAR commands, and allow you to use them without needing to know Ansible.

# What does this pack allow you to manage?

* Datacenter
* Cluster
* ESXi Hosts
* Datastores
* DRS Groups/Rules
* DVS
* Folders
* Guests
* Users
* PortGroups
* vSwitches
* Resource Pools
* Tags
* Content Library
* vMotion
* Licenses
* Files in Datastores
