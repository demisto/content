
#### Integrations

##### Palo Alto Networks Cortex XDR - Investigation and Response

- Deprecated the ***xdr-get-alerts*** command. Use ***xdr-issue-list*** instead.
- Deprecated the ***xdr-update-alert*** command. Use ***xdr-issue-update*** instead.
- Added support for the ***xdr-issue-create*** command, which creates a new issue.
- Added support for the ***xdr-issue-update*** command, which updates an existing issue.
- Added support for the ***xdr-issue-list*** command, which retrieves issues based on filters.
