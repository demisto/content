
#### Integrations

##### Palo Alto Networks Cortex XDR - Investigation and Response

- Added input validation for malicious commands to ***core-run-script-execute-commands***. \
**WARNING** - This change may potentially break playbooks that use the command. The input validation can be overridden using the ***override_validation*** flag
  - Added a new optional argument ***override_validation*** to prevent input validation (It is not recommended to override the validation).
  - Added a new optional argument ***is_chained_command***. Can be set to `true` to override validation for special characters: `||`, `&&`, `;`.
