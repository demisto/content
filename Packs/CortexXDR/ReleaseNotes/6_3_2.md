
#### Integrations

##### Cortex XDR - XQL Query Engine

- Added support for **xdr-xql-library-create** command which creates or updates xql queries in the library.
- Added support for **xdr-xql-library-delete** command which deletes xql queries from the library.
- Added support for **xdr-xql-library-list** command which gets a list of xql queries from the library.
