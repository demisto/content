
#### Integrations

##### Palo Alto Networks Cortex XDR - Investigation and Response

- Added support for the ***xdr-case-list*** command, which returns a list of cases based on specified filters.
- Added support for the ***xdr-case-artifact-list*** command, which retrieves a list of artifacts associated with a specific case.
- Added support for the ***xdr-case-update*** command, which updates an existing case.
