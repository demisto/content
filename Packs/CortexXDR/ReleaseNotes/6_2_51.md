
#### Integrations

##### Palo Alto Networks Cortex XDR - Investigation and Response

- Added support for the **xdr-asset-group-delete** command that deletes an asset group.
- Added support for the **xdr-asset-schema-get** command that gets the schema of the asset inventory.
- Added support for the **xdr-asset-group-list** command that returns a list of asset groups.
- Added support for the **xdr-asset-group-create** command that creates an asset group.
- Added support for the **xdr-asset-schema-field-options-get** command that gets the enum values of a specified field in the asset schema.
- Added support for the **xdr-asset-list** command that returns a list of assets.
- Added support for the **xdr-asset-group-update** command that updates an asset group.
