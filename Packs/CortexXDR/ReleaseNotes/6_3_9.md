
#### Integrations

##### Palo Alto Networks Cortex XDR - Investigation and Response

- Deprecated the **xdr-get-incident-extra-data** command.
- Deprecated the **xdr-insert-cef-alerts** command.
- Deprecated the **xdr-get-cloud-original-alerts** command.
- Deprecated the **xdr-insert-parsed-alert** command.

- Added support for the **xdr-endpoint-triage** command, which initiates a forensics triage on the specified endpoints. You must have Cortex XDR with the Forensics add-on. The agent also must have the Forensics license enabled.
- Added support for the **xdr-vulnerability-details-get** command, which gets vulnerability details by ID.
- Added support for the **xdr-healthcheck-run** command, which runs a system health check on the Cortex XDR environment.
- Added support for the **xdr-endpoint-triage-preset-list** command, which gets a list of available triage presets.
