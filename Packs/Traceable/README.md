This Pack provides an integration to consume the Incidents from the Threat Activities detected in the Traceable AI API Security Platform. With this integration the Threat Activities detected by the Traceable Platform can be raised within Cortex Xsoar as Incidents allowing you to perform corrective actions on the detected Activites.

## What does this pack do?
- Based on the Integration Configuration pull the Threat Activities detected by the Traceable Platform into Cortex Xsoar as Incidents
- Ability to only pull in selected Activities narrowed by `Security Score Category`, `Threat Category`, `IP Reputation Level`, `IP Abuse Velocity` and `IP Location Type`

_For more information, visit the [Traceable Official Website](https://www.traceable.ai/)_
