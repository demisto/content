
#### Integrations
##### Traceable
- Updated the Docker image to: *demisto/python3:3.10.13.75921*.
- Updated the integration to only publish new incidents that have not been created before.
- Added the use of instance cache to store the state of incidents published.
- Added new commands to manage the instance cache - `list_incident_cache`, `purge_incident_cache`.
- Added new configuration `span_query_batch_size`.
- Added new configuration `timegap_between_repeat_incidents`.
- Added new configuration `isFetchUniqueIncidents`.
- Removed `anomalousAttribute` and `serviceName` from list of optional attributes for Incident/Traceable event.
- Introduced some performance improvements.
