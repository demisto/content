
#### Integrations

##### New: Cymulate v3

- New: Added a new integration- **Cymulate v3** that fetches findings from completed Cymulate assessments as Cortex XSOAR incidents using the V2 Assessment API.


#### Incident Fields

##### New: Cymulate Affected Assets

- New: Added a new incident field- **Cymulate Affected Assets** that displays the assets affected by the finding.

##### Cymulate Attack Type

- Updated the **Cymulate Attack Type** incident field to support the **Cymulate Assessment Findings** incident type.

##### New: Cymulate CIS Compliance Support

- New: Added a new incident field- **Cymulate CIS Compliance Support** that displays CIS compliance information related to the finding in markdown format.

##### New: Cymulate Defense Bypassed

- New: Added a new incident field- **Cymulate Defense Bypassed** that displays which security defenses were bypassed by the attack (multi-select field).

##### New: Cymulate Environment

- New: Added a new incident field- **Cymulate Environment** that displays the environment ID where the finding was detected.

##### New: Cymulate Finding Type

- New: Added a new incident field- **Cymulate Finding Type** that displays the type or category of the finding.

##### New: Cymulate Mitigation Type

- New: Added a new incident field- **Cymulate Mitigation Type** that displays the type of mitigation recommended for the finding.

##### Cymulate Module

- Updated the **Cymulate Module** incident field to support the **Cymulate Assessment Findings** incident type.

##### New: Cymulate Nist Compliance Support

- New: Added a new incident field- **Cymulate Nist Compliance Support** that displays NIST compliance information related to the finding in markdown format.

##### New: Cymulate Softwares

- New: Added a new incident field- **Cymulate Softwares** that displays the software information associated with the finding in markdown format.

##### Cymulate Status

- Updated the **Cymulate Status** incident field to support the **Cymulate Assessment Findings** incident type.

##### Cymulate Template Name

- Updated the **Cymulate Template Name** incident field to support the **Cymulate Assessment Findings** incident type.

##### Cymulate Test Case

- Updated the **Cymulate Test Case** incident field to support the **Cymulate Assessment Findings** incident type.


#### Incident Types

##### New: Cymulate Assessment Findings

- New: Added a new incident type- **Cymulate Assessment Findings** that represents assessment findings from Cymulate's breach and attack simulation platform.

#### Layouts

##### New: Cymulate Assessment Findings Layout

- New: Added a new layout- **Cymulate Assessment Findings Layout** that provides a structured view for Cymulate Assessment Findings incidents with all relevant fields organized for easy review and analysis.

#### Mappers

##### New: Cymulate Assessment Findings - Incoming Mapper

- New: Added a new mapper- **Cymulate Assessment Findings - Incoming Mapper** that maps incoming Cymulate assessment finding data to Cortex XSOAR incident fields for the Cymulate Assessment Findings incident type.
