For the CyberX Api Version 2.5

Allows demisto to integrate with CyberX and
perform actions with these alerts. 

To get the alerts from CyberX with UUID:
Send the data discovered by CyberX to Demisto by defining a forwarding rule in the CyberX system (via QRadar forwarding  to a Demisto Syslog Integration) and select the Remote support alert handling option which is only available in the QRadar option.
