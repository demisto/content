
#### Integrations

##### CrowdStrike Falcon Intelligence Sandbox

- Added the **network_settings** argument, which allows configuring the sandbox network behavior (default, tor, simulated, or offline), to the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

- Updated the predefined values of the **environment_id** argument for the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

- Deprecated the **enable_tor** argument (use **network_settings** instead) in the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**
