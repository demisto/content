
#### Integrations

##### CrowdStrike Falcon Intelligence Sandbox

- Added the **network_settings** argument to the following commands, allowing configuration of sandbox network behavior (default, tor, simulated, or offline):  
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

- Updated the predefined values of the **environment_id** argument in the following commands:  
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

- Deprecated the **enable_tor** argument (use **network_settings** instead) in the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**
