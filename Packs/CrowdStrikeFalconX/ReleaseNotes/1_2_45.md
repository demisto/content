
#### Integrations

##### CrowdStrike Falcon Intelligence Sandbox

- Added the **network_settings** argument to the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

  The argument allows configuring the sandbox network behavior (default, tor, simulated, or offline).

- Updated the predefined values of the **environment_id** argument for the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

  The updated values reflect the currently supported sandbox environments, including macOS Catalina 10.15 and Windows 11 (64-bit).

- Deprecated the **enable_tor** argument in the following commands:
  - **cs-fx-submit-url**
  - **cs-fx-submit-uploaded-file**

  Use the **network_settings** argument instead.
