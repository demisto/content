### Supported versions
Supports Fortinet Fortiweb VM version 1 and version 2
Insert Fortiweb API version (V1/V2). If your Fortiweb version is above 6.3.6, choose V2, otherwise choose V1.

### Authentication
When using Cortex XSOAR credentials, insert your Fortiweb username to the credential username and the Fortiweb password to the credential password.
