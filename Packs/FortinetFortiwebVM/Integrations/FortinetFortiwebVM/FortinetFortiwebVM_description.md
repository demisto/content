### Supported versions
Supports Fortinet Fortiweb VM cersion 1 and version 2
Insert Fortiweb API Version (V1/V2). If your Fortiweb version above 6.3.6, choose V2, otherwise choose V1.

### Authentication
When using XSOAR credentials insert your Fortiweb username to the credential username and the Fortiweb password to the credential password.
