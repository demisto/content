
#### Integrations

##### Google Gemini

- Added support for the Vertex AI backend using Service Account keys (in addition to the existing AI Studio API Key authentication).
- Updated the supported model list to reflect Google's deprecation schedule:
  - Removed retired models: ***gemini-1.5-pro***, ***gemini-1.5-flash***, ***gemini-1.5-flash-8b***, ***gemini-2.5-pro-preview-06-05***, ***gemini-2.5-flash-preview-05-20***, ***gemini-2.0-flash-preview-image-generation***, ***text-embedding-004***, and ***models/embedding-001***.
  - Added current stable models: ***gemini-2.5-pro*** and ***gemini-2.5-flash***.
  - Added preview models: ***gemini-3.1-pro-preview***, ***gemini-3.1-flash-preview***, and ***gemini-3.1-flash-lite***.
  - Removed unused ***models/aqa*** and embedding models from the dropdown — the integration's only command (***google-gemini-send-message***) targets the `:generateContent` endpoint, which is incompatible with these models.
  - Removed ***models/gemini-3.1-flash-image-preview*** from the dropdown — image-output parts are not yet rendered by ***google-gemini-send-message***.
  - Kept ***gemini-2.0-flash*** and ***gemini-2.0-flash-lite*** as deprecated (legacy only) — operational until June 1, 2026.
- Changed the default model from ***gemini-2.5-flash-preview-05-20*** (retired) to ***gemini-2.5-flash***.
- Upgraded the Docker image to: *demisto/google-api-py3*.
