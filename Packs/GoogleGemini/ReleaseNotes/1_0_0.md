# Google Gemini Integration - Release Notes 1.0.0

## Initial Release

### New Integration
- **NEW**: Google Gemini AI Assistant integration for XSOAR
- **NEW**: AI-powered analysis and chat capabilities

### Features

#### googlegemini-chat Command
- Send prompts to Google Gemini AI models
- **Parameters:**
  - `prompt` (required): The prompt or question to send to the AI model
  - `model` (optional): Override default model for specific requests
  - `max_tokens` (optional): Maximum response tokens (1-10000, default: 10000)
  - `temperature` (optional): Response randomness (0.0-1.0, default: 0.7)
  - `history` (optional): Conversation history in JSON format for context

#### Supported Models
- gemini-2.0-flash, gemini-2.0-flash-lite
- gemini-1.5-flash, gemini-1.5-flash-8b, gemini-1.5-pro
- gemini-2.5-flash-preview-05-20 (default)
- gemini-2.5-pro-preview-05-06
- gemini-2.0-flash-preview-image-generation
- gemini-2.5-flash-preview-native-audio-dialog
- gemini-2.5-flash-exp-native-audio-thinking-dialog
- text-embedding-004, models/aqa

#### Configuration
- Gemini API URL (default: https://generativelanguage.googleapis.com)
- API Key authentication
- Default model selection
- SSL certificate verification options
- System proxy support

#### Outputs
- `GoogleGemini.Chat.prompt`: Original prompt
- `GoogleGemini.Chat.response`: AI model response
- `GoogleGemini.Chat.model`: Model used for generation
- `GoogleGemini.Chat.temperature`: Temperature setting used

### Technical Requirements
- **Minimum XSOAR Version**: 6.10.0
- **Docker Image**: gemini:3.12.8.1983910
- **Category**: AI/ML
