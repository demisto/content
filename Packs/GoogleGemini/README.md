# Google Gemini Pack

This pack contains the Google Gemini integration for XSOAR.

## Requirements

- Cortex XSOAR version 6.10.0 or later.
- A Google Gemini API Key.

## Integrations

### GoogleGemini
Google Gemini Large Language Model Integration for AI-powered analysis and chat capabilities.

- **Chat**: Send prompts to Gemini models.
- **Configurable Models**: Supports various Gemini models.
