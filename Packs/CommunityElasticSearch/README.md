## What does this pack do?

- Enables the use of Elastic Security for SIEM to manage security operations and search Elastic logs.
- Manages Elastic Security cases, detection alerts, rules, and value lists through the Kibana integration.
- Designed to be used in combination with the Elasticsearch v2 integration.

The Kibana integration supports Elasticsearch 6.0.0 and later, and was tested with Elasticsearch versions 6.6.2, 7.3, 8.4.1, and 9.3.1.
