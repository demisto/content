
#### Integrations

##### New: Browser Use Local

- New: Added a second integration in this pack that runs the open-source [`browser-use`](https://github.com/browser-use/browser-use) library inside the integration container — Chromium + LLM execute on-prem (BYO Anthropic / Google / OpenAI / Browser Use API key). Heavier and slower than the cloud integration, but fully self-contained. Available from Cortex XSOAR 6.10.0.
