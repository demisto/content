
#### Integrations

##### New: Browser Use

- New: Added a new integration that uses the Browser Use Cloud API (v3) to dispatch AI-driven browser automation tasks (Claude / Gemini agents inside a stealth Chromium browser). Supports running and tracking agent tasks, raw stealth browser sessions (with a CDP URL), persistent profiles for cookie/login reuse, and workspaces for file storage. Available from Cortex XSOAR 6.10.0.
