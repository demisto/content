
#### Indicator Fields

##### New: Organization Prevalence

Added new number field to indicate how frequently a specific indicator type is detected within a particular organization.

##### New: Globally Prevalence

Added new number field to indicate how frequently a specific indicator type is detected across all organizations.

##### New: Organization First Seen

Added new date field to indicate the first time a specific organization encountered an indicator.

##### New: Organization Last Seen

Added new date field to indicate the most recent time a specific organization encountered an indicator.

##### New: Global First Seen

Added new date field to indicate the first time an indicator was detected globally across all organizations.

##### New: Global Last Seen

Added new date field to indicate the most recent time an indicator was detected globally across all organizations.
