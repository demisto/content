## Farsight Security DNSDB®

The Farsight DNSDB® integration requires a subscription to DNSDB Enterprise
Edition, DNSDB Community Edition, or DNSDB Export. Please visit
https://www.farsightsecurity.com/get-started/ for more information about how to
get started. You will need to configure the DNSDB Integration with the API Key
and DNSDB server that is provided to you by Farsight.
