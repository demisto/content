## Netscout Arbor Sightline Help

### Generate API Key
In order to generate a new API Key, follow the instructions in the Sightline REST API Documentation: **Administartion** > **Sightline REST API Documentation** > **Access and Authorization** > **REST API tokens** > **Generating and managing REST API tokens**

### Configuration params
**NOTE**: If using 6.0.2 or lower version, put your API Key in the **Password** field, leave the **User** field empty.
