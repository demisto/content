Mandiant Attack Surface Management (ASM) provides comprehensive visibility of the extended enterprise, enabling security
teams to proactively mitigate real-world threats. Mandiant ASM scans corporate assets and cloud resources daily,
identifying application and service technologies. The module assesses exposure risks to the organization, assigns severity
levels, and can create Incidents within Cortex XSOAR.

# Key Features
- Configure your Mandiant ASM and Cortex XSOAR and integration via API key.
- Select a single project and multiple collections to feed issues into XSOAR
- Set a minimum severity threshold on the issues presented to the team.
- Synchronize issue management between XSOAR and Attack Surface Management; reflect status changes and remediation progress in both products.
