
#### Triggers Recommendations

##### Cortex Exposure Management GCP Virtual Machine Remediation

- Added a new triggers recommendation - Cortex Exposure Management GCP Virtual Machine Remediation. This automation rule is responsible for handling remediation of Cortex Exposure Management issues for Virtual Machines in GCP.

##### Cortex Exposure Management AWS Virtual Machine Remediation

- Added a new triggers recommendation - Cortex Exposure Management AWS Virtual Machine Remediation. This automation rule is responsible for handling remediation of Cortex Exposure Management issues for Virtual Machines in AWS EC2.

##### Cortex Exposure Management Azure Virtual Machine Remediation

- Added a new triggers recommendation - Cortex Exposure Management Azure Virtual Machine Remediation. This automation rule is responsible for handling remediation of Cortex Exposure Management issues for Virtual Machines in Azure.

##### Removed: trigger recommendation Cortex Exposure Management RdpServer Remediation

- Removed : Deprecated trigger recommendation for legacy RDP server remediation in cloud environments. Automation offered by this rule is replaced by the new triggers in this release.

##### Removed: trigger recommendation Cortex Exposure Management SshServer Remediation

- Removed : Deprecated trigger recommendation for legacy RDP server remediation in cloud environments. Automation offered by this rule is replaced by the new triggers in this release.
