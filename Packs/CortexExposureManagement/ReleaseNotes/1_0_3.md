
#### Triggers Recommendations

##### Cortex Exposure Management Enrichment

- Updated the Cortex Exposure Management Enrichment triggers recommendation to omit issues with specific ASM rule IDs that are now handled in other triggers.

##### Cortex Exposure Management GCP Virtual Machine Remediation

- Updated triggers recommendation - Cortex Exposure Management GCP Virtual Machine Remediation. This automation rule is responsible for handling remediation of Cortex Exposure Management issues for Virtual Machines in GCP, and replaces an issue filter that previously handled only RDP Server exposures.

##### Cortex Exposure Management AWS Virtual Machine Remediation

- Updated triggers recommendation - Cortex Exposure Management AWS Virtual Machine Remediation. This automation rule is responsible for handling remediation of Cortex Exposure Management issues for Virtual Machines in AWS EC2, and replaces an issue filter that previously handled only SSH Server exposures.

##### Cortex Exposure Management Azure Virtual Machine Remediation

- Added a new triggers recommendation - Cortex Exposure Management Azure Virtual Machine Remediation. This automation rule is responsible for handling remediation of Cortex Exposure Management issues for Virtual Machines in Azure.
