
#### Playbooks

##### Post Intrusion Ransomware Investigation

- Replaced the **Block Indicators - Generic v2** sub-playbook with the new **Block Indicators - Generic v3** playbook.
- Added input `AutoBlockIndicators` - This input determines if to block the indicators automatically or manually using the sub-playbook `Block Indicators - Generic v3`.
- Added input `UserVerification` - This input determines whether blocking IPs is should be done with or without user verification.  The IPs blocking is done by using the **Block Indicators - Generic v3** sub-playbook.
