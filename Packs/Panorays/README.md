# Panorays
This pack provides integration with the **Panorays** platform to monitor and manage internal security findings and posture.

## What does this pack do?
This pack allows you to integrate Panorays' security posture data into Cortex XSOAR to automate the management of your organization's internal findings.

* **Automated Incident Ingestion:** Automatically fetch and create incidents in XSOAR based on new internal security findings identified by Panorays for your organization.
* **Internal Visibility:** Retrieve detailed lists of self-assessment findings, including severity, category, and affected assets, directly within the XSOAR war room.
* **Posture Management:** Accelerate your response to internal security gaps by centralizing Panorays findings alongside your other security tools.

For more information, visit the [Panorays website](https://panorays.com).
