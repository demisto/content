## How to get your Panorays API Key

1. Log in to your Panorays platform.
2. Navigate to **Settings → Integrations → Public API**.
3. Click **Generate API Key** and copy the token.

For full instructions, see the [Panorays Public API documentation](https://support.panoraysapp.com/en/articles/520347-panorays-public-api).
