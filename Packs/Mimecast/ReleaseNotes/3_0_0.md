#### Integrations

##### New: Mimecast Event Collector v2

- **New**: Added a new integration that fetches Audit events and SIEM logs for various SIEM event types, using Mimecast API 2.0 with OAuth2 authentication.
- Added the ***mimecast-get-events*** command for retrieving Mimecast Audit events and SIEM logs.

##### Mimecast Event Collector (Deprecated)

- Deprecated the integration following [the announcement about Mimecast API 1.0 End of Life](https://mimecastsupport.zendesk.com/hc/en-us/articles/39704312201235-API-Integrations-API-1-0-End-of-Life-Mar-2025). Use Mimecast Event Collector v2 instead.
