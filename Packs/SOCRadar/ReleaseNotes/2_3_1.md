#### Integrations

##### SOCRadar Incidents

- Added **Critical** severity level support to incident fetching and severity mapping.
- Migrated API Key credential from `type: 4` to `type: 9` with `displaypassword` and `hiddenusername` for improved security.

##### SOCRadar Incidents v4

- Added the following incident custom fields for enriched incident data:
  - **SOCRadar Response** - Response action steps from the alarm.
  - **SOCRadar Mitigation** - Default mitigation plan from the alarm type.
  - **SOCRadar Detection And Analysis** - Detection and analysis guidance.
  - **SOCRadar Post Incident Analysis** - Post-incident analysis recommendations.
  - **SOCRadar Compliance** - Related compliance frameworks (GDPR, ISO 27001, NIST, PCI DSS, etc.).
  - **SOCRadar Related Assets** - Related assets associated with the alarm.
  - **SOCRadar Related Entities** - Related entities (domains, CVEs, IPs, etc.).
  - **SOCRadar Incident Link** - Direct link to the alarm in SOCRadar platform.
  - **SOCRadar Incident Content** - Full alarm content details.
