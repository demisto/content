This playbook triggers automated remediation for all SafeBreach generated indicators generated by insights. Then it reruns related insights and tags remaining indicators as not remediated ("NotRemediated" tag).

This playbook uses the following sub-playbooks, integrations, and scripts.

* SafeBreach Rerun Insights
