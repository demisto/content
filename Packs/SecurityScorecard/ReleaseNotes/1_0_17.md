
#### New: SecurityScorecard Event Collector

New: **SecurityScorecard Event Collector** integration (Available from Cortex XSIAM 8.11.0).

- Collects history events from SecurityScorecard for ingestion into Cortex XSIAM.
- Enriches each event with detailed information from the event's detail URL.
- Handles API rate limiting gracefully by sending collected events and waiting for the next fetch cycle.
- Supports deduplication of events across fetch cycles.
