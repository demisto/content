
#### Integrations

##### CybelAngel

- The **First fetch timestamp** parameter is no longer available.
- Added support fetching new events: Credentials watchlist, Domain watchlist.
- Added support for **Event Types To Fetch** parameter.
- Changed the name of **The maximum number of events per fetch** parameter to **The maximum number of reports per fetch**.
- Added support for **The maximum number of Domain watchlist events  per fetch** parameter.
- Added support for **The maximum number of Credential watchlist events per fetch** parameter.
- Added support for **Fetch events** parameter.
- Added support for *event_type* argument in the **cybelangel-get-events** command.
- Updated the Docker image to: *demisto/python3:3.12.8.3296088*.
