#### Integrations

##### New: Zscaler Internet Access via ZIdentity

- Added a new integration **Zscaler Internet Access via ZIdentity** that authenticates using OAuth 2.0 client credentials via ZIdentity, replacing the legacy session-based authentication.
- Supports the following commands:
  - **zia-denylist-list** - Gets a list of URLs and IPs on the denylist.
  - **zia-denylist-update** - Updates the denylist (add, remove, or overwrite).
  - **zia-allowlist-list** - Gets a list of URLs on the allowlist.
  - **zia-allowlist-update** - Updates the allowlist (add, remove, or overwrite).
  - **zia-category-list** - Gets information about URL categories.
  - **zia-category-update** - Updates a URL category.
  - **zia-url-quota-get** - Gets URL quota information.
  - **zia-ip-destination-group-list** - Lists IP destination groups.
  - **zia-ip-destination-group-update** - Updates an IP destination group.
  - **zia-ip-destination-group-add** - Adds a new IP destination group.
  - **zia-ip-destination-group-delete** - Deletes an IP destination group.
  - **zia-user-list** - Lists users.
  - **zia-user-update** - Updates a user.
  - **zia-groups-list** - Lists groups.
  - **zia-departments-list** - Lists departments.
  - **zia-sandbox-report-get** - Gets a sandbox analysis report for an MD5 hash.
  - **zia-activate-changes** - Activates saved configuration changes.
  - **url** - Looks up URL classifications.
  - **ip** - Looks up IP address classifications.
  - **domain** - Looks up domain classifications.
