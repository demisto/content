
#### Integrations

##### Zscaler Internet Access

- Added the *category-name* argument to support Zscaler API changes that require the category name for custom category operations to the following commands:
  - ***zscaler-category-add-url***
  - ***zscaler-category-add-ip***
  - ***zscaler-category-remove-url***
  - ***zscaler-category-remove-ip***
- Fixed an issue where modifying custom URL categories resulted in HTTP 400 errors. The integration now includes the 'configuredName' and 'superCategory' fields required by the Zscaler API for custom category operations.
- Updated the Docker image to: *demisto/python3:3.12.12.6796194*.
