### WithSecure Elements Integration
This integration allows you to interact with WithSecure Elements API to:
- Query security events from EPP, EDR, Collaboration Protection, and Exposure Management
- Manage devices and perform remote operations
- Handle Broad Context Detections (BCDs)
- Monitor organizations and invitations
- Manage device invitations and profiles

### Authentication
Access to Elements API uses OAuth 2.0 with client credentials flow. You need:
- Client ID
- Client Secret
- API scope (connect.api.read and/or connect.api.write)
