#### Layouts

##### New: RDP Sessions

- New: This incident type represent an RDP session hunt (Available from Cortex XSOAR 6.9.0).

#### Playbooks

##### New: RDP Bitmap Cache - Detect and Hunt

- New: This playbook automates the collection and forensic analysis of RDP sessions cache data by collecting the cache files and convert it to an image, extract readable text from the image and build IOC's from text and finally enrich any extracted indicators for further hunting (Available from Cortex XSOAR 6.9.0).
##### New: Set list of indicator types

- New: This playbook takes IOCs as input and adds the type into a list to be able to calculate stats about the types of the extracted indicators (Available from Cortex XSOAR 6.9.0).
##### New: Set RDP Bitmap Cache Overall Score

- New: This playbook sets the RDP bitmap cache overall score (Available from Cortex XSOAR 6.9.0).

#### Scripts

##### New: SetRDPOverallScore

- New: Sets the overall score for the strings similarity scoring for the text extracted from the RDP cache image. (Available from Cortex XSOAR 6.9.0).
