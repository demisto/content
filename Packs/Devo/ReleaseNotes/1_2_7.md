#### Integrations
##### Devo v2
- Created new parameter Fetch Incidents Lookback Time in yml file. This parameter specifies the time in past from which you wish to retrieve alerts.
- Created new parameter Fetch Incident Time Frame in yml file. This parameter is used to set window lookback time. Whenever this parameter is set it will set the `from_time` to use the window to pull all alerts.
- Updated the Docker image to: *demisto/devo:1.0.0.72596*.
