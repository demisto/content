#### Integrations

##### AWS SNS SMS Communication

- Added **Reply Code Mode** configuration parameter with two options:
  - **random** (default): Generates 4-digit reply codes (e.g., 1234, 5678) - existing behavior
  - **sequential**: Uses simple incrementing numbers (1, 2, 3...) across active questions for simpler UX
- Sequential mode automatically assigns the next available number, skipping codes already in use by active entitlements
- Reply code validation now accepts any numeric code (supports both modes)
