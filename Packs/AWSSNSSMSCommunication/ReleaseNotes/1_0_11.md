#### Integrations

##### AWS SNS SMS Communication

- Added **aws-sns-sms-list-entitlements** command to list all active entitlements with phone numbers, reply codes, age, and answer status. Supports filtering by phone number and optionally showing answered entitlements.
- Added comprehensive debug logging throughout the integration using *demisto.debug()* for better troubleshooting of entitlement processing, SMS reply handling, and AWS client operations.
