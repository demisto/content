#### Integrations

##### AWS SNS SMS Communication

- **Added test/debug command: aws-sns-sms-inject-reply**
  - Simulates SMS reply for testing without requiring actual SQS messages
  - Bypasses AWS infrastructure to test entitlement processing logic
  - Arguments: phone_number, reply_code
  - Returns detailed success/failure information
  - Shows all active entitlements and codes on failure
  - Useful for debugging Ask task integration and reply code mapping

- **Use case:**
  1. Send notification with SMSAskUser script
  2. Check codes with aws-sns-sms-list-entitlements
  3. Test reply processing with aws-sns-sms-inject-reply
  4. Verify entitlement handling without AWS SMS infrastructure
