#### Integrations

##### AWS SNS SMS Communication

- **Removed non-functional URL decoding for built-in Ask tasks**
  - Built-in XSOAR Ask tasks are NOT supported with SMS
  - XSOAR generates URLs after send-notification completes
  - Cannot intercept or decode URLs in the integration

- **SMSAskUser is the ONLY supported method for Ask-style communication:**
  - Use the SMSAskUser automation script in playbooks
  - Creates entitlement GUID and formats message properly
  - Format: "Question - Reply option1 or option2: GUID@incident|task"
  - Integration parses this format and generates reply codes
  - Final SMS: "Question\nOption1 (1234) or Option2 (5678)"

- **Direct send-notification also supported:**
  - For simple one-way notifications without entitlements
  - No reply codes generated unless SMSAskUser format detected
