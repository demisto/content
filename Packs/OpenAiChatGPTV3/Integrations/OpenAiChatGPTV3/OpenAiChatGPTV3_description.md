## OpenAi ChatGPT V3
- This section explains how to configure the instance of OpenAi ChatGPT V3 in Cortex XSOAR.
- Login to OpenAi API: https://platform.openai.com/account/api-keys
- Click on Create New Secret Key
- Copy the key to the left parameter
- Go to the following link: https://platform.openai.com/docs/api-reference/chat
- Copy the URL after POST command to the left panel "Default Value is: https://api.openai.com/v1/chat/completions"
- Click Test
