
# Palo Alto Networks Threat Vault Pack

## Overview

The Threat Vault API provides Palo Alto Networks customers with an active Advanced Threat Prevention or Threat Prevention subscription with the ability to access threat signature metadata and other pertinent information that's only available in Threat Vault.

## Use Cases

- Retrieve detailed information about specific threat signatures.
- Access the latest threat intelligence data from Palo Alto Networks.
- Enhance threat hunting and incident response capabilities with up-to-date threat metadata.
