
#### Integrations

##### New: Proofpoint Cloud Threat Response

- New: Added a new integration **Proofpoint Cloud Threat Response** that fetches Cloud Threat Response (CTR) incidents into Cortex XSOAR and exposes the following commands (Available from Cortex XSOAR 6.10.0):
  - ***proofpoint-ctr-incidents-list*** - List CTR incidents with optional filters.
  - ***proofpoint-ctr-incident-get*** - Retrieve a specific CTR incident by ID.
