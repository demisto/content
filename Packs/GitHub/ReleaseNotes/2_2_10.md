#### Integrations

##### GitHub

- Updated the ***_github-trigger-workflow_*** command to support workflow runs that return a JSON body (HTTP 200), in addition to the standard 204 No Content response. When run details are returned, they are stored in the `GitHub.WorkflowRun` context with the following fields: `WorkflowRunID`, `RunUrl`, and `HtmlUrl`.
- Added the ***_github-get-workflow-run_*** command to retrieve details of a specific workflow run by its ID. Outputs are stored in `GitHub.WorkflowRun` with fields: `ID`, `Name`, `HeadBranch`, `HeadSha`, `DisplayTitle`, `RunNumber`, `Event`, `Status`, `Conclusion`, `WorkflowID`, `CreatedAt`, `UpdatedAt`, `Url`, and `HtmlUrl`.
