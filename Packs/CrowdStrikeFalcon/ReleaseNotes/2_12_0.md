
#### Mappers

##### CrowdStrike Falcon Mapper

- Updated the **CrowdStrike Falcon Mapper** mapper to support NGSIEM Case mapping. Added **External Category Name** and **External System ID** field mappings.

#### Playbooks

##### CrowdStrike Falcon Malware - Incident Enrichment

- Updated the **CrowdStrike Falcon Malware - Incident Enrichment** playbook to support NGSIEM cases. Replaced the **Get Detections by Incident** sub-playbook with **Get Detections by Case**. Renamed the input from **DetectionOrIncidentID** to **DetectionOrCaseID**. Added integration availability check.

##### New: CrowdStrike Falcon - Get Detections by Case

New: Added the **CrowdStrike Falcon - Get Detections by Case** playbook that enables getting CrowdStrike Falcon detection (alerts) details based on the CrowdStrike case ID.<~XSIAM> (Available from Cortex XSIAM 2.0).</~XSIAM>

##### CrowdStrike Falcon Malware - Investigation and Response

- Updated the **CrowdStrike Falcon Malware - Investigation and Response** playbook to support NGSIEM cases. Updated sub-playbook references and formatting.

##### CrowdStrike Falcon - SIEM ingestion Get Incident Data

- Updated the **CrowdStrike Falcon - SIEM ingestion Get Incident Data** playbook to support NGSIEM cases. Replaced the **cs-falcon-list-incident-summaries** command with the **cs-falcon-list-case-summaries** command. Replaced the **Get Detections by Incident** sub-playbook with **Get Detections by Case**. Added integration availability check.

##### CrowdStrike Falcon - False Positive Incident Handling

- Updated the **CrowdStrike Falcon - False Positive Incident Handling** playbook to support NGSIEM cases. Added case resolution via the **cs-falcon-resolve-case** command and case tagging via the **cs-falcon-add-case-tag** command. Added integration availability check.

##### CrowdStrike Falcon - True Positive Incident Handling

- Updated the **CrowdStrike Falcon - True Positive Incident Handling** playbook to support NGSIEM cases. Added case resolution via the **cs-falcon-resolve-case** command and case tagging via the **cs-falcon-add-case-tag** command.

##### CrowdStrike Falcon - Get Detections by Incident

- Deprecated. Use the **CrowdStrike Falcon - Get Detections by Case** playbook instead.
