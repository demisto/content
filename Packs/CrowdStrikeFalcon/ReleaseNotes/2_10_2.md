##### CrowdStrike Falcon

- Fixed an issue where the **fetch-assets** Spotlight vulnerability flow could cause engine OOM (Out of Memory) by accumulating unbounded fire-and-forget send tasks. Added per-severity semaphore backpressure to limit concurrent pending XSIAM sends to 5 per severity stream.
- Optimized Spotlight state metadata to store AID counts instead of full AID lists, reducing memory footprint and log verbosity.
