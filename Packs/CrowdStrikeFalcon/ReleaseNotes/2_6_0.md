
#### Integrations

##### CrowdStrike Falcon

- Added support for **cs-falcon-add-case-tag** command that adds tags to the specified case.
- Added support for **cs-falcon-get-evidence-for-case** command that get evidence for a specific case.
- Added support for **cs-falcon-delete-case-tag** command that deletes a tag from the specified case.
- Added support for **cs-falcon-list-case-summaries** command that lists case summaries.
- Added support for **cs-falcon-resolve-case** command that resolves or updates a case.
- Deprecated the **cs-falcon-resolve-incident** command. Use the cs-falcon-resolve-case command instead.
- Deprecated the **cs-falcon-list-incident-summaries** command. Use the cs-falcon-list-case-summaries command instead.
- Deprecated the **cs-falcon-get-incident-behavior** command. Use %%% instead.
- Deprecated the **cs-falcon-get-detections-for-incident** command. Use the cs-falcon-get-evidence-for-case command instead.
