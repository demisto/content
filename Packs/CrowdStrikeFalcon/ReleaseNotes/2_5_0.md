#### Integrations

##### CrowdStrike Falcon

- Added support for **CNAPP Alert fetch query** parameter, use this parameter to add a custom query for **CrowdStrike Falcon CNAPP Alert** fetch type.
- Added the **CNAPP Alert** fetch option to the **Fetch types** parameter.
- Added support for fetching **CrowdStrike Falcon CNAPP Alert** incident-type, to obtain alerts of this type, you will need one of the following subscriptions:
    - Falcon Cloud Security with Containers CNAPP
    - Falcon Cloud Security CNAPP
    - Falcon Cloud Security with Containers Runtime Protection
    - Falcon for Managed Containers Runtime Protection
    - Falcon Cloud Security Proactive
    - Falcon Cloud Security with Containers
    - Falcon for Managed Containers
