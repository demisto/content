
#### Integrations

##### CrowdStrike Falcon

- Fixed an issue where the Spotlight asset-enrichment fetch failed an entire batch when CrowdStrike returned a partial-success response (HTTP 400 with valid resources and invalid device IDs). The successfully-resolved devices are now ingested and invalid device IDs are logged and skipped.
- Reduced the Spotlight asset-enrichment batch size to bound the per-request response size and memory usage.
