
#### Integrations

##### TAXII2 Server

- Fixed an issue where the **labels** field in TAXII 2.1 responses for SDO types (*indicator*, *malware*, *report*, *threat-actor*, *tool*) was missing custom tags from the indicator.
- Fixed an issue where the **labels** field in indicator SDO objects (converted from SCO types) only contained a single score-based label and ignored all custom tags assigned to the indicator.
