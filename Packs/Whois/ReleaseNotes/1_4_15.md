#### Integrations
##### Whois
- Added support for Rate Limit handling with **rate_limit_retry_count** instance configuration field, and a command param. Default value: 3 times.
- Added support for Rate Limit handling with **rate_limit_wait_seconds** instance configuration field, and a command param. Default value: 120 seconds.
- Added support for Rate Limit handling with **rate_limit_errors_suppressed** instance configuration field, and a command param. Default value: *False*.
