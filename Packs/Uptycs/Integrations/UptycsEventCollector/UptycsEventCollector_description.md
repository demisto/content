## Uptycs Event Collector

Use this integration to collect events from the Uptycs cloud-native security analytics platform.

To create an API Key:

1. On the **API Client Management** list page, click **Create**.
2. Specify the **Name** and **Description** for the API key.
3. Select the roles who can use this API key.
4. From the **Expiration** drop-down list, select a duration for API expiry. In the drop-down list, in addition to default presets, you also have the option to define a custom time-frame or opt for no expiration.
5. (Optional) Click **+Add** to specify specific IPs to add them to the allow list, including CIDR and optional descriptions. You can add a maximum of 50 CIDRs to both the Global and Client IP Allowlist.
6. Click **Download** the API key or click the **Copy** icon corresponding to the Key and Secret.
