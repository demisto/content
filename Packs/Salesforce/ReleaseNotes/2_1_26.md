
#### Integrations

##### Salesforce

- Added the ***salesforce-list-file-public-links*** command to list all *ContentDistribution* records (public links) for a *ContentDocument* ID.
- The ***salesforce-remove-file-public-links*** command deletes a single *ContentDistribution* record by ID (use with *salesforce-list-file-public-links*).
- Added the ***salesforce-draft-knowledge-article*** command to move a published Knowledge article to Draft using the standard *createDraftFromOnlineKnowledgeArticle* action. By default (*unpublish=true*) the online version is removed from public visibility and an editable draft is created.
- The ***salesforce-archive-knowledge-article*** and ***salesforce-draft-knowledge-article*** commands now automatically clear a lingering draft (Salesforce allows only one draft per article) that would otherwise block the action, and then retry so the online version is reliably archived or unpublished.
- Fixed an issue where a failed ***salesforce-archive-knowledge-article*** command reported an uninformative *null* error instead of the actual Salesforce failure detail.
