
#### Integrations

##### Microsoft 365 Defender

- Added 14 new commands:
  - **Alert Management:**
    - ***microsoft-365-defender-alerts-list*** - List alerts with filters for severity, status, and category.
    - ***microsoft-365-defender-alert-get*** - Get a single alert by ID.
    - ***microsoft-365-defender-alert-update*** - Update alert status, classification, and determination.
    - ***microsoft-365-defender-alert-create-comment*** - Add a comment to an alert.
    - ***microsoft-365-defender-incident-alerts-list*** - List all alerts for a specific incident.
  - **Device Response Actions:**
    - ***microsoft-365-defender-machine-isolate*** - Isolate a device from the network.
    - ***microsoft-365-defender-machine-unisolate*** - Release a device from isolation.
    - ***microsoft-365-defender-machine-scan*** - Initiate an antivirus scan on a device.
    - ***microsoft-365-defender-machine-collect-investigation-package*** - Collect a forensic investigation package.
    - ***microsoft-365-defender-machine-restrict-execution*** - Restrict code execution on a device.
    - ***microsoft-365-defender-machine-unrestrict-execution*** - Remove code execution restrictions.
  - **Threat Indicators:**
    - ***microsoft-365-defender-indicator-create*** - Create a custom threat indicator.
    - ***microsoft-365-defender-indicator-list*** - List custom threat indicators.
    - ***microsoft-365-defender-indicator-delete*** - Delete a threat indicator.
