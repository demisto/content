
#### Integrations

##### Proofpoint Threat Response (Beta)

- Fixed an issue where is case the incidents creation time is far from the alert delivery time the **proofpoint-tr-verify-quarantine** failed th retrieve quarantined emails.
- Added support for *delivery_quarantined_limit* argument in the **proofpoint-tr-verify-quarantine** command. By default, will be set to 120 seconds.
- Added support for *mail_delivered_time* argument in the **proofpoint-tr-verify-quarantine** command. If not provided, this argument will use the value of the *Time* argument.
