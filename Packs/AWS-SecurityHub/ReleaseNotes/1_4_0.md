
#### Integrations

##### New: AWS - Security Hub v2

New: Added a new integration for AWS Security Hub V2, which manages unified, OCSF-formatted security and compliance findings using the AWS Security Hub V2 API.

- Fetches AWS Security Hub V2 findings as Cortex XSOAR incidents, with configurable first fetch time, maximum incidents per fetch, minimum severity, and additional OCSF field filters.
- Added the following commands:
  - ***aws-securityhub-security-hub-enable***
  - ***aws-securityhub-security-hub-disable***
  - ***aws-securityhub-findings-get***
  - ***aws-securityhub-findings-batch-update***
- Added support for incident mirroring between AWS Security Hub and Cortex XSOAR.
  - Incoming: each mirrored finding is re-fetched by its OCSF finding UID and changes (including severity) are reflected in the Cortex XSOAR incident. The mirrored severity reflects the finding's current severity as shown in the AWS Security Hub console.
  - Incoming: when a finding is set to *Resolved* or *Suppressed* in AWS Security Hub, the corresponding Cortex XSOAR incident is closed. When it returns to *New* or *In Progress*, the incident is reopened.
  - Outgoing: severity, status, and comment changes made in Cortex XSOAR are pushed back to the finding in AWS Security Hub. Changing the incident's severity (including the built-in **Severity** field) is mirrored to the finding's severity in AWS Security Hub.
- Added the **Incident Mirroring Direction** parameter.
- Added the **Resolve finding of closed incident from XSOAR in AWS Security Hub** parameter.

#### Mappers

##### New: AWS Security Hub v2 - Incoming Mapper

New: Added a new incoming mapper that maps OCSF-formatted findings to Cortex XSOAR incident fields for incoming mirroring, including finding details, severity, status, resources, remediation, and compliance information.

##### New: AWS Security Hub v2 - Outgoing Mapper

New: Added a new outgoing mapper that maps Cortex XSOAR incident fields (severity, status, and comment) to AWS Security Hub V2 finding fields for outgoing mirroring.
