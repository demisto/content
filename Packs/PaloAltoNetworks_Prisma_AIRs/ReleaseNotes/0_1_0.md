
#### Integrations

##### Palo Alto Networks - Prisma AIRs AI Security

Initial release of the Prisma AIRs AI Security integration, providing 100 commands across the Prisma AIRs portfolio.

- **Runtime AI Security**
  - Scan single prompts/responses against security profiles for AI threats, including prompt injection, toxic content, topic violations, DLP, URL categories, and malicious code (***prisma-airs-runtime-scan***).
  - Bulk scanning with CSV input and session grouping (***prisma-airs-runtime-bulk-scan***).
  - Query runtime scan logs over a time window (***prisma-airs-runtime-scan-logs***).
- **Security Profiles & Custom Topics**
  - Full CRUD for runtime security profiles (***prisma-airs-runtime-profiles-list/-get/-create/-update/-delete***).
  - Custom topic guardrail CRUD plus apply-to-profile (***prisma-airs-runtime-topics-list/-get/-create/-update/-delete/-apply***).
- **API Keys, Customer Apps & Deployment Profiles**
  - Runtime API key lifecycle (***prisma-airs-runtime-api-keys-list/-create/-regenerate/-delete***).
  - Customer application management, consumption, and violation breakdowns (***prisma-airs-runtime-customer-apps-list/-get/-update/-consumption/-violations/-delete***).
  - List deployment profiles (***prisma-airs-runtime-deployment-profiles-list***).
- **DLP Configuration (DLP v2 API)**
  - Data profiles (***prisma-airs-runtime-dlp-profiles-list/-get/-create/-patch/-replace/-delete***).
  - Data patterns (***prisma-airs-runtime-dlp-patterns-list/-get/-create/-patch/-replace/-delete***).
  - Dictionaries (***prisma-airs-runtime-dlp-dictionaries-list/-get/-create/-patch/-replace/-delete***).
  - Filtering profiles (***prisma-airs-runtime-dlp-filtering-profiles-list/-get/-replace***).
- **Model Security**
  - Security group, rule, and rule-instance management for model supply-chain sources (***prisma-airs-model-security-groups-***, ***prisma-airs-model-security-rules-***, ***prisma-airs-model-security-rule-instances-***).
  - Model scan creation, retrieval, evaluations, violations, files, and labels (***prisma-airs-model-security-scans-***, ***prisma-airs-model-security-labels-***).
- **AI Red Teaming**
  - Target CRUD and probing (***prisma-airs-redteam-targets-***).
  - Static, dynamic, and custom scans with status, abort, categories, and reporting (***prisma-airs-redteam-scan-create***, ***prisma-airs-redteam-scans-list***, ***prisma-airs-redteam-scan-get/-abort***, ***prisma-airs-redteam-categories-list***, ***prisma-airs-redteam-report-get***).
  - EULA management and custom prompt/prompt-set management with CSV upload (***prisma-airs-redteam-eula-***, ***prisma-airs-redteam-prompts-***, ***prisma-airs-redteam-prompt-sets-***).
- **Features**
  - Dual authentication: OAuth2 (Strata Cloud Manager management, DLP, model security, and red teaming) and a Runtime API Key (regional Scanner API).
  - Configurable regional Scanner and DLP base URLs.

- **Supported Modules:** cloud_runtime_security, xsiam, cloud
- **Categories:** Cloud Security
