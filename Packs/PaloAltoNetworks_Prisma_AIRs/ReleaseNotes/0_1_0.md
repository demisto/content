
#### Integrations

##### Palo Alto Networks - Prisma AIRs AI Security

- Initial release of the Prisma AIRs AI Security integration.
- **New Commands:**
  - **prisma-airs-runtime-scan**: Scan a single prompt against a security profile for AI security threats.
  - **prisma-airs-runtime-profiles-list**: List all runtime security profiles.
- **Features:**
  - OAuth2 authentication with Strata Cloud Manager
  - Runtime AI security scanning capabilities
  - Security profile management
  - Detection of topic violations, prompt injection, toxic content, and DLP violations
- **Supported Modules:** cloud_runtime_security, xsiam, cloud
- **Categories:** Cloud Security, CI/CD
