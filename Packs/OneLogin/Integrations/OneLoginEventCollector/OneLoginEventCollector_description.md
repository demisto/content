To start fetch events from OneLogin API you need to get an API Credential Pair (Client ID and Client Secret) from the OneLogin UI.

### Create an OneLogin API Credential Pair
To create the Client ID and Client Secret in the OneLogin UI:
1. Access OneLogin as an account owner or administrator.
2. Go to **Developers** > **API Credentials**.
3. On the API Access page, click **New Credential**.
4. Select a scope for the credentials. (Authentication Only, Read Users, Manage users, etc.)
5. Click **Save**.

For more information, see the [OneLogin documentation](https://developers.onelogin.com/api-docs/1/getting-started/working-with-api-credentials).
