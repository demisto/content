To start fetch events from OneLogin API you need to get an API Credential Pair (Client ID and Clien Secret) from the OneLogin UI, to do this please see the following section.

### Creating an OneLogin API Credential Pair
Creating the Client ID and Client Secret in the OneLogin UI:
1. Access OneLogin as an account owner or administrator.
2. Go to Developers > API Credentials.
3. On the API Access page, click New Credential.
4. Select a scope for the credentials. (Authentication Only, Read Users, Manage users etc.)
5. Click Save.

For more information please see the [OneLogin documentation](https://developers.onelogin.com/api-docs/1/getting-started/working-with-api-credentials).
