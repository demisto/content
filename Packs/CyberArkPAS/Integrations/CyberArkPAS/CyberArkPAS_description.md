## CyberArk PAS Integration Desription
- Use this integration to create and list CyberArk accounts. This integration was integrated and tested with version 11.4 of CyberArk
  Privileged Account Security.
- We currently support CyberArk Authentication, so you should use a local CyberArk Admin account, if the authentication is successful we refresh the token generated by the API every 5 minutes.
