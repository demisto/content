#### Integrations

##### New: CyberArk Identity Security Platform

- New integration for CyberArk Identity Security Platform (ISP) that provides audit event streaming capabilities.
- Supports OAuth2 Client Credentials authentication flow with API key.
- Fetches audit events from CyberArk Identity Security Platform for XSIAM ingestion.
- Includes cursor-based pagination for efficient event retrieval.
- Features automatic event deduplication using UUID tracking.
- Available Commands:
  - ***cyberark-isp-get-events***: Retrieve audit events with flexible date range filtering.
