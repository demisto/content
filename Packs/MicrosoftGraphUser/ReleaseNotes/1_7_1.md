
#### Integrations

##### Entra ID Users

- Added support for MFA flow to support Entra ID based CAP.
    - Added The **Allow secret generators commands execution** parameter - checking this box will allow running commands that generate and print secrets. make sure to add restrictions to the related commands. for more information please refer to the following guide  <mfa guide>.
    - Added the ***msgraph-user-create-mfa-client-access-token*** command - Issue a new access token for the MFA app.
    - Added the ***msgraph-user-create-mfa-client-secret*** command - Issue a new client secret for the MFA app.
    - Added the ***msgraph-user-request-mfa*** command - pops a synchronous MFA request for the given user. This is a blocking call that waits for user response or timeout.
    - Added the ***msgraph-user-get-user-default-auth-method*** command - retrieves the authentication preferences for a user, including the default method.
- Updated the Docker image to: *demisto/crypto:1.0.0.6932158*.
