
#### Integrations

##### Azure Active Directory Users

- Added support for **msgraph-user-tap-policy-list** command that this command will only return a single object in the collection 
as a user can have only one temporary access pass method.
- Added support for **msgraph-user-tap-policy-delete** command that this command will delete a specific temporary access pass policy.
- Added support for **msgraph-user-tap-policy-create** command that create a new temporary access pass authentication method object on a user. 
a user can only have one temporary access pass that's usable within its specified lifetime. .
- Updated the Docker image to: *demisto/crypto:1.0.0.3174650*.
