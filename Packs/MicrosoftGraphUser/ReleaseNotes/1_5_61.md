
#### Integrations

##### Entra ID Users

- Added support for **msgraph-user-get-owned-devices** command that retrieves the properties from the owned devices of a user.
- Added support for **msgraph-user-get-auth-methods** command that retrieves the properties from the owned devices of a user.
- Added support for **msgraph-user-get-groups** command that retrieves the groups a user is part of.
- Added support for **msgraph-user-validate-password** command that validates whether a password is valid according to the password policies.
- Added support for **msgraph-user-get-by-sam** command that retrieves the properties and relationships of a user object by its onpremisessamaccountname. for more information, visit: https://docs.microsoft.com/enus/graph/api/userupdate?view=graphrest1.0).
permissions:  user.read (delegated)  user.read.all (application).
