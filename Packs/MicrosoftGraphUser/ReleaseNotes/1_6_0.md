
#### Integrations

##### Entra ID Users

- Added support for **msgraph-user-get-groups** command that retrieves the groups a user is part of.
- Added support for **msgraph-user-get-owned-devices** command that retrieves the properties of the owned devices of a user.
- Added support for **msgraph-user-get-auth-methods** command that retrieves a list of authentication methods registered to a user.
