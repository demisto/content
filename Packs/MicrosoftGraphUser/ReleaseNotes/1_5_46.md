#### Integrations

##### Azure Active Directory Users

- Added the ***msgraph-user-tap-policy-create*** command that creates a new Temporary Access Pass (TAP) policy for a user.
- Added the ***msgraph-user-tap-policy-delete*** command that deletes a specific TAP policy.
- Added the ***msgraph-user-tap-policy-list*** command that lists all TAP policies for a user.
- Aligned the integration commands to the latest content standards.
