#### Breaking Changes

##### Cyble Threat Intel
- Removed TAXII-based commands: `cyble-vision-fetch-taxii` and `cyble-vision-get-collection-names`.
- Removed all TAXII ingestion logic. The integration now uses REST API endpoints for fetching IOCs.
- Updated command outputs and internal logic to match the new REST API model.

#### Enhancements

##### Cyble Threat Intel
- Added new lookup command for querying indicators 
- Improved IOC feed performance and reliability by switching to REST-based ingestion.
