
#### Integrations

##### CohesityHelios

- Updated the ransomware alerts API from v1 to v2 (`/v2/mcm/alerts`).
- Updated the alert filtering to use *alertName* instead of *alertCode*.
- Updated the ***cohesity-helios-ignore-anomalous-object*** and ***cohesity-helios-restore-latest-clean-snapshot*** commands to use *alert_id* instead of *object_name*.
- Improved the implementation of the API request wrapper with debug logging and improved error handling.
- Updated the integration docker image to **demisto/python3:3.12.13.9022922**.
- Updated the command outputs with context fields: *cluster_id*, *cluster_name*, *entity_id*, and *job_id*.
- Updated the command outputs by removing unused context fields: *anomalous_object_name*, *anomalous_object_env*, and *anomaly_strength*.

#### Incident Fields

- Updated the **Cohesity Helios Alert Id** incident field and set it as unsearchable.

#### Mappers

##### Cohesity Helios Incoming mapper

#### Layouts
