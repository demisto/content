
#### Integrations

##### Rapid7 InsightVM

- Deleted the **Long running integration** parameter.
- Added support for **Assets Fetch Interval** parameter that the interval in hours between each fetch of assets. default is 24 hours.

##### Rapid7 Nexpose

- **Breaking Change**: Replaced the long-running-execution command with the `fetch-assets` command for asset and vulnerability collection. The integration now uses the platform's built-in fetch-assets scheduling mechanism instead of a persistent long-running container.
- Added the *Assets Fetch Interval* parameter to configure the interval between asset fetches (default: 24 hours).
- Updated the Docker image to **demisto/fastapi:0.125.0.7178097**.
