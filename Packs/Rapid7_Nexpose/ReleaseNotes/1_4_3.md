#### Integrations

##### Rapid7 InsightVM

- Fixed an issue where large **fetch-assets** runs could fail with *FATAL ERROR during streaming or sending events* (aiohttp `TransferEncodingError` / connection timeouts) because the integration opened too many concurrent connections to the XSIAM ingestion endpoint. The number of concurrent send requests is now capped, transient transport errors are retried with backoff, and an explicit request timeout was added.
