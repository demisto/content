## ANY.RUN Cloud Sandbox
This section explains how to configure the instance of ANY.RUN Cloud Sandbox in Cortex XSOAR.  
API-KEY from your ANY.RUN account is required.

## Generate API token
* Follow [ANY.RUN Sandbox](https://app.any.run/)
* Profile > API and Limits > Generate > Copy


## Add instance
1. Navigate to **Settings** > **Integrations** > **Servers & Services**.
2. Search for ANY.RUN.
3. Click **Add instance** to create and configure a new integration instance.
4. Insert ANY.RUN API-KEY into the **Password** parameter.
5. Click **Test** to validate the URLs, token, and connection.


| **Parameter**    | **Description**                                                                                                                                                | **Required** |
|------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------| --- |
| Password         | ANY.RUN API-KEY without prefix                                                                                                                               | True | | True |

## Note:
Please use "ANY.RUN" as username.
