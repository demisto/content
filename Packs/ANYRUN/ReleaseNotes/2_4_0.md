
#### Integrations

##### ANY.RUN Cloud Sandbox

- New: Added the **Root URL** configuration parameter (default: *any.run*), used to build the interactive analysis link and IOC descriptions instead of a hardcoded *app.any.run* host. Falls back safely to the default when left blank.
- Updated: Removed the *obj_ext_browser* argument from the **anyrun-detonate-url-android** command. This option is not supported by ANY.RUN's Android sandbox API and caused detonation to fail.
- Updated: The **anyrun-download-analysis-sample** command no longer returns both the pcap and the file sample when *download_type* is *pcap*; the branches are now mutually exclusive.
- Updated: The **anyrun-get-analysis-report** command with *report_format=summary* now returns the report as a downloaded JSON file (*anyrun_report_\<task_uuid\>.json*) instead of a JSON context output.
- Updated the Docker image to: *demisto/anyrun-sdk:1.0.0.10440656*.
- Documentation: Fixed a typo in the setup instructions.

##### ANY.RUN TI Feed

- New: Indicators now include *tags* (from feed labels) and a *publications* field populated from the feed's external references. Domain indicators additionally populate *communitynotes* with the same reference links.
- New: Added the **Incremental Feed** configuration parameter (enabled by default) so that indicators absent from a given delta fetch are no longer incorrectly flagged as "Removed from feed".
- Updated the Docker image to: *demisto/anyrun-sdk:1.0.0.10440656*.

##### ANY.RUN TI Lookup

- Updated the Docker image to: *demisto/anyrun-sdk:1.0.0.10440656*.

#### Playbooks

##### ANYRUN Detonate Url Android

- Updated: Removed the *obj_ext_browser* input, task argument, and input section. Android detonation does not support browser selection via the API, and passing this argument caused Android URL detonation to fail.
- Updated the playbook inputs: *opt_timeout* default changed from *60* to *120*.

##### ANYRUN Detonate Url Linux

- Fixed the *brandname* value used by the "Is there an active ANY.RUN instance?" condition task, from *ANY.RUN Sandbox* to *ANY.RUN Cloud Sandbox*, matching the integration's actual brand name.

##### ANYRUN Detonate Url Windows

- Updated the playbook inputs: *opt_timeout* default changed from *60* to *120*.
