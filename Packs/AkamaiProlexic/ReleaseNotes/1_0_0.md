#### Integrations

##### Akamai Prolexic

- New: Added the **Akamai Prolexic** integration. Collects DDoS detection critical events and general events from the Akamai Prolexic Analytics API and forwards them to Cortex XSIAM as the ``akamai_prolexic_raw`` dataset (vendor=``akamai``, product=``prolexic``).
