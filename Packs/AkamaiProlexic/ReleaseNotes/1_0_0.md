#### Integrations

##### Akamai Prolexic Event Collector

- New: Added the **Akamai Prolexic Event Collector** integration. Collects DDoS detection critical events and general events from the Akamai Prolexic Analytics API and forwards them to Cortex XSIAM as the ``akamai_prolexic_raw`` dataset (vendor=``akamai``, product=``prolexic``).
