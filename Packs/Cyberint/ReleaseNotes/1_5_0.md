
#### Integrations

##### Check Point EM ThreatCloud Intelligence Feed

- Updated the integration display name from **Check Point EM Premium Feed** to **Check Point EM ThreatCloud Intelligence Feed**. The integration ID and the *service* value written to indicators in TIM remain *Cyberint Premium Feed*, so existing instances and indicator filters continue to work without reconfiguration.

##### Check Point EM Feed

- Added the ***cyberint-cve-enrich*** command, which enriches one or more CVE identifiers with Check Point Exposure Management vulnerability intelligence (EPSS, CVSS, CWE, the Cyberint CVE score and active-exploitation evidence).
- Added the ***cyberint-credential-leak-lookup*** command, which looks up leaked employee or customer credentials associated with a company domain.

#### Playbooks

##### Check Point - IOC Enrichment and Triage

- New: Enriches IOC entities (IP, domain, URL, file hash) found in an incident with Check Point Exposure Management threat intelligence and applies triage decision logic, escalating the incident when a malicious indicator is found (Available from Cortex XSOAR 6.10.0).

##### Check Point - Credential Leak Validation and Response

- New: Validates and responds to leaked employee or customer credentials reported by Check Point Exposure Management, driving an automated or semi-automated remediation flow followed by user and SOC notification (Available from Cortex XSOAR 6.10.0).

##### Check Point - Vulnerability Exploitation Monitoring

- New: Enriches CVEs from a vulnerability-management incident with Check Point Exposure Management vulnerability intelligence and prioritizes them based on real-world exploitation (Available from Cortex XSOAR 6.10.0).

##### Check Point - Phishing Takedown

- New: Automates or semi-automates the takedown of high-confidence phishing websites detected by Check Point Exposure Management, including takedown submission and status polling (Available from Cortex XSOAR 6.10.0).
