
#### Integrations

##### CheckPoint Firewall v2

- Added support for **checkpoint-access-section-delete** command that deletes the specified access section.
- Added support for **checkpoint-access-section-get** command that shows an existing access section using object name or uid.
- Added support for **checkpoint-access-section-add** command that adds a new access section.
- Added support for **checkpoint-service-group-add** command that adds a new service group object.
- Added support for **checkpoint-service-group-get** command that shows an existing service group object using object name or uid.
- Added support for **checkpoint-service-group-update** command that updates the specified service group object.
- Added support for **checkpoint-service-group-list** command that gets a list of all service group objects.
- Added support for **checkpoint-service-group-delete** command that deletes the specified service group object.
- Added support for **checkpoint-access-section-update** command that updates the specified access section.
- Added support for **checkpoint-service-group-clone** command that clones an existing service group object.
