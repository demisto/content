#### Integrations

##### Google Cloud Platform

- Added support for installing the **GCP** content pack in the **Cortex XSOAR** and **Cortex XSIAM** Marketplaces (with version < 3.0).
- Added the following configuration parameters used when running the integration outside the Cortex Cloud or Cortex XSIAM (version >= 3.0) connector flow:
  - *Service Account Private Key (JSON)* - the full content of a GCP Service Account private key JSON file (includes guidance on creating the service account and generating the key).
  - *GCP Project ID* - the default GCP project ID to authenticate against.
- Updated the authentication flow to resolve GCP credentials from the integration parameters in Cortex XSOAR or Cortex XSIAM, while keeping the Cortex Cloud or Cortex XSIAM (version >= 3.0) connector flow unaffected.
- Updated the *Test* functionality to call the Cloud Resource Manager `testIamPermissions` API for Cortex XSOAR / Cortex XSIAM (version < 3.0).
- Added support for the *Use system proxy settings* and *Trust any certificate (not secure)* parameters for Cortex XSOAR / Cortex XSIAM (version < 3.0).
