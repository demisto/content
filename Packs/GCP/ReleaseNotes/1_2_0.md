#### Integrations

##### Google Cloud Platform

- Added support for **Cortex XSOAR** and **Cortex XSIAM** (version < 3.0) Marketplace deployments. The integration can now be configured with a GCP Service Account private key JSON and a default *GCP Project ID*.
- Updated the *Service Account Private Key (JSON)* parameter description to clarify that it is required for Cortex XSOAR and Cortex XSIAM (version < 3.0), and optional for Cortex Cloud or Cortex XSIAM (version >= 3.0).
- Added the *GCP Project ID* integration parameter, used by the **Test** button for Cortex XSOAR and Cortex XSIAM (version < 3.0) to validate credentials.
- Updated the **Test** button to call the Cloud Resource Manager `testIamPermissions` API directly when running on Cortex XSOAR or Cortex XSIAM (version < 3.0), validating the provided service account credentials are functional.
- Updated the integration description to cover the supported platforms.
- Updated the *project_id* arguments in all commands to make the argument optional on Cortex XSOAR and Cortex XSIAM (version < 3.0), while remaining required on Cortex Cloud or Cortex XSIAM (version >= 3.0).
- Added support for the *Use system proxy settings* and *Trust any certificate (not secure)* parameters when running on Cortex XSOAR and Cortex XSIAM (version < 3.0).
