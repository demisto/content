- Added support for the following quick actions:
  - **Disable GCP Firewall Rule**, disabling default GCP firewall rules that are overly permissive.
  - **Remove GCP Bucket AllUsers Access**, revoking 'allUsers' permission access in GCP Storage buckets.
  - **Remove GCP Bucket AllAuthenticatedUser Access**, revoking 'allAuthenticatedUsers' permission access in GCP Storage buckets.
  - **Make GCP Bucket Private**, revoking both 'allUsers' and 'allAuthenticatedUsers' access in a GCP Storage bucket.
  - **Enable GCP Subnet Flow Logs**, enabling VPC Flow Logs for a GCP subnet.
  - **Enable Private IP Access on Subnet**, enabling Private Google Access for a GCP subnet.
  - **Enable GKE Cluster Intra-Node Visibility**, enabling intra-node visibility on a GKE cluster.
  - **Enable Master Authorized Networks on GKE**, enabling Master Authorized Networks on a GKE cluster.
  - **Enable GCP Bucket Versioning**, enabling versioning on a GCP Storage bucket.
  - **Set GCP Bucket Access to Uniform**, enabling uniform bucket-level access on a GCP Storage bucket.
