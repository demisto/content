Use the GitHub MCP integration to connect securely with a GitHub Model Context Protocol (MCP) server and access its tools in real time.

## What does this pack do

This integration provides tools primarily intended for use with Agentix Actions. It enables you to:

- Retrieve a list of available tools in the GitHub MCP server.
- Call a specific tool on the GitHub MCP server with optional input parameters.
