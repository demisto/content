
#### Scripts

##### SpecTestScriptLargeMemory

- New: Test script **with** the `spec` field set to `L` (large memory allocation).
- Validates the new SDK `spec` field for dedicated worker memory configuration on Scripts.
- Supports operations: `transform`, `analyze`, `aggregate`.

##### SpecTestScriptDefault

- New: Test script **without** the `spec` field (uses platform default worker memory).
- Serves as a baseline for validating that the absence of the `spec` field on Scripts is handled correctly.
- Supports operations: `echo`, `reverse`, `count`.
