
#### Integrations

##### Spec Test Large Memory

- New: Test integration **with** the `spec` field set to `L` (large memory allocation). Used to validate the new SDK `spec` field for dedicated worker memory configuration.

##### Spec Test Default

- New: Test integration **without** the `spec` field. Uses the platform's default worker memory allocation. Serves as a baseline for validating that the absence of the `spec` field is handled correctly.
