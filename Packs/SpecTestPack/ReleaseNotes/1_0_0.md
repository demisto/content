
#### Integrations

##### Spec Test Large Memory

- New: Test integration **with** the `spec` field set to `L` (large memory allocation).
- Validates the new SDK `spec` field for dedicated worker memory configuration.
- Includes commands: `spec-test-large-process-resource`, `spec-test-large-list-resources`.

##### Spec Test Default

- New: Test integration **without** the `spec` field (uses platform default worker memory).
- Serves as a baseline for validating that the absence of the `spec` field is handled correctly.
- Includes commands: `spec-test-default-get-item`, `spec-test-default-list-items`.
