Splits a list into group.

e.g. [1,2,3,4] -> ["1,2", "3,4"].

## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | Utilities |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| input | An input list. |
| chunk | Size of each group. |
| type | Group as list or string. |
| delimiter | Separate elements using delimiter (If type is String). |

## Outputs

---

| **Path** | **Description** |
| --- | --- |
| Group | The list after grouping. |
