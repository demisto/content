
#### Scripts

##### New: MakePair

- This transformer will create a list of dictionary by aggregating elements from two arrays.
The one is given by `value` (with `array1_key`), another is given by `array2` (with `array2_key`). (Available from Cortex XSOAR 6.8.0).
##### New: SortBy

- This transformer will sort an array of dictionary values by keys in ascending or descending order. (Available from Cortex XSOAR 6.8.0).
##### New: DedupBy

- This transformer will remove elements of the array that contain an identical combination of values for the keys given. (Available from Cortex XSOAR 6.8.0).
