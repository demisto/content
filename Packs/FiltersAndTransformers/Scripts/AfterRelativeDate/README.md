Checks the given datetime has occurred after the provided relative time.

## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | date, filter, Condition |
| Cortex XSOAR Version | 5.0.0 |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| left | Date value to check - Can be any non-timezone aware time format. ex. "2020-01-01T00:00:00" |
| right | Relative time ex. "6 months ago" |

## Outputs

---
There are no outputs for this script.
