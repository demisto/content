
#### Integrations

##### Orca

- New command `orca-set-alert-severity`. Change alert severity.
- New command `orca-get-alert-event-log`. Get alert event log.
- New command `orca-set-alert-status`. Change alert status.
- New command `orca-verify-alert`. Run alert verification.
- New command `orca-download-malicious-file`. Download alert malicious file if file exists.
- Updated the Docker image to: *demisto/python3:3.10.11.54132*.
