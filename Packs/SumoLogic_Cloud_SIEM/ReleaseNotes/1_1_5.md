#### Integrations
##### Sumo Logic Cloud SIEM
Added the followings:

###### Incident Types
1. Sumo Logic Insight 
- New incident type corresponding to Sumo Logic Cloud SIEM Insight

###### Incident Fields
1. Sumo Logic Insight Signals
- **Sumo Logic Insight Signals** field corresponding to all Sumo Logic Cloud SIEM Signals contributing to the Insight
2. Sumo Logic Insight Entity
- **Sumo Logic Insight Entity** field corresponding to the Sumo Logic Cloud SIEM Entity associated with the Insight 
3. Sumo Logic Insight Involved Entities
- **Sumo Logic Insight Involved Entities** field corresponding to all entities involved in the Insight

###### Layouts
1. Sumo Logic Insight
- New layout for Sumo Logic Insights (Available from Cortex XSOAR 6.0.0).

###### Mappers
1. Sumo Logic Insight - Incoming Mapper
- New incoming mapper for Sumo Logic Insight (Available from Cortex XSOAR 6.0.0).
