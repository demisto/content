## Get an API Token
1. Log in to IllusionBLACK.
2. Go to the **Orchestrate** tab under the **API Tokens** page.
3. Click **ADD**. 
4. Enter the necessary details and select all checkboxes for the permissions.
5. After creating the token, click the **edit** button and copy the ***X-CLIENT-AUTH***  and ***X-CLIENT-ID*** fields to use in Demisto.
