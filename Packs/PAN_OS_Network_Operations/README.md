PAN-OS Network Operations is a collection of content that automates many common tasks that PAN-OS NGFW administrators face, while also presenting an interface for displaying NGFW and Panorama details directly in XSOAR.

## Use Cases

 - Automating Large batches of firewall upgrades
 - Running tests before and after configuration changes or software and content updates
 - Visualizing the PAN-OS NGFW operational details and state
 - Checking and fixing configuration issues
