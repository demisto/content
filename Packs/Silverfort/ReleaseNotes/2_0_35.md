
#### Integrations

##### Silverfort

- Added support for **External API Key** parameter that for silverfort 5.3 and above, you must include the external api key for external calls.
- Added support for **Operational API Key** parameter that for silverfort 5.3 and above, you must include the operational api key for calls using sam_account/domain instead of upn. (format is "operational user id:operational user secret").
