## Silverfort Integration
- In order to use this integration fill Server URL &  API KEY arguments 
- To generate an API token:
    1. From the Silverfort Admin Console, navigate to Settings > Advanced.
    2. In the Authentication Tokens section, click Generate Token.
    3. Copy the generated token and save it in a safe place.
    
For more information please click [here](https://support.silverfort.com/hc/en-us/articles/360035867373-Silverfort-Risk-Engine-API-Reference?flash_digest=8587c2d918370791570535f5fa2d14df25498437)
