### Broker VM
You will need to use the information described [here](https://docs-cortex.paloaltonetworks.com/r/Cortex-XDR/Cortex-XDR-Pro-Administrator-Guide/Configure-the-Broker-VM).\
You can configure the specific vendor and product for this instance.

1. Go to CONFIGURATION > Full Configuration > Box > Infrastructure Services > Syslog Streaming.
2. Click Lock.
3. Set Enable Syslog Streaming to yes.
4. Click Send Changes and Activate.

More details, please see [here](https://campus.barracuda.com/product/cloudgenfirewall/doc/96026562/how-to-configure-syslog-streaming/)
