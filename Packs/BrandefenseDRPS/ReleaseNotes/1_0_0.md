
#### Integrations
##### Brandefense Digital Risk Protection Services
- Initial release of the Brandefense DRPS integration for Cortex XSOAR.
- **23 commands** across 7 categories: Reputation/Enrichment, Incident Management, Intelligence, Threat Intelligence, Asset & Risk Management, Operational, and Indicator Management.
- Fetch incidents and intelligence reports with timestamp + code-based deduplication.
- IoC enrichment commands (`ip`, `domain`, `url`, `file`) with **DBotScore** and standard indicator contexts for auto-enrichment.
- IoC caching with 6-hour TTL and up to 5,000 entries to reduce redundant API calls.
- **brandefense_get_indicators** command to retrieve Consolidated Data and Incident indicators by type (leak, phishing_site, credit_card, cve, social_media, sensitive_file_disclosure, malicious-file, malicious_ads).
- **brandefense_get_ioc_list** command to consolidate all IoC types from the last N days into a single list.
- **brandefense_create_confirmed_phishing** command to create confirmed phishing address incidents.
- **brandefense_takedown_request** command to request takedown for phishing URLs.
- **time_range** dropdown added to 6 commands (incidents, intelligence, assets, compromised devices, audit logs, indicators) for easy date filtering without manual date entry.
- Threat search via Brandefense CTI platform with polling support.
- Incident management: view details, change status, retrieve indicators and related incidents.
- Intelligence reports with filtering by category, keyword, and time period.
- Asset monitoring, compromised device detection, and domain risk assessments.
- Audit log retrieval with filtering.
- Built-in rate limiting: 300ms throttle between API calls, pagination, and configurable fetch limits.

#### Classifiers
##### Brandefense - Classifier
- Routes Brandefense events to the correct incident type (Incident vs Intelligence) based on the `brandefense_type` field.

##### Brandefense - Incoming Mapper
- Maps Brandefense API fields to XSOAR incident fields for both Brandefense Incident (17 fields) and Brandefense Intelligence (12 fields) types.

#### Incident Types
- **Brandefense Incident** — Security incidents from the Brandefense platform.
- **Brandefense Intelligence** — Threat intelligence reports from Brandefense.

#### Incident Fields
- 16 custom incident fields covering incident metadata, intelligence metadata, shared fields (reference URL, indicators, MITRE tactics).
