# Pack Documentation
This pack is used to access Recorded Future Identity data.

## Integration
The integration is used to access the data from the API.

### Available Actions
* Identity data actions
    * Using the new Recorded Future Identity module.
    * Available actions: recordedfuture-identity-lookup; recordedfuture-identity-search.

## Playbooks
All the playbooks are meant to be used as sub-playbooks to get Identities data.
### Available sub-playbooks
* Recorded Future External Usecase Playbook
* Recorded Future Workforce Usecase Playbook
