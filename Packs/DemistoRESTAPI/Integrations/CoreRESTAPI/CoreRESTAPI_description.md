Provide the API KEY by following the instructions:  
<~XSIAM>
Creating API keys is done in the Cortex XSIAM interface, under Settings -> Configurations -> API Keys  
</~XSIAM>
<~XSOAR>
Creating API keys is done in the Cortex XSOAR interface, under Settings -> Integrations -> API Keys  
</~XSOAR>

Provide Auth ID only when using <~XSIAM> Cortex XSIAM </~XSIAM> <~XSOAR> Cortex XSOAR 8.0.0 and above. If the version is less than 8.0, you can leave it empty </~XSOAR>.
The Auth ID can be found next to the generated API Key in the "API Keys" page.
