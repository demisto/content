#### Integrations

##### Rubrik Security Cloud

- Added support for fetching **Sensitive Data Object** as a Cortex XSOAR incident.
- Added support for the *Sensitive Data Object Types* parameter to filter **Sensitive Data Object** using object types.
- Added support for the *Sensitive Data Object Sensitivity Levels* parameter to filter **Sensitive Data Object** using sensitivity levels.
- Added support for the ***rubrik-sensitive-data-object-get*** command.
- Added support for the ***rubrik-sensitive-data-object-file-get*** command.
