Generate an as built document, as HTML, based on the running XSOAR instance. Requires an instance of the Demisto API integration configured.

## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| author | Who has prepared this as-built. |
| customer | Whom this as-built was prepared for. |
| days | Days back to search for incidents. |
| size | Maximum number of results to return - default 1000. |
| playbook | Retrieve more information such as dependencies from a specific use case. For multiple playbooks enter comma-separated values. |

## Outputs

---
There are no outputs for this script.
