## [20.2.4] - 2020-02-25
#### New Playbook
This playbook queries indicators based on a pre-defined query and adds the found indicators to a ArcSight Active list. To query indicators, go to playbook inputs, choose "from indicators" and set your query. for example type:ip. Your Active List (resource) ID should also be defined in the playbook inputs.
