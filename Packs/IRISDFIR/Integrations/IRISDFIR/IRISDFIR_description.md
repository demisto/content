## IRIS
How to set up API Keys in IRIS web console.


**Creating an API Key**
1. Go to your IRIS web console, on left side panel, click Advanced > Access Control and Add User.
2. Recomended to create the user as service account, for that when creating the user click on "Use as service account".
3. Click save.
4. Click on the created user and note the API key.
5. Use the API to configure the IRIS integration.
