## Symantec Cloud Secure Web Gateway Event Collector
Use this integration to collect operation logs from Symantec Cloud Secure Web Gateway.

In order to use this integration, you need to enter your Symantec Cloud Secure Web Gateway credentials in the relevant integration instance parameters.


### To obtain API Key:
Create a User API Key that serves as the username and password in the Sync API syntax.

1. In Service Mode, select **Account Maintenance** > **MDM, API Keys**.
2. Create a username/password key.
3. In the API section (bottom half of the screen), click **Add API Key**.
4. Define a Username and Password. The username must be unique as it used by the Web Security Service. If the Web Security Service detects any other API with the same name, the service displays an error message and you must define another name.
5. Click **Add**.
