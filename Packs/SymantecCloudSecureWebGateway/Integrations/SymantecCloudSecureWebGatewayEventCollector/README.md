Palo Alto Networks Symantec Cloud Secure Web Gateway Event Collector integration for XSIAM.

## Configure Symantec Cloud Secure Web Gateway Event Collector in Cortex


| **Parameter** | **Required** |
| --- | --- |
| Server URL | True |
| User name | True |
| Password | True |
| Trust any certificate (not secure) | False |
| Use system proxy settings | False |
| Fetch interval in seconds. The minimum threshold is 30 seconds. | True |
