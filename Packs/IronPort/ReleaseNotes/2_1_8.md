
#### Integrations

##### Cisco ESA

- Added support for the ***cisco-esa-message-filter-list*** command that retrieves all message filters or a specific message filter by name.
- Added support for the ***cisco-esa-message-filter-create*** command that creates a new message filter.
- Added support for the ***cisco-esa-message-filter-update*** command that updates an existing message filter.
- Added support for the ***cisco-esa-message-filter-delete*** command that deletes a message filter.
- Updated the Docker image to: *demisto/python3:3.12.13.9059085*.
