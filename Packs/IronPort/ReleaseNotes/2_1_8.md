
#### Integrations

##### Cisco ESA

- Added support for **cisco-esa-message-filter-list** command that retrieve all message filters or a specific message filter by name.
- Added support for **cisco-esa-message-filter-create** command that create a new message filter.
- Added support for **cisco-esa-message-filter-update** command that update an existing message filter.
- Added support for **cisco-esa-message-filter-delete** command that delete a message filter.
- Updated the Docker image to: *demisto/python3:3.12.13.9059085*.
