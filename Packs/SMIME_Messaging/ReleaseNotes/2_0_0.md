
#### Integrations

##### SMIME Messaging
- Updated the Docker image to: *demisto/m2crypto:1.0.0.106628*.

- Command ***smime-sign-and-encrypt*** updated to allow:
  - Sign only / Encrypt only modes.
  - Multiple recipients (to, cc, bcc).
  - Adding attachments.
- Added support for html messages.
- Updated crypto algorithms in **smime-sign-and-encrypt**
  - Encryption now using AES256.
  - Signature now using SHA256.
- Removed a string decode test case with undefined expected result
- Command ***smime-encrypt-email-body*** is deprecated. Use ***smime-sign-and-encrypt*** instead.
- Command ***smime-sign-email*** is deprecated. Use ***smime-sign-and-encrypt*** instead.
