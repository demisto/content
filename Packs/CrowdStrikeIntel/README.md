CrowdStrike Falcon Intel v2 is designed to enhance cybersecurity defenses by utilizing CrowdStrike's technical threat intelligence feed. This integration allows users to perform various actions and commands directly from within Cortex to assess and mitigate potential threats.

## What Does This Pack Do?

- Run reputation commands on files, URLs, domains, and IP addresses.
- Search for threat actors by various parameters like name, description, activity dates, motivations.
- Search for Indicators of Compromise (IoCs) such as file hashes, URLs, domains, and IP addresses.
- Search for intelligence reports based on criteria such as name, associated actors, target industries, and creation dates.
