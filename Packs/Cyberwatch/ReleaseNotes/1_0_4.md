#### Integrations

##### Cyberwatch
- Added commands : 
- cyberwatch-list-sysadmin-assets : Get a list of Sysadmin assets (/assets/servers).
- cyberwatch-fetch-sysadmin-asset : Fetch details on one Sysadmin asset.
- cyberwatch-list-compliance-assets : Get a list of Compliance assets (/compliance/assets).
- cyberwatch-fetch-compliance-asset : Fetch details on one Compliance asset.
- cyberwatch-send-declarative-data-asset : Upload Declarative Data for an asset.
- cyberwatch-get-declarative-data-asset : Retrieve Declarative Data for an asset.
