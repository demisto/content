#### Scripts

##### SearchCases

- Adding support to variety of filtering options and updated some context outputs:
  - Added new arguments to filter cases:
    - **case_domain**
    - **severity**
    - **asset_ids**
    - **asset_groups**
    - **hosts**
    - **tags**
    - **assignee**
    - **case_name**
    - **case_description**
    - **modification_start_time**
    - **modification_end_time**

  - The following arguments have changed their name:
    - start_time -> creation_start_time
    - end_time -> creation_end_time

  - Expanded outputs:
    - Added new outputs:
      - **Core.Case.asset_accounts**
      - **Core.Case.asset_group_ids**
      - **Core.Case.asset_ids**
      - **Core.Case.asset_names**
      - **Core.Case.asset_providers**
      - **Core.Case.asset_regions**
      - **Core.Case.asset_types**

  - The following outputs have been removed and are now retrievable using **get-enriched-case-data** argument set to true:
    - **Core.Case.detection_time**
    - **Core.Case.notes**
    - **Core.Case.xdr_url**
    - **Core.Case.manual_description**
