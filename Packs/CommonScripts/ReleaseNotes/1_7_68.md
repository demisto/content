
#### Scripts
##### New: ConcatFormat
- Returns a string concatenated with given prefix & suffix which supports DT expressions. (Available from Cortex XSOAR 6.5.0).
##### New: FormatTemplate
- Build a text from a template which can include DT expressions. (Available from Cortex XSOAR 6.5.0).
##### New: SetWithTemplate
- Set a value built by a template in context under the key you entered. (Available from Cortex XSOAR 6.5.0).
