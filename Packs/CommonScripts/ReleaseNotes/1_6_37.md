
#### Scripts
##### New: KeyValueToJson
- This script will take in an array of values and an array of keys. It will create a JSON dictionary using the key and value pairs based on their indicies. For example; value=[1, 2, 3] key=['a', 'b', 'c'] will return {'a': 1, 'b':2, 'c': 3}. (Available from Cortex XSOAR 5.5.0).
