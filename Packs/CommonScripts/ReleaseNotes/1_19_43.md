##### URLSSLVerification

    - Check for SSL Certificate for all redirect chain.
        - if http redirect to https with certificate verified=True.
        - if all chain is http return verified=False. 
    - set_http_as_suspicious updated logics.
        - if set_http_as_suspicious='true' and url is http DbotScore = Suspicious.
        - if set_http_as_suspicious='true' and url is https DbotScore = Unknown.
