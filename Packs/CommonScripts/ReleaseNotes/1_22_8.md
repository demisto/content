
#### Scripts

##### AppendindicatorFieldWrapper

- Fixed an issue where adding tags to URL-type indicators whose values contain commas (e.g. `http://example.com/path?a=1,2,3`) failed with an "Indicator not found" error. The script now calls `appendIndicatorField` once per indicator value instead of passing all values together, preventing the server from incorrectly splitting URL values on commas. (XSUP-57677)
