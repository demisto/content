Generates a configuration file for Cortex XSOAR As-Built document.
A Json file is generated that can be downloaded and used to generate the As-Built document for Cortex XSOAR.

## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| playbook | Retrieve information such as dependencies from a specific use case. Fetches all the custom data configured for the specific playbook. For multiple playbooks enter comma-separated values. |

## Outputs

---
There are no outputs for this script.
