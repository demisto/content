## Script Data

---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | filter |
| Cortex XSOAR Version | 5.5.0 |

## Inputs

---

| **Argument Name** | **Description** |
| --- | --- |
| right | The prefix to set as maximum. |
| left | A comma-separated list of CIDR to validate \(IPv4 or IPv6\). |
