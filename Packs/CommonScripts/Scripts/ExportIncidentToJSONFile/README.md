This automation exports the content from the incident and context to JSON and return the resulting JSON file to the war room.

## Script Data
---

| **Name** | **Description** |
| --- | --- |
| Script Type | python3 |
| Tags | Utility |

## Inputs
---

| **Argument Name** | **Description**                                                                      |
|-------------------|--------------------------------------------------------------------------------------|
| filename | Name of the file to create in the war room, defaults to incident.json.               |
| type | Defines the type of the export, defaults to export Incident to json without context. |

## Outputs
---

There are no outputs for this script.
