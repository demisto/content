# DocuSign

DocuSign is the world's leading electronic signature and digital transaction management platform. Use this pack to fetch Customer events and User data logs from DocuSign for threat detection and compliance monitoring in Cortex XSIAM.

## What does this pack do?

The DocuSign Event Collector integration provides comprehensive event collection capabilities for DocuSign:

- **Customer events Collection**
- **User data Collection**

- `Docusign` - Event Collector integration
