
#### Integrations

##### ThreatConnect Feed

- Added support for the **Group Query** parameter that filters indicators using ThreatConnect Query Language (TQL).
- Added support for the **Use Indicator Query For Group** parameter that when checked, uses the query from the **Indicator Query** parameter for group filtering. Default is "true".
- Updated the Docker image to: *demisto/python3:3.12.12.5490952*.
