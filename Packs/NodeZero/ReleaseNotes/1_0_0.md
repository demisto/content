
#### Integrations

##### NodeZero

New: **NodeZero** integration for the Horizon3.ai autonomous penetration testing platform.
- Added support for fetching HIGH and CRITICAL severity weaknesses as incidents.
- Added automatic pagination for large result sets.
- Added deduplication to prevent duplicate incident creation.

#### Incident Types

##### NodeZero Weakness

New: **NodeZero Weakness** incident type for tracking discovered vulnerabilities.

#### Incident Fields

- New: **NodeZero Attack Paths Count** - Number of attack paths associated with the weakness.
- New: **NodeZero CISA KEV** - Indicates if the vulnerability is in CISA's Known Exploited Vulnerabilities catalog.
- New: **NodeZero Operation ID** - The pentest operation that discovered the weakness.
- New: **NodeZero Ransomware Use** - Indicates known ransomware campaign usage.
- New: **NodeZero Weakness Category** - Category classification of the weakness.
- New: **NodeZero Weakness Proven** - Indicates if the weakness has proof of exploitability.
- New: **NodeZero Weakness Score** - Severity score of the weakness.

#### Classifiers

##### NodeZero - Classifier

New: **NodeZero - Classifier** for categorizing fetched weaknesses.

#### Mappers

##### NodeZero - Incoming Mapper

New: **NodeZero - Incoming Mapper** for mapping weakness data to incident fields.
