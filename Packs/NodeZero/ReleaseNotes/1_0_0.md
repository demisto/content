
#### Integrations

##### NodeZero

New: **NodeZero** integration for the Horizon3.ai autonomous penetration testing platform.
- Added support for fetching HIGH and CRITICAL severity weaknesses as incidents.
- Added automatic pagination for large result sets.
- Added deduplication to prevent duplicate incident creation.

#### Incident Types

##### NodeZero Weakness

New: **NodeZero Weakness** incident type for tracking discovered vulnerabilities.

#### Incident Fields

##### NodeZero Attack Paths Count

- New: Number of attack paths associated with the weakness.

##### NodeZero CISA KEV

- New: Indicates if the vulnerability is in CISA's Known Exploited Vulnerabilities catalog.

##### NodeZero Operation ID

- New: The pentest operation that discovered the weakness.

##### NodeZero Ransomware Use

- New: Indicates known ransomware campaign usage.

##### NodeZero Weakness Category

- New: Category classification of the weakness.

##### NodeZero Weakness Proven

- New: Indicates if the weakness has proof of exploitability.

##### NodeZero Weakness Score

- New: Severity score of the weakness.

#### Classifiers

##### NodeZero - Classifier

New: **NodeZero - Classifier** for categorizing fetched weaknesses.

#### Mappers

##### NodeZero - Incoming Mapper

New: **NodeZero - Incoming Mapper** for mapping weakness data to incident fields.
