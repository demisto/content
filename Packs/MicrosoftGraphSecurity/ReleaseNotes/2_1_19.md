#### Integrations
##### Microsoft Graph Security

- Updated the Docker image to: *demisto/crypto:1.0.0.54987*.
- Added the command **msg-create-alert-comment**, use this command to create comments for the V2 alerts.
- Added the **MS graph security version** integration param, this param will decide the version of the api to use. Note: the default version is V2 which causes breaking changes and demands reconfiguring with new credentials.
- Added the **Fetch incidents of the given service sources only.** integration param, use this param to filter fetched alerts by service source when using V2 of the API.
- Added support for V2 of the API for the following commands: **msg-search-alerts**, **msg-get-alert-details**, **msg-update-alert**. For more information, please refer to the integration readme.
- Note: changed the display name of the following integration params: **First fetch time range** and **How many incidents to fetch each time** to **First fetch timestamp (<number> <time unit>, e.g., 12 hours, 7 days)** and **Max incidents per fetch** respectively.

#### Mappers

##### New: Graph Security Alert
- Added a mapper **Graph Security Alert**, use this mapper to map incidents fetched from the V2 of the api.
