
#### Integrations

##### Microsoft Graph Security

- Added support for **msg-list-ediscovery-case-hold-policy** command that get a list of the ediscovery hold policy objects and their properties.
- Added support for **msg-export-result-ediscovery-data** command that export results from an estimated ediscovery search.
- Added support for **msg-create-ediscovery-case-hold-policy** command that create a new ediscovery hold policy object.
- Added support for **msg-delete-ediscovery-case-hold-policy** command that delete an ediscovery hold policy object.
- Added support for **msg-update-ediscovery-case-policy** command that update the properties of an ediscovery hold policy object.
- Added support for **msg-list-case-operation** command that get a list of the case operation objects and their properties.
