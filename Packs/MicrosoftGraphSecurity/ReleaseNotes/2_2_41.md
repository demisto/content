
#### Integrations

##### Microsoft Graph Security

- Added support for **msg-get-last-estimate-statistics-operation** command that retrieves the most recent ediscovery estimate statistics operation for a given search in microsoft purview. use this command after running 'msg-run-estimates-tatistics' to check the results.
- Added support for **msg-run-estimate-statistics** command that starts an ediscovery estimate statistics operation in microsoft purview. the operation calculates the size and number of items matching the search query.
- Added support for **msg-start-ediscovery-report** command, which starts an eDiscovery export report for a given case and search in Microsoft Purview and returns an operation URL for polling.
- Added support for **msg-get-ediscovery-report** command, which checks the status of an existing eDiscovery export report and downloads it if ready.
- Updated the documentation for the **msg-purge-ediscovery-data** command.
- Fixed an issue where the following commands failed with exception:
  - *msg-auth-test*
  - *msg-list-threat-assessment-requests*
