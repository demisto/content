
#### Integrations

##### Microsoft Graph Security

- Added support for **msg-get-last-estimate-statistics-operation** command that retrieves the most recent ediscovery estimate statistics operation for a given search in microsoft purview. use this command after running 'msgrunestimatestatistics' to check the results.
- Added support for **msg-run-estimate-statistics** command that starts an ediscovery estimate statistics operation in microsoft purview. the operation calculates the size and number of items matching the search query.
