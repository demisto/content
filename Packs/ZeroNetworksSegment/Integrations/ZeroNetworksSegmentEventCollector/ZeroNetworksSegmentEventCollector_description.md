### Zero Networks Segment Event Collector
To interact with the Zero Networks API, you need to generate an API key. Follow the steps below to create and configure your API token:

### Generating an API Key

#### 1. **Login to the Zero Networks Portal**

Go to [portal.zeronetworks.com](https://portal.zeronetworks.com).

Log in using your email address.

Authenticate using the code sent to your email.

#### 2. **Navigate to API Key Settings**

Go to `Settings` → `System` → `Integrations` → `API`.

Click on `Add new token`.

#### 3. **Fill in the Required Fields**

***Token Name:*** Provide a name for your token.

***Access Type:*** Choose one of the following:
- **Read only:** Allows only GET API requests.
- **Full access:** Allows GET, PUT, POST, and DELETE API requests.

***Expiry:*** Set the token expiration period (1-36 months).

#### 4. **Create and Copy the Token**

Click `Add` to create the token.

Copy the token immediately. **Note:** The token will not be displayed again after this step.

If needed, the token can be regenerated at any time.
