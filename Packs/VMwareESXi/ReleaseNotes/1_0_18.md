#### Modeling Rules

##### VMware ESXi Modeling Rule

- Updated the crond process-spawn mapping (ESXi command-execution sub-type) to set **target process name** to the basename of the executed program and **target process path** to its full path when present.
- Set **event outcome** to *UNKNOWN* for crond process-start events (the log carries no success/failure result).
- Set **event name** to *process_start* for crond process-start events.
