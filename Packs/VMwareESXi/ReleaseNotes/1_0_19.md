
#### Modeling Rules

##### VMware ESXi Modeling Rule

Improved implementation of the extraction logic for ESXi SSH (*sshd-session*), interactive shell (*shell*), and kernel process-spawn (*vmkernel ... started from ... with cmdline*) command-execution logs.

#### Parsing Rules

##### VMware ESXi Parsing Rule

Fixed an issue where the syslog header capture (RFC 3164 and RFC 5424 branches) did not reliably split *hostname*, *process*, and *message* for ESXi command-execution log lines.
