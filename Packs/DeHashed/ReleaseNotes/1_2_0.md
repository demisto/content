
#### Integrations

##### DeHashed

- Updated the integration to support DeHashed API v2.
- Updated the Docker image to: *demisto/fastapi:0.125.0.8198656*.
- **Breaking change**: Removed the following context output paths:
  - `DeHashed.Search.Vin`
  - `DeHashed.Search.ObtainedFrom`
- **Breaking change**: Removed support for the **contains** value of the *operation* argument in the ***dehashed-search*** command. To perform a contains-style search, use the **regex** operator (or wildcards in the value) instead.
