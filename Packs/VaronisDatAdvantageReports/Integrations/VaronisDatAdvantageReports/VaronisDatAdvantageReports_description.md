## Varonis DatAdvantage Reports
### This section explains how to configure the instance of Varonis DatAdvantage Reports  in Cortex XSOAR:
- You're required to have a Varonis user/service account to authenticate with NTLM. This account requires permissions to read and use the ReportAPI within the Data Security Platform. 
- This integration uses v1 authentication. Ensure that the ReportAPI is enabled within the DSP server and v1 Authentication is enabled.
