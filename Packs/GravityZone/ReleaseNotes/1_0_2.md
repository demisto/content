
#### Integrations

##### GravityZone

- Added support for **gz-endpoint-create-memory-dump** command that creates an endpoint memory dump task. when polling is true or not specified, waits for completion and returns the download url. when polling is false, returns the task id immediately.
- Added support for **gz-endpoint-memory-dump-status** command that checks the status of an endpoint memory dump task using a task id returned by gzendpointcreatememorydump. polling is enabled by default (when not specified).
- Added support for **gz-endpoint-users-loggedin** command that retrieves active loggedin sessions for an endpoint.
- Added support for **gz-poll-endpoint-users-loggedin-status** command that checks endpoint active sessions task status. this command is not intended for direct use.
