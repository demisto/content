
#### Integrations

##### GravityZone

- Added the **gz-endpoint-create-memory-dump** command, which initiates a memory dump on the specified endpoint. Once the dump is successfully generated, the resulting archive is uploaded to the Bitdefender cloud. Important: This feature can be enabled upon request by contacting Bitdefender Enterprise Support.
- Added the **gz-endpoint-memory-dump-status** command, which verifies the status of a task initiated via ***gz-endpoint-create-memory-dump***.
- Added the **gz-endpoint-users-loggedin** command, which retrieves details of all active user sessions on a specific managed endpoint.
- Added the **gz-poll-endpoint-users-loggedin-status** command, which verifies the status of a task initiated via **gz-endpoint-users-loggedin**. This command is invoked automatically and is not available for direct use.
