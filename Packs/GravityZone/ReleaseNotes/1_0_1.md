
#### Integrations

##### GravityZone

- Added support for **gz-endpoint-users-loggedin** command that retrieves active logged-in sessions for an endpoint.
- Added support for **gz-poll-endpoint-users-loggedin-status** command that checks endpoint active sessions task status. This command is not intended for direct use.
- Added support for **gz-endpoint-create-memory-dump** command that creates an endpoint memory dump task and returns the download url when it becomes available.
- Added support for **gz-poll-endpoint-memory-dump-status** command that checks endpoint memory dump task status. This command is not intended for direct use.
