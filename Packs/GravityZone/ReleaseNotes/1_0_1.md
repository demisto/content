
#### Integrations

##### GravityZone

- Added support for **gz-endpoint-create-memory-dump** command that creates an endpoint memory dump task. When polling is true or not specified, waits for completion and returns the download URL. When polling is false, only returns the task id.
- Added support for **gz-endpoint-memory-dump-status** command that checks the status of an endpoint memory dump task using the task id returned by gz-endpoint-create-memory-dump.
- Added support for **gz-endpoint-users-loggedin** command that retrieves active logged-in sessions for an endpoint.
- Added support for **gz-poll-endpoint-users-loggedin-status** command that checks endpoint active sessions task status. This command is not intended for direct use.
