
#### Integrations

##### GravityZone

- Updated the ***gz-endpoint-list*** command to recursively list endpoints across all managed companies.
- Added FQDN, Company ID, and Group ID fields to the ***gz-endpoint-list*** command results.
- Added `name` and `node_id` filters to the ***gz-endpoint-list*** command (`node_id` is the company or group node in the GravityZone network tree).
