# Cisco ETD Connector

## Configure Cisco ETD Connector on Cortex XSIAM

1. Navigate to Settings → Configurations → Integrations.
2. Search for CiscoETDConnector.
3. Click Add Instance.
4. Configure:
   - ETD API Base URL
   - Client ID
   - Client Secret
   - API Key
   - First Fetch Time
5. Enable Fetch Events.
6. Click Test.
7. Save & Enable.
