
#### Integrations

##### OpenCTI

- Added support for **opencti-malware-create** command that creates a new malware.
- Added support for **opencti-intrusion-set-create** command that creates a new intrusionset.
- Added support for **opencti-grouping-create** command that creates a new grouping.
- Added support for **opencti-incident-response-create** command that creates new incident response.
- Added support for **opencti-vulnerability-create** command that creates a new vulnerability.
- Added support for **opencti-add-object-to-container** command that add object to container (report, grouping, case-rft, case-rfi, case-incident).
- Added support for **opencti-campaign-create** command that creates a new campaign.
- Added support for **opencti-report-create** command that creates a new report.
- Added support for **opencti-threat-actor-group-create** command that creates a new threat actor group.
- Updated the Docker image to: *demisto/opencti:1.0.0.7499058* for pycti==6.9.0 compatibility.
