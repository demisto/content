
#### Integrations

##### Flashpoint Ignite

- Added support for **flashpoint-ignite-vulnerability-get** command that retrieves detailed information about a specific vulnerability by its Flashpoint ID.
- Added support for **cve** command that retrieves detailed information about a specific CVE by its CVE ID.
- Added support for **flashpoint-ignite-vulnerability-list** command that list vulnerabilities using provided filters.
- Added support for **flashpoint-ignite-product-list** command that list products using provided filters.
- Added support for **flashpoint-ignite-vulnerability-package-list** command that retrieves a list of packages that are affected by a particular vulnerability.
- Added support for **flashpoint-ignite-vendor-list** command that list vendors using provided filters.
- Added support for **flashpoint-ignite-vulnerability-library-list** command that retrieves a list of libraries that are affected by a particular vulnerability.
