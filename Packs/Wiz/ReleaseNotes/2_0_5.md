#### Integrations

##### Wiz

- Fixed an issue where the ***wiz-get-issues*** command and incident fetching would hang indefinitely on heavy queries (e.g., `issue_type=TOXIC_COMBINATION`). Wiz GraphQL API and authentication calls are now bounded by a 115-second request timeout, allowing the integration to raise an actionable error instead of being abruptly terminated by the 5-minute Docker script timeout.

- Updated the Docker image to: *demisto/python3:3.12.13.9059085*.
