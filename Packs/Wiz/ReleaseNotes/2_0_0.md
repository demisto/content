## WizDefend Integration v2.0.0

### New Features
- Added WizDefend integration for security detections and threat management
- Support for fetching and filtering detections by various criteria
- Support for retrieving and managing threats
- Incident creation from Wiz detections

### Commands
- **wiz-get-detections**: Retrieve security detections with filtering options
- **wiz-get-detection**: Get specific detection by ID  
- **wiz-get-threats**: Retrieve threats with filtering options
- **wiz-get-threat**: Get specific threat by issue ID

### Configuration
- Configurable detection severity levels
- Platform and origin filtering
- Cloud account/organization filtering
- Customizable fetch intervals and limits
