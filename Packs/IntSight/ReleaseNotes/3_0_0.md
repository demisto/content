
#### Integrations

##### New: Rapid7 - Threat Command (IntSight)

- Rapid7 Insight - Threat Command allows managing alerts, CVEs, IOCs, and assets by accounts and MSSP accounts. (Available from Cortex XSOAR 6.8.0).

##### IntSights (Deprecated)

- Deprecated. Use Rapid7 Threat Command instead.
