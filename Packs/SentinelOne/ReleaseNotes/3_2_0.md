##### SentinelOne v2

- sentinelone-get-alerts - changed created_until mandatory field to optional
- sentinelone-get-alerts - updated created_from and created_until command arguments to accept date inputs like "10 days", "2 hours","5 months"
- sentinelone-get-threats - updated created_after, created_before, created_from and created_until command arguments to accept date inputs like "10 days", "2 hours","5 months"
