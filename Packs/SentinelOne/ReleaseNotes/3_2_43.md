#### Integrations

##### SentinelOne v2

- Improved ***Add-hash-to-blocklist*** and ***remove-hash-from-blocklist*** commands to support multiple scopes.
- Enhanced ***get-agent-request*** command to support multiple agent IDs.
- Command arguments accept account_ids, site_ids, and group_ids.
- When site IDs are provided in the command, they are combined with the ***block_site_ids***.
