#### Integrations

##### SentinelOne Activity and Alerts

- Added **EXTERNAL_URL** field to all events for easy navigation to the SentinelOne console:
  - **Threats**: Links to `{base_url}/incidents/threats/{threat-id}/overview`
  - **Alerts**: Links to `{base_url}/incidents/alerts/{alert-id}/overview`
  - **Activities**: Links to `{base_url}/activity`
