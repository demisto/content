#### Integrations

##### Amazon Web Services
- Updated the Docker image to: *demisto/boto3py3:1.0.0.8187750*.

- Added support for the following commands:  
  - ***aws-logs-log-group-create*** - to create a CloudWatch Logs log group.
  - ***aws-logs-log-stream-create*** - to create a log stream for a specified log group.
  - ***aws-logs-log-group-delete*** - to delete a log group and permanently delete all archived log events associated with it.
  - ***aws-logs-log-stream-delete*** - to delete a log stream and permanently delete all archived log events associated with it.
  - ***aws-logs-log-events-filter*** - to list log events from a specified log group, with support for filtering by pattern, time range, and log stream.
  - ***aws-logs-log-groups-describe*** - to list log groups with optional filtering by prefix, pattern, class, and cross-account querying.
  - ***aws-logs-log-streams-describe*** - to list log streams for a specified log group with optional filtering and ordering.
  - ***aws-logs-retention-policy-put*** - to set the retention policy for a log group.
  - ***aws-logs-retention-policy-delete*** - to delete the retention policy for a log group.
  - ***aws-logs-log-event-put*** - to upload a log event to a specified log stream.
  - ***aws-logs-metric-filter-put*** - to create or update a metric filter associated with a log group.
  - ***aws-logs-metric-filter-delete*** - to delete a metric filter.
  - ***aws-logs-metric-filters-describe*** - to list metric filters with optional filtering by log group, prefix, metric name, or namespace.
  - ***aws-ssm-commands-list*** - that returns a list of commands requested by users.
  - ***aws-ssm-command-cancel*** - that cancels a specified SSM command and polls until the cancellation is confirmed.
  - ***aws-ssm-tag-add*** - that adds or overwrites one or more tags for a specified SSM resource.
  - ***aws-ssm-tag-remove*** - that removes tag keys from a specified SSM resource.
  - ***aws-ssm-tags-list*** - that returns a list of the tags assigned to a specified SSM resource.
  - ***aws-ssm-inventory-list*** - that retrieves AWS SSM inventory information for managed nodes.
  - ***aws-ssm-associations-list*** - that returns all State Manager associations in the current AWS account and Region.
  - ***aws-ssm-association-get*** - that describes the association for a specified target or managed node.
  - ***aws-ssm-association-versions-list*** - that retrieves all versions of an association for a specific association ID.
  - ***aws-ssm-documents-list*** - that returns all SSM documents in the current AWS account and Region.
  - ***aws-ssm-document-describe*** - that describes a specified SSM document.
  - ***aws-ssm-automation-executions-list*** - that provides details about all active and terminated Automation executions.
  - ***aws-ssm-automation-execution-run*** - that initiates an automation runbook execution and polls until it reaches a terminal state.
  - ***aws-ssm-automation-execution-cancel*** - that stops a running Automation execution and polls until the cancellation is confirmed.
