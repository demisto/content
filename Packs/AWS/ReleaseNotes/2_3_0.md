#### Integrations

##### Amazon Web Services
- Updated the Docker image to: *demisto/boto3py3:1.0.0.8187750*.

- Added support for the ***aws-logs-log-group-create*** command to create a CloudWatch Logs log group.
- Added support for the ***aws-logs-log-stream-create*** command to create a log stream for a specified log group.
- Added support for the ***aws-logs-log-group-delete*** command to delete a log group and permanently delete all archived log events associated with it.
- Added support for the ***aws-logs-log-stream-delete*** command to delete a log stream and permanently delete all archived log events associated with it.
- Added support for the ***aws-logs-log-events-filter*** command to list log events from a specified log group, with support for filtering by pattern, time range, and log stream.
- Added support for the ***aws-logs-log-groups-describe*** command to list log groups with optional filtering by prefix, pattern, class, and cross-account querying.
- Added support for the ***aws-logs-log-streams-describe*** command to list log streams for a specified log group with optional filtering and ordering.
- Added support for the ***aws-logs-retention-policy-put*** command to set the retention policy for a log group.
- Added support for the ***aws-logs-retention-policy-delete*** command to delete the retention policy for a log group.
- Added support for the ***aws-logs-log-event-put*** command to upload log event to a specified log stream.
- Added support for the ***aws-logs-metric-filter-put*** command to create or update a metric filter associated with a log group.
- Added support for the ***aws-logs-metric-filter-delete*** command to delete a metric filter.
- Added support for the ***aws-logs-metric-filters-describe*** command to list metric filters with optional filtering by log group, prefix, metric name, or namespace.
