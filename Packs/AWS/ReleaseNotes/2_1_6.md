#### Integrations

##### Amazon Web Services
- Added support for **aws-ec2-network-interface-attribute-modify** command that modifies the specified network interface attribute.  
- Added support for **aws-ec2-regions-describe** command that describes the Regions that are enabled for your account, or all Regions.  
- Added support for **aws-s3-bucket-create** command that creates a new S3 bucket.  
- Added support for **aws-s3-buckets-list** command that returns a list of all buckets owned by the authenticated sender of the request.  
- Added support for **aws-ssm-inventory-entries-list** command that returns a list of inventory items.  
- Added support for **aws-ssm-command-run** command that runs commands on one or more managed nodes.
