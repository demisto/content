#### Integrations

##### Amazon Web Services

- Added support for the following commands:
  - ***aws-network-firewall-resource-policy-delete*** - that deletes a resource policy that you created in an aws-network-firewall-resource-policy-put request.
  - ***aws-network-firewall-resource-policy-put*** - that creates or updates an AWS Identity and Access Management policy for your rule group or firewall policy.
  - ***aws-network-firewall-resource-policy-describe*** - that retrieves a resource policy that you created in an aws-network-firewall-resource-policy-put request.
  - ***aws-network-firewall-tags-for-resource-list*** - that retrieves the tags associated with the specified resource.
  - ***aws-network-firewall-resource-tag*** - that adds the specified tags to the specified resource.
  - ***aws-network-firewall-resource-untag*** - that removes the tags with the specified keys from the specified resource.
  - ***aws-network-firewall-logging-configuration-describe*** - that returns the logging configuration for the specified firewall.
  - ***aws-network-firewall-logging-configuration-update*** - that sets the logging configuration for the specified firewall.
