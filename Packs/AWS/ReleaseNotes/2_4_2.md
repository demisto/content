#### Integrations

##### Amazon Web Services

- Added support for **aws-network-firewall-subnet-change-protection-update** command that modifies the flag, SubnetChangeProtection, which indicates whether it is possible to change the subnets that the firewall is associated with. required permissions: network-firewall:UpdateSubnetChangeProtection.
- Added support for **aws-network-firewall-subnets-disassociate** command that removes the specified subnet associations from the firewall. This removes the firewall endpoints from the subnets and removes any network filtering protections that the endpoints were providing. required permissions: network-firewall:DisassociateSubnets.
- Added support for **aws-network-firewall-subnets-associate** command that associates the specified subnets in the Amazon VPC to the firewall. You can specify one subnet for each of the Availability Zones that the VPC spans. This is used to provide the firewall endpoints for the subnets. required permissions: network-firewall:AssociateSubnets.
