
#### Integrations

##### Amazon Web Services

- Added support for the ***Enable RDS Event Subscription*** quick action that enables RDS event notification subscription.
- Added support for the ***Enforce SSL Only on Bucket*** quick action that enforces SSL only access on the S3 bucket by applying a bucket policy that denies all non https requests.
- Added support for the ***Set S3 ACL to BucketOwnerEnforced*** quick action that sets the S3 bucket ACL ownership to bucket owner enforced.
