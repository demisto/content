
#### Integrations

##### Amazon Web Services

- Added support for ***Enable RDS Event Subscription*** Quick Action that enables the RDS event notification subscription.
- Added support for ***Enforce SSL Only on Bucket*** Quick Action that enforces SSL only access on the S3 bucket by applying a bucket policy that denies all non https requests.
- Added support for ***Set S3 ACL to BucketOwnerEnforced*** Quick Action that sets the S3 bucket ACL ownership to bucket owner enforced.
