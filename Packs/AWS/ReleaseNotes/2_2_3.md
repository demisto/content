#### Integrations

##### Amazon Web Services

- Added support for **Enable IMDSv2** quick action, which enables instance metadata services v2.0 for the EC2 instance.
- Added support for **Block S3 Public Access** quick action, which blocks all public access to the S3 bucket.
- Added support for **Enable Control Plane Logging** quick action, which enables control plane logging for an EKS cluster.
- Added support for **Set EKS cluster to Private / Disable Public Access** quick action, which disables public access to the EKS cluster's API endpoint, making it only accessible from within the cluster's VPC.
