#### Integrations

##### Amazon Web Services

- Added support for **aws-ec2-enable-imdsv2-quick-action**, which enables instance metadata services v2.0 for the EC2 instance.
- Added support for **aws-s3-public-access-block-quick-action**, which blocks all public access to the S3 bucket.
- Added support for **aws-eks-enable-control-plane-logging-quick-action**, which enables control plane logging for an EKS cluster.
- Added support for **aws-eks-disable-public-access-quick-action**, which disables public access to the EKS cluster's API endpoint, making it only accessible from within the cluster's VPC.
