#### Integrations

##### Amazon Web Services

- Added support for the following commands:
  - ***aws-network-firewall-subnets-associate*** - that associates the specified subnets in the Amazon VPC to the firewall. Multiple subnets can be provided through the *subnet_mappings* argument in the form 'SubnetId=<id>,IPAddressType=<type>;SubnetId=<id>'.
  - ***aws-network-firewall-subnets-disassociate*** - that removes the specified subnet associations from the firewall.
  - ***aws-network-firewall-subnet-change-protection-update*** - that modifies the SubnetChangeProtection flag, which indicates whether it is possible to change the subnets that the firewall is associated with.
