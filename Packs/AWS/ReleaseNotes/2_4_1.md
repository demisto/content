#### Integrations

##### Amazon Web Services
- Added support for the following commands:
  - ***aws-network-firewall-rule-group-create*** - to create a rule group. 
  - ***aws-network-firewall-rule-group-delete*** - to delete a rule group.
  - ***aws-network-firewall-rule-group-describe*** - to returns the data objects for the specified rule group. 
  - ***aws-network-firewall-rule-groups-list*** - to retrieves the metadata for the rule groups.
  - ***aws-network-firewall-rule-group-update*** - to updates the rule settings for the specified rule group.
