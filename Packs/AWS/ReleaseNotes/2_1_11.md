
#### Integrations

##### Amazon Web Services

- Added support for **aws-iam-user-policy-put-quick-action** command that blocks all actions for an iam user by attaching a "deny all" inline policy.
- Added support for **aws-iam-login-profile-delete-quick-action** command that removes console access for an iam user by deleting their login profile (password).
- Added support for **aws-rds-event-subscription-modify-quick-action** command that enables the rds event notification subscription.
- Added support for **aws-s3-bucket-policy-put-quick-action** command that enforces sslonly access on the s3 bucket by applying a bucket policy that denies all nonhttps requests.
- Added support for **aws-iam-access-key-update-quick-action** command that deactivates the iam user's access keys by setting their status to inactive.
- Added support for **aws-s3-bucket-ownership-controls-put-quick-action** command that sets the s3 bucket acl ownership to bucketownerenforced, disabling acls for the bucket.
- Added support for **aws-iam-role-from-instance-profile-remove-quick-action** command that detaches the iam role from an ec2 instance by removing the instance profile association.
