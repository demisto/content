
#### Integrations

##### Amazon Web Services

- Added support for ***Enable RDS Event Subscription* - aws-rds-event-subscription-modify-quick-action** command that enables the rds event notification subscription.
- Added support for ***Enforce SSL Only on Bucket* - aws-s3-bucket-policy-put-quick-action** command that enforces sslonly access on the s3 bucket by applying a bucket policy that denies all nonhttps requests.
- Added support for ***Set S3 ACL to BucketOwnerEnforced* - aws-s3-bucket-ownership-controls-put-quick-action** command that sets the s3 bucket acl ownership to bucketownerenforced, disabling acls for the bucket.
