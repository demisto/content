#### Integrations

##### AWS - EC2

- Added the following commands:
  - **aws-ec2-addresses-describe** - to describes one or more Elastic IP addresses.
  - **aws-ec2-address-allocate** - to allocates an Elastic IP address to your AWS account.
  - **aws-ec2-address-associate** - to associates an Elastic IP address with an instance or network interface.
  - **aws-ec2-address-disassociate** - to disassociates an Elastic IP address from an instance or network interface.
  - **aws-ec2-address-release** - to releases an Elastic IP address back to the IP address pool.
