
#### Integrations

##### Amazon Web Services

- Added the following commands:
  - ***aws-cloudtrail-trails-describe***.
  - ***aws-s3-delete-bucket-website***: Removes the website configuration for a bucket.
  - ***aws-rds-event-subscription-modify***: Modifies an existing RDS event notification subscription.
  - ***aws-s3-bucket-ownership-controls-put***: Creates or modifies OwnershipControls for an Amazon S3 bucket.
  - ***aws-ec2-subnet-attribute-modify***: Modifies a subnet attribute.
  - ***aws-s3-bucket-encryption-get*** that retrieves the default encryption configuration for an Amazon S3 bucket. Shows the server-side encryption settings applied to new objects stored in the bucket.
  - ***aws-s3-bucket-policy-delete*** that deletes the bucket policy from an Amazon S3 bucket. This operation removes all policy-based access controls from the bucket, potentially changing access permissions.
  - ***aws-s3-bucket-policy-get*** that retrieves the bucket policy for an Amazon S3 bucket. Returns the policy document in JSON format if one exists.
  - ***aws-s3-public-access-block-get*** that retrieves the public access block configuration for an Amazon S3 bucket. Shows the current settings that control public access to the bucket and its objects.
  - ***aws-ec2-instances-start*** that starts an Amazon EBS-backed instance that was previously stopped.
  - ***aws-ec2-instances-stop***  that stops an Amazon EBS-backed instance.
  - ***aws-ec2-instances-describe*** that describes specified instances or all instances.
  - ***aws-ec2-instances-run*** that launches a specified number of instances using an AMI you have access to.
  - ***aws-ec2-instances-terminate*** that shuts down specified instances.
  - ***aws-s3-bucket-encryption-get*** that retrieves the default encryption configuration for an Amazon S3 bucket. Shows the server-side encryption settings applied to new objects stored in the bucket.
  - ***aws-s3-bucket-policy-delete*** that deletes the bucket policy from an Amazon S3 bucket. This operation removes all policy-based access controls from the bucket, potentially affecting access permissions.
  - ***aws-s3-bucket-policy-get*** that retrieves the bucket policy for an Amazon S3 bucket. Returns the policy document in JSON format if one exists.
  - ***aws-s3-public-access-block-get*** that retrieves the public access block configuration for an Amazon S3 bucket. Shows the current settings that control public access to the bucket and its objects.
  - ***aws-ec2-describe-vpcs*** that describes specified VPCs or all VPCs.
  - ***aws-ec2-describe-subnets*** that describes specified subnets or all subnets.
  - ***aws-ec2-describe-ipam-resource-discoveries***  that describes specified IPAM resource discoveries or all IPAM resource discoveries.
  - ***aws-ec2-describe-ipam-resource-discovery-associations*** that describes specified IPAM resource discovery associations or all IPAM resource discovery associations.
  - ***aws-s3-file-upload*** that uploads a file to an Amazon S3 bucket.
  - ***aws-s3-file-download*** that downloads a file from an Amazon S3 bucket.
  - ***aws-billing-budgets-list*** that lists configured budgets for a given AWS account.
  - ***aws-billing-cost-usage-list*** that retrieves actual cost and usage data for a given time range and optional service filter.
  - ***aws-billing-forecast-list*** that forecasts AWS spending over a given future time period using historical trends.
  - ***aws-billing-budget-notification-list*** that lists the notifications that are associated with a budget.
  - ***aws-kms-key-rotation-enable*** that enables automatic rotation for a symmetric customer-managed KMS key. Not supported for asymmetric/HMAC keys, keys with imported material, or custom key stores.
  - ***aws-elb-load-balancer-attributes-modify*** that modifies attributes for Classic Elastic Load Balancer, including Cross-Zone Load Balancing, Access Logs, Connection Draining, Connection Settings, and Additional Attributes.
  - ***aws-lambda-function-configuration-get*** command that retrieves configuration information about a Lambda function.
  - ***aws-lambda-function-url-config-get*** command that returns the configuration for a Lambda function URL.
  - ***aws-lambda-policy-get*** command that returns the resource-based IAM policy for a Lambda function.
  - ***aws-lambda-invoke*** command that invokes a Lambda function.
  - ***aws-lambda-function-url-config-update*** command that updates the configuration for a Lambda function URL.
  - ***aws-ec2-latest-ami-get*** command that gets the latest AMI.
  - ***aws-ec2-network-acl-create*** command that creates a network ACL in a VPC.
  - ***aws-ec2-ipam-discovered-public-addresses-get*** command that gets the public IP addresses that have been discovered by IPAM.
  - ***aws-ec2-tags-create*** command that adds or overwrites one or more tags for the specified Amazon EC2 resource or resources.
  - ***aws-s3-bucket-website-get*** command that returns the website configuration for a bucket.
  - ***aws-s3-bucket-acl-get*** command that return the access control list (ACL) of a bucket.
  - ***aws-ec2-snapshot-permission-modify*** command.
  - ***aws-ec2-snapshot-create*** command.
  - ***aws-eks-cluster-describe*** command.
  - ***aws-eks-access-policy-associate*** command.
  - ***aws-ecs-cluster-settings-update*** command.
  - ***aws-acm-certificate-options-update*** command that updates Certificate Transparency (CT) logging for an AWS Certificate Manager (ACM) certificate (ENABLED or DISABLED).

- Added support for the following quick actions:
  - ***Enable S3 Versioning***, which enables versioning on the S3 bucket to retain multiple versions of objects.
  - ***Enable S3 Access Logging***, which enables server access logging for the S3 bucket.
  - ***Set Bucket ACL to Private***, which sets the Access Control List (ACL) for the S3 bucket to private.
  - ***Disable S3 Website Hosting***, which removes the static website hosting configuration from an S3 bucket.
  - ***Detach Security Group from EC2***, which removes the associated security group from the EC2 instance profile.
  - ***Set AMI to Private***, which revokes public launch permissions for the EC2 AMI.
  - ***Set Snapshot to Private***, which revokes public access to the EC2 snapshot.
  - ***Enable IAM Authentication on RDS Cluster***, which enables 'iam database authentication' for the RDS cluster.
  - ***Enable RDS Cluster Deletion Protection***, which enables deletion protection for the RDS DB cluster to prevent accidental deletion.
  - ***Set RDS Cluster Snapshot to Private***, which revokes public access to the RDS DB cluster snapshot, making it private.
  - ***Disable Public Access on RDS***, which disables public access for an AWS RDS database instance by modifying the instance settings.
  - ***Enable Copy Tags on RDS Snapshot***, which enables 'copy tags to snapshots' for the RDS instance.
  - ***Enable Automatic Backup for RDS DB***, which enables the automatic backup settings for the RDS DB instance with a default retention period of 30 days.
  - ***Enable IAM Authentication on RDS***, which enables 'iam database authentication' for the RDS DB instance.
  - ***Enable Deletion Protection on RDS***, which enables 'deletion protection' for the RDS DB instance.
  - ***Enable RDS Auto Upgrade***, which enables automatic minor version upgrades for the RDS database instance.
  - ***Enable Multi-AZ for RDS***, which enables Multi-AZ deployment for the RDS database instance to enhance availability and durability.
  - ***Set RDS Snapshot to Private***, which revokes public access to the RDS DB snapshot, making it private.
  - ***Enable CloudTrail Logging***, which enables logging of a CloudTrail.
  - ***Enable CloudTrail Log Validation***, which enables log file validation for the reported CloudTrail.
  - ***Enable Control Plane Logging***, which enables control plane logging for an EKS cluster.
  - ***Set EKS cluster to Private***, which disables public access to the EKS cluster's API endpoint, making it only accessible from within the cluster's VPC.
  - ***Enable IMDSv2***, which enables instance metadata services v2.0 for the EC2 instance.
  - ***Block S3 Public Access***, which blocks all public access to the S3 bucket.
