#### Integrations

##### AWS

- Added the **aws-ec2-volumes-describe** command to describe the specified EBS volumes or all of your EBS volumes.
- Added the **aws-ec2-volume-modify** command to modify several parameters of an existing EBS volume, including volume size, volume type, and IOPS capacity.
- Added the **aws-ec2-volume-create** command to create an EBS volume that can be attached to an instance in the same Availability Zone.
- Added the **aws-ec2-volume-attach** command to attach an EBS volume to a running or stopped instance and expose it to the instance with the specified device name.
- Added the **aws-ec2-volume-detach** command to detach an EBS volume from an instance.
- Added the **aws-ec2-volume-delete** command to delete the specified EBS volume. The volume must be in the available state (not attached to an instance).
