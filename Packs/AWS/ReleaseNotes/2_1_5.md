#### Integrations

##### Amazon Web Services

- Added support for the **aws-s3-bucket-objects-list** command that list objects in an s3 bucket.
- Added support for the **aws-s3-bucket-delete** command, which deletes an S3 bucket.

  - **aws-ec2-addresses-describe** - to describe one or more Elastic IP addresses.
  - **aws-ec2-address-allocate** - to allocate an Elastic IP address to your AWS account.
  - **aws-ec2-address-associate** - to associate an Elastic IP address with an instance or network interface.
  - **aws-ec2-address-disassociate** - to disassociate an Elastic IP address from an instance or network interface.
  - **aws-ec2-address-release** - to release an Elastic IP address back to the IP address pool.
