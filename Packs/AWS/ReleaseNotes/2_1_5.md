#### Integrations

##### AWS - EC2

- Added the following commands:
  - **aws-ec2-image-available-waiter** to wait until an Amazon Machine Image (AMI) becomes available.
  - **aws-ec2-images-describe** to describe EC2 images (AMIs, AKIs, and ARIs) with support for filtering by owners, executable users, and inclusion of deprecated/disabled images.
  - **aws-ec2-image-create** to create Amazon Machine Images (AMIs) from Amazon EBS-backed instances with support for custom block device mappings and tag specifications.
  - **aws-ec2-image-deregister** to deregister Amazon Machine Images (AMIs).
  - **aws-ec2-image-copy** to copy Amazon Machine Images (AMIs) across AWS regions.
