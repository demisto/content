PingCastle is a free, Windows-based utility to audit the risk level of your AD infrastructure and check for vulnerable practices.

##### What does this pack do?

The Integrations and Playbooks in this pack provide the following functionality:
- Listen for PingCastle reports.
- Create an Incident from a report sent by PingCastle.
- Upload the XML Report to the War Room as a file available for download.
