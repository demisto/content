
#### Integrations

##### Entra ID Identity And Access

- Added validation of the *ips* argument for the ***msgraph-identity-ip-named-locations-create*** and ***msgraph-identity-ip-named-locations-update*** commands. IP ranges that are not in CIDR notation (for example "0.0.0.0") now return a clear error before the API call instead of failing with a Microsoft Graph error.
