#### Integrations

##### Azure Active Directory Identity And Access

- Added support for **msgraph-identity-ca-policy-create** command that creates a conditional access policy.
- Added support for **msgraph-identity-ca-policy-delete** command that delete specific conditional access policy by id.
- Added support for **msgraph-identity-ca-policies-list** command that retrieve one or all conditional access policies from microsoft graph api.
- Added support for **msgraph-identity-ca-policy-update** command that updates a conditional access policy.
