
#### Classifiers

##### Microsoft Graph Identity and Access Classifier

- Updated the Microsoft Graph Identity and Access Classifier classifier to include 3 new risk event types: attackerinTheMiddle, attemptedPrtAccess, userReportedSuspiciousActivity

#### Integrations

##### Entra ID Identity And Access

- Updated the Docker image to: *demisto/crypto:1.0.0.5490413*.
- Updated issue description according to https://learn.microsoft.com/en-us/entra/id-protection/concept-identity-protection-risks.
  - Added the parameter **Override Microsoft Entra ID Protection risk level**, use this parameter to replace Microsoft ID protection alerts severity by a user defined severity for issues generated in Cortex XSOAR/XSIAM.
  - Added the parameter **Issue severity**, use this parameter to define severity of issues generated in Cortex XSOAR/XSIAM if **Override Microsoft Entra ID Protection risk level** has been selected.
- Added the ***msgraph-identity-audit-signin-event-get*** command. Use this command to retrieve the sign-in event associated with a risk detection.

#### Mappers

##### Microsoft Graph Identity and Access Incoming Mapper

- Updated the Microsoft Graph Identity and Access Incoming Mapper mapper to map Mapped *Event ID* and *Original Alert ID* issue fields.
