#### Integrations
##### Darkmon
- New: Initial Marketplace release of the Darkmon Threat Intelligence integration.
- New: 18 commands covering indicator enrichment, IOC feed (TIM), compromised data,
  board protection, VIP email leaks, ransomware (articles + mentions), landscape
  news, newly registered domains, telnet brute-force IOCs, CVE feed, and a
  fully dynamic global search.
- New: Reputation commands (`ip`, `url`, `domain`, `email`, `file`) populate
  `DBotScore` and `Common.<Type>` per the XSOAR mandatory contract.
- New: Feed integration with full TIM controls: `feedReputation`, `feedReliability`,
  `feedExpirationPolicy`, `feedExpirationInterval`, `feedFetchInterval`,
  `feedBypassExclusionList`, `tlp_color`, `feedTags`, `limit`.
- New: API base URL is configurable per-instance for dev/staging targeting.
- New: Optional `redact_secrets` toggle (default on) keeps passwords and card
  numbers out of War Room output while preserving raw values in `rawJSON`.
- New: Reputation commands accept arrays / CSV input: `!ip 1.1.1.1,8.8.8.8`.
- New: Default newest-first sort on time-series commands (`firstSeen,desc`,
  `timestamp,desc`, `publishedAt,desc`); overridable via `sort` argument.
- New: `dmontip-global-search` `type` predefined values aligned with the latest
  `tip-common` `SearchKey` enum: `Source` replaces `Actor`, `CVE` replaces
  `Vulnerability`. Update any saved invocations or playbook tasks that pinned
  the old values.

#### Playbooks
##### Darkmon - Enrich IP / Domain / URL / File / Email
- New: Indicator enrichment sub-playbooks that wrap the reputation commands and
  return `DBotScore` + `Common.<Type>`. Drop into any existing playbook.

##### Darkmon - Compromised Credentials Sweep
- New: Pulls new compromised accounts every 4 hours, classifies by domain
  (customer-affected vs. external), creates one incident per affected account.

##### Darkmon - VIP Email Monitor
- New: Lists board-protected emails hourly; for each, fetches accounts/combo
  lists/public breaches and creates an incident if new entries appear.

##### Darkmon - Ransomware Mentions Watch
- New: Every 6 hours, polls company ransomware mentions; creates a critical
  incident per new mention with dedup pre-process rule.

##### Darkmon - Brand-Targeted NRD Watch
- New: Daily sweep of newly registered domains; flags typosquats against the
  customer's configured brand list.

##### Darkmon - Critical CVE Pipeline
- New: Daily filter of new CVEs by CVSS >= 9 and customer tech-stack tags;
  one incident per match.

##### Darkmon - Phishing Email Triage
- New: Extracts IOCs from a reported phishing email, enriches via Darkmon,
  blocks malicious indicators via the generic switchboard.

##### Darkmon - Compromised Account Response
- New: Identifies the affected user via the configured directory, suspends,
  forces password reset, revokes sessions, notifies SOC + user.

##### Darkmon - EDR Alert Enrichment
- New: Runs on alerts from CrowdStrike / SentinelOne / Defender; enriches IOCs
  with Darkmon and adjusts incident severity / containment accordingly.

##### Darkmon - Ransomware Victim Response
- New: Triggered when our company name surfaces in Darkmon ransomware mentions;
  spins up a war-room channel and notifies the CISO.

##### Darkmon - Email Deep Dive
- New: Analyst-driven: given an email, runs all four Darkmon angles
  (boardprotection presence, boardemails x3, global search) and presents a
  unified view.

##### Darkmon - Generic Notify / Block Indicator / User Action
- New: Provider-agnostic switchboard sub-playbooks. Configure once which provider
  the customer has (Slack/Teams/email; PAN/Fortinet/Umbrella; AD/Okta/Azure
  AD); every incident-response playbook calls these.

#### Layouts
- New: Custom indicator layouts for IP, Domain, URL, File, Email showing Darkmon
  classification, IPs seen, event count, and last-seen timestamps.
- New: Custom incident layouts for each Darkmon-specific incident type.

#### Wizards
