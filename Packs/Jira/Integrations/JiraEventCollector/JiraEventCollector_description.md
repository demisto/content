Use the Jira Event Collector integration to get Audit and logs using REST APIs.

Before you start, you need to get the tenant ID, username, and password. For more information, see [Jira Documentation](https://docs.cyberark.com/Product-Doc/OnlineHelp/Idaptive/Latest/en/Content/Integrations/SIEM-PlatformEvents/Identity%20Platform%20API%20Usage%20Guide%20for%20ArcSight.pdf), **Prerequisite for Accessing INGA Events** section (pages 4-10).

## Configuration Parameters

**Server URL**    
The endpoint to get the logs. For example, ``https://{{tenant}}.atlassian.net``.

**User name and Password**    
The user name and password.
