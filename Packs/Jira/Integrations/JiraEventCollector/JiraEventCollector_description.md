Use the Jira Event Collector integration to get Audit and logs using REST APIs.

Before you start, you need to get the your-domain, username, and password. For more information, see [Jira Documentation](https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/#ad-hoc-api-calls).

## Configuration Parameters

**Server URL**    
The endpoint to get the logs. For example, ``https://{{your-domain}}.atlassian.net``.

**User name and Password**    
The user name and password.
