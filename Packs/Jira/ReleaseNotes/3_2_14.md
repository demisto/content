
#### Integrations

##### Atlassian Jira v3

- Added the **RawDescription** field to the ***fetch-incidents*** context data. This field contains the raw description of the Jira issue.
- Added the **Ticket.RawDescription** field to the context data of the following commands:
  - ***jira-issue-query***
  - ***jira-edit-issue***
  - ***jira-get-issue***
  - ***jira-issue-assign***
