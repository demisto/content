
#### Integrations

##### Atlassian Jira v2

Added support for additional arguments in the ***jira-get-organizations*** command.
    - *projectKey*: The ID of the service desk from which the organization list will be returned. This can alternatively be a project identifier.
    - *start*: The starting index of the returned objects. Base index 0.
    - *limit*: The maximum number of items to return per page. Default, 50.
    - *accountId*: The account ID of the user, which uniquely identifies the user across all Atlassian products. For example, 5b10ac8d82e05b22cc7d4ef5.
