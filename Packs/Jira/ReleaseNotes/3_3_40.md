
#### Integrations

##### Atlassian Jira v3

- Fixed an issue where mirrored comments, attachments, and field updates could silently stop syncing from Jira to Cortex XSOAR when comparing timestamps across timezones, due to a naive/aware datetime mismatch.
- Fixed an issue where **Close Mirrored XSOAR Incident** only closed the incident for Jira issues whose status was literally named `Done`. It now also recognizes any status in Jira's built-in `Done` status category, so custom terminal statuses (e.g. `Closed`, `Won't Fix`) close the mirrored incident as well.
- Added additional debug logging around comment mirroring and modified-issue polling to help diagnose sync issues.
