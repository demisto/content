
#### Integrations

##### Vega

- Initial release of the Vega integration for Cortex XSOAR.

#### Classifiers

##### Vega Mapper

- Updated the incoming mapper to stop mapping raw Vega alert and incident JSON into incident labels, preventing duplicate field values from appearing in the War Room.
- Added mapping from Vega alert `id` to the **Alert ID** incident field.

#### Scripts

##### vega-fetch-alert-events

- Layout widget script that loads alert events for the Vega Alert investigation layout.

##### vega-get-alert-events

- Vega integration command that fetches alert events from the Vega API.

#### Integrations

##### Vega

- Mirrored Vega incident comments now appear in the War Room as plain comment notes instead of verbose tables.
- Added incoming and outgoing comment mirroring support through `get-remote-data`, `get-modified-remote-data`, and `update-remote-system`.
- Normalized `verdictReasoning` to display `N/A` when null.
- Mapped the Vega alert UUID (`id`) to the **Alert ID** incident field and use it to fetch alert events.
- Fixed alert events detection: rows containing `cid`/`eid` return "No alert events found"; all other shapes are rendered normally.
- Renamed `vega-fetch-alert-events` integration command to `vega-get-alert-events` and the layout script to `vega-fetch-alert-events`.

#### Incident Fields

##### Vega Verdict Reasoning

- Changed the field type from long text to short text.

#### Layouts

##### Vega Alert / Vega Incident

- Added Vega Verdict Reasoning below Vega Verdict in the layout properties section.
