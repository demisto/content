# Unit 42 by Palo Alto Networks

This pack provides threat intelligence feeds from Unit 42, the global threat intelligence team at Palo Alto Networks.

## What does this pack do?

The Unit 42 Feed integration allows you to:

- Fetch indicators from Unit 42 threat intelligence feeds
- Get real-time updates on the latest threats
- Enrich your security operations with actionable threat intelligence

## Integrations in this pack

- **Unit 42 Feed** - Fetches indicators from Unit 42 threat intelligence feeds.
