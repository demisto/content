
#### Integrations

##### FireEyeHX v2
- Added support for the **fireeye-hx-delete-triage-acquisition** command that deletes a specified triage acquisition.
- Added support for the **fireeye-hx-get-triage-acquisition-package** command that downloads the specified triage acquisition as a ".mans" file.
- Added support for the **fireeye-hx-get-triage-acquisition** command that gets information about a triage acquisition.
- Added support for the **fireeye-hx-triage-acquisition** command that submits a triage request to the HX server.
- Added support for the **fireeye-hx-host-get-acquisitions** command that gets all known acquisitions for a host.
