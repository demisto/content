
#### Integrations

##### Microsoft Defender for Endpoint

Add File indicator to the ***microsoft-atp-get-file-statistics*** command and fill the new indicator fields created under the CommonTypes pack:

- **New: Organization Prevalence**
- **New: Global Prevalence**
- **New: Organization First Seen**
- **New: Organization Last Seen**
- **First Seen By Source**
- **Last Seen By Source**
