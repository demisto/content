
#### Integrations

##### Microsoft Defender for Endpoint

- Deleted the **ID** parameter.
- Deleted the **Token** parameter.
- Added support for *interval_in_seconds* argument in the **microsoft-atp-stop-and-quarantine-file** command.
- Added support for *action_ids* argument in the **microsoft-atp-stop-and-quarantine-file** command.
- Added support for *polling* argument in the **microsoft-atp-stop-and-quarantine-file** command.
- Added support for *timeout_in_seconds* argument in the **microsoft-atp-stop-and-quarantine-file** command.
