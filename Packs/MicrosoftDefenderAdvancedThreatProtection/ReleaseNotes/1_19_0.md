
#### Scripts

##### New: MicrosoftAtpIsolateMachine

Added the ***MicrosoftAtpIsolateMachine*** script that isolates a machine from accessing external networks.


##### New: MicrosoftAtpUnisolateMachine

Added the ***MicrosoftAtpUnisolateMachine*** script that reverses the isolation of a machine.


##### New: MicrosoftAtpEndpoint

Added the ***MicrosoftAtpEndpoint*** script that gets machines that have communicated with Microsoft Defender for Endpoint cloud.


##### New: MicrosoftAtpScIndicatorCreate

Added the ***MicrosoftAtpScIndicatorCreate*** script that creates a new indicator in Microsoft Defender for Endpoint.
