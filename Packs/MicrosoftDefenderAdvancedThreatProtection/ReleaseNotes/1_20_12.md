
#### Integrations

##### Microsoft Defender for Endpoint

- Added the *polling* argument to the ***microsoft-atp-stop-and-quarantine-file*** command to support polling.
- Added the *timeout_in_seconds* argument to the ***microsoft-atp-stop-and-quarantine-file*** command. Use this argument to set the timeout when running the command in polling mode.
- Added the *interval_in_seconds* argument to the ***microsoft-atp-stop-and-quarantine-file*** command. Use this argument to set the polling interval when running the command in polling mode.
- Fixed issue in ***endpoint*** command where too many query filter expressions provided would cause failure.
