#### Integrations

##### Microsoft Defender for Endpoint

- Fixed an issue where the ***microsoft-atp-sc-indicator-create*** and ***microsoft-atp-sc-indicator-update*** commands forced a default expiration time of 14 days on the *expiration_time* argument. The argument is now truly optional — when omitted, no expiration is sent to the API and the indicator will not expire.
- Updated the Docker image to: *demisto/crypto:1.0.0.7704170*.
