
#### Integrations

##### Microsoft Defender for Endpoint

- Added support for **microsoft-atp-get-machine-software** command that get the specific machine's software details.
- Added support for **microsoft-atp-get-machine-missing-kbs** command that get the specific machine's missing security updates (kbs).
- Added support for **microsoft-atp-get-machine-vulnerabilities** command that get the specific machine's vulnerabilities.
- Updated the Docker image to: *demisto/crypto:1.0.0.2005673*.
