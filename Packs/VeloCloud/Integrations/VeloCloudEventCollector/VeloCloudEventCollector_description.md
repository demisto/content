## VeloCloud Event Collector Help

### Setup Integration

- Set the `Server URL` to the URL for the VeloCloud tenant
- Set the `Enterprise ID` to the enterprise ID for the tenant this is collecting from
- Set the `API Key` to an API key with read access to the event log on the tenant
