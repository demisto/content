Arista VeloCloud is a cloud-delivered SD-WAN (Software-Defined Wide Area Network) solution that is part of the Arista Networks portfolio. It is designed to simplify, optimize, and secure enterprise and branch connectivity across wide-area networks (WANs).

For more information about Arista VeloCloud, visit: https://www.arista.com/en/solutions/sd-wan

## What does this pack do?

This pack includes the VeloCloud Event Collector integration that:

- Collects Edge events in an enterprise or Edge context
