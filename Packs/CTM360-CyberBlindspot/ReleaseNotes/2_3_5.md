- Unified incident fetch for XSOAR and XSIAM with numeric severity mapping.
- Updated the Docker image to: *demisto/python3:3.12.13.9059085*.

##### CTM360 HackerView

- Added support for **Module To Use** parameter that choose the module: light scan or deep scan.
- Unified incident fetch for XSOAR and XSIAM with numeric severity mapping.
- Updated the Docker image to: *demisto/python3:3.12.13.9059085*.
