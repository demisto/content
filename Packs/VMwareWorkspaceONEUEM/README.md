VMware Workspace ONE Unified Endpoint Management, manages and secure every device for any use case with industry standard unified endpoint management (UEM). It is a single solution for modern, over-the-air management of desktops, mobile, rugged, wearables, and IoT.

### What does this pack do?

The content pack through Workspace One UEM v2 API provides the following capabilities:

- Search enrolled devices through username, platform name, model name, last seen etc.

- Get network information of devices including voice and data roaming status, IP address and WiFi details.

- Get security data like whether the device is compromised, is data protection enabled, does it use block and file level encryption.

- List available OS updates for specified devices.
