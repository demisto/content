
#### Integrations

##### EWS Extension Online Powershell v3

- Added support for **ews-remove-tenant-allow-block-list-items-quick-action** command that remove items from the tenant allow/block lists.
- Added support for **ews-new-tenant-allow-block-list-items-quick-action** command that add new items to the tenant allow/block lists.
- Added support for **ews-release-quarantinemessage-quick-action** command that release quarantine messages.
