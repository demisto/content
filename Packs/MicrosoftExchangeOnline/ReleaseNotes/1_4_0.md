
#### Integrations

##### EWS Extension Online Powershell v3

- Added the ***ews-export-quarantinemessage*** command, which executes the *Export-QuarantineMessage* cmdlet to export and save details of quarantined messages for further analysis and record-keeping.
- Added the ***ews-get-quarantinemessage*** command, which executes the Get-QuarantineMessage cmdlet to retrieve and display details of quarantined messages, including message IDs, identities, and quarantine reasons.
- Added the ***ews-release-quarantinemessage*** command, which executes the Release-QuarantineMessage cmdlet to release specific quarantined messages, allowing them to be delivered to the intended recipients after administrative review.
