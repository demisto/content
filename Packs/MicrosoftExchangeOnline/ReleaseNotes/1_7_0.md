#### Integrations

##### EWS Extension Online Powershell v3

- Added the ***ews-message-trace-list*** command using the Get-MessageTraceV2 PowerShell replacing **ews-message-trace-get** cmdlet.

- **Breaking changes:** The ***ews-message-trace-get*** command was deprecated and replaced with the ***ews-message-trace-list*** command. The following input parameter changes apply:

**Removed Parameters:**

- `page` - pagination parameter removed. Due to the API limitations, pagination is no longer available in the new command
- `page_size` - pagination parameter removed. Due to the API limitations, pagination is no longer available in the new command

**Modified Parameters:**

- `message_id` - now accepts multiple values
- `status` - now accepts multiple values

**Replacement Parameters:**

- `limit` - replaces `page` and `page_size` for result limiting

**New Parameters Added:**

- `subject` - filters results by message subject
- `subject_filter_type` - specifies how subject is evaluated (Contains/EndsWith/StartsWith)
- `starting_recipient_address` - used with end_date for handling partial results without duplication
