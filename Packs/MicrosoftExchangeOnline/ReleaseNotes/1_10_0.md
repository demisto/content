
#### Integrations

##### New: O365 Message Trace

- Added the **O365 Message Trace** integration, which ingests Exchange Online Message Trace events into Cortex XSIAM. Message Trace enables tenant administrators to track the lifecycle of an email, determine its delivery status (delivered, pending, failed, or quarantined), and understand the actions applied to it.
- Added the following commands:
	- ***o365-message-trace-generate-login-url***, which generates the login URL for the authorization code flow.
	- ***o365-message-trace-auth-test***, which tests connectivity to Microsoft.
	- ***o365-message-trace-auth-reset***, which clears the saved access token / refresh token from the integration context, so on the next run a fresh authorization code (or client-credentials token) is requested.
	- ***o365-message-trace-get-events***,  which retrieves Exchange Online Message Trace events (for development and debugging only).
