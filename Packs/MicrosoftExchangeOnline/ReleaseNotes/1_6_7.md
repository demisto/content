
#### Integrations

##### EWS Extension Online Powershell v3

- Added support for **ews-remove-tenant-allow-block-list-items-quick-action** command that remove items from the tenant allow/block lists.   you can delete items by their value or by unique id.  uses powershell cmdlet removetenantallowblocklistitems cmdlet.
- Added support for **ews-new-tenant-allow-block-list-items-quick-action** command that add new items to the tenant allow/block lists.  uses powershell newtenantallowblocklistitems cmdlet.
- Added support for **ews-release-quarantinemessage-quick-action** command that release quarantine messages. uses powershell releasequarantinemessage cmdlet.
