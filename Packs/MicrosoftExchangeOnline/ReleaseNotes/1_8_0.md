
#### Integrations

##### O365 - Security And Compliance - Content Search v2

- Adding a new authentication method to the integration: *Delegated User Authentication*. See the integration Description for more information.
- The following commands are only available when using the Delegated User Authentication method, as per the [Microsoft Update](https://mc.merill.net/message/MC1131771):
  - o365-sc-new-search-action 
  - o365-sc-case-hold-policy-create 
  - o365-sc-case-hold-policy-set 
  - o365-sc-case-hold-policy-delete 
  - o365-sc-case-hold-rule-create
  - o365-sc-case-hold-rule-delete
- The **Export** parameter has been deprecated from the **o365-sc-new-search-action** command, as per [Microsoft's update](https://techcommunity.microsoft.com/blog/microsoft-security-blog/upcoming-changes-to-microsoft-purview-ediscovery/4405084). The following 'Export' dependent arguments been deprecated inside the **o365-sc-new-search-action** command: 
  - *share_point_archive_format*.
  - *scenario*.
  - *notify_email*.
  - *format*.
  - *include_sharepoint_document_versions*.
  - *scope*.
  - *notify_email_cc*.
- Updated the Docker image to: *demisto/pwsh-exchangev3:1.0.0.4904712*.
