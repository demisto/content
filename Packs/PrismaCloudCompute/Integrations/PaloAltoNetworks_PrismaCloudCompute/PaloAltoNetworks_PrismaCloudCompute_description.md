This integration provides the ability to import Palo Alto Networks - Prisma Cloud Compute alerts into Demisto.

Important: Before you can use the Prisma Cloud Compute integration on Demisto, there are several configuration steps requeired on the Prisma Cloud Platform.

1. Login to your Prisma Cloud Compute console.
2. Navigate to Manage -> Alerts.
3. Create a new alert profile by clicking the 'Add Profile' button.
4. Choose 'Demisto' from the provider list on the left and choose what would you like Demisto to be alerted about from the alert triggers on the right.
5. Click 'Save' to save the alert profile.
