
#### Scripts

##### New: PCComputeContainerComplianceIssuesButton

- New: This script runs the "prisma-cloud-compute-hosts-scan-list" command for a specific container id and returns details about its compliance issues, if found. If any compliance issues found, it will create a new tab in the layout called "Detailed Compliance Issues" showing the issues details.
Returns the following fields for each compliance ID:
- Compliance ID
- Cause
- Severity
- Title
- Description
