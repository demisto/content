## ThreatConnect v2
1. In **ThreatConnect** UI go to **settings** -> **Org settings**.
2. Go to **Apps** and click **+** to add a new app.
3. Enter the job name and then in the **Run Program** filed select **ThreatConnect Cross Intel Sharing**.
4. Configure the **Source ThreatConnect API Access ID** (`Access ID`).
5. Configure the **Source ThreatConnect API Secret Key** (`Secret Key`).
6. Finish the configuration with other fields.
