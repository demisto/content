# How To get your AIMA API Key
1. Access your AIMA environment.
2. Navigate to **AIMA -> Profile -> Settings -> API KEYS**. 

# Important
AIMA CAP Module is optional. You should use the cap-api key to run CAP Module commands if it is included in your packet.

# How To get your AIMA CAP API Key
1. Access your AIMA environment.
2. Navigate to **CAP -> Settings -> API KEYS**.
