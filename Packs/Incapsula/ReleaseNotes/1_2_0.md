
#### Integrations

##### Imperva Incapsula

- Added 6 new commands to support the V2 Policy Management API:
  - ***incap-policy-list***
  - ***incap-policy-create***
  - ***incap-policy-update***
  - ***incap-policy-delete***
  - ***incap-asset-on-policy-check***
  - ***incap-asset-on-policy-apply***
- Deprecated the ***incap-modify-site-acl-config*** command as the V1 API it uses has been deprecated. Please use the new Policy Management commands instead.
