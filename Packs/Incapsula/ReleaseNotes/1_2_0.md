
#### Integrations

##### Imperva Incapsula

- Added 3 new commands to support the V2 Policy Management API:
  - ***incap-get-all-policies***
  - ***incap-create-policy***
  - ***incap-modify-policy***
- Deprecated the ***incap-modify-site-acl-config*** command as the V1 API it uses has been deprecated. Please use the new Policy Management commands instead.
