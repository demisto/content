#### Integrations

##### IPQualityScore

- Added the ***ipqs-ip-reputation*** command, which replaces the deprecated ***ip*** command.
- Added the ***ipqs-email-reputation*** command, which replaces the deprecated ***email*** command.
- Added the ***ipqs-url-reputation*** command, which replaces the deprecated ***url*** command.
- Added the ***ipqs-phone-reputation*** command to run validation and reputation checks on phone numbers.
- Added the ***ipqs-username-leaked*** command to check if a username has been exposed in known data breaches.
- Added the ***ipqs-password-leaked*** command to check if a password has been exposed in known data breaches.
- Added the ***ipqs-email-leaked*** command to check if an email address has been exposed in known data breaches.
- Added the ***ipqs-file-scan*** command to upload and scan a file using the IPQualityScore Malware File Scanner API.
- Added the ***ipqs-url-file-scan*** command to submit a URL to the IPQualityScore Malware File Scanner API.
- Deprecated the ***ip*** command. Use ***ipqs-ip-reputation*** instead.
- Deprecated the ***email*** command. Use ***ipqs-email-reputation*** instead.
- Deprecated the ***url*** command. Use ***ipqs-url-reputation*** instead.
- Added the *Phone Suspicious Score Threshold* parameter to configure the fraud score threshold for marking a phone number as suspicious.
- Added the *Phone Malicious Score Threshold* parameter to configure the fraud score threshold for marking a phone number as malicious.
- Updated the Docker image to: *demisto/python3:3.12.13.10230415*.
