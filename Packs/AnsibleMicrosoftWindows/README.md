This pack enables the management of Microsoft Windows hosts using Ansible modules. The Ansible engine is self-contained and pre-configured as part of this pack onto your XSOAR server. The Ansible modules have been exposed as XSOAR commands, and allow you to use them without needing to know Ansible.

# What does this pack allow you to manage?

* Users, Groups
* Event/Audit Logs
* Hostname
* Shares
* User profiles
* Windows Firewall
* Processes
* Files/Folders, ACLs, Disks, Partitions
* Interact with User using Messages, Toasts, and Text to Speech
* Install software using Chocolatey
* Services
* Windows Features
* Power State and Plan
* Download files
* Issue URI requests
* Create wakeonlan packets
* Time
* File contents
* Scheduling
* Proxy
* Hosts file
* Domain
* Shortcuts
* Pagefile
* IIS

In addition to this the Windows Integration can be used to retrieve information about:

* General system info
* Hardware
* Services
* Installed software and licensing
