
#### Scripts

##### SendEmailReply
Added option to dynamically add recipient domain to EmailDomainAllowList 
every time a new email thread to a new domain is sent. This prevents 
users from having to manually maintain an allowlist.
##### PreprocessEmail
Added functionality to check an email domain allowlist to prevent spam 
from creating incidents as pre-processing rules to do so cannot currently 
be used in conjunction with the PreprocessEmail preprocessing script.
