
#### Scripts

##### SendEmailReply

- Fixed an issue where email replies were not grouped into the same thread/conversation in the recipient's email client when using mail sender integrations other than Gmail Single User (e.g., O365 Outlook Mail via Graph API, EWS). The RFC 2822 `References` header is now always included in reply emails when a message ID is available, ensuring proper email threading across all supported mail sender integrations.
