
#### Integrations

##### Cortex Platform - Core

- Improved the ***core-get-cases*** command enrichment performance by using a single bulk API call instead of per-case requests when *get_enriched_case_data* is enabled.
- Removed the 10-case limit for enriched case data retrieval — enrichment now works for any number of cases.
- Enriched case data now returns the full case and issues data from the API response, including all issue fields.
