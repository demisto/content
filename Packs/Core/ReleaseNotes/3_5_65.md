
#### Integrations

##### Investigation & Response

- Deprecated the following quick actions:
  - **Isolate Endpoint**
  - **Quarantine File**
  - **Restore File From Quarantine**
  - **Run Malware Scan**
  - **Retrieve File**
  - **Run Endpoint Script**
  - **Terminate CGO**

 These quick actions remain fully functional and can still be used, so existing playbooks and saved response actions are not affected. Replacement quick actions with the same names are being rolled out gradually and may not yet be available in your tenant. Until the new quick actions appear, continue using the current ones, which are marked with a "(deprecated)" suffix to distinguish them from the new versions.
