
#### Integrations

##### XQL Query Engine

- Added support for the **xdr-xql-library-create** command which creates or updates XQL queries in the library.
- Added support for the **xdr-xql-library-delete** command which deletes XQL queries from the library.
- Added support for the **xdr-xql-library-list** command which gets a list of XQL queries from the library.
