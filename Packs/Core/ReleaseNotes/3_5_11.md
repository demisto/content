#### Integrations

##### Cortex Platform - Core

- Fixed an issue in the **core-get-cases** command where requesting enriched case data for more than 10 cases returned an error and no data. The command now falls back to standard case data automatically.
- Fixed an issue in the **core-get-case-extra-data** command where a missing or non-numeric `case_id` caused an unhandled API error. The command now validates the `case_id` and returns a clear error message.
