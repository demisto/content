### Binalyze AIR Extended

Use this integration to connect Cortex XSOAR to Binalyze AIR for endpoint forensic acquisition, triage, case, task, repository, and isolation operations.

### Setup

1. Log in to Binalyze AIR.
2. Go to the API token or integrations section.
3. Create a new API token with the minimum required permissions.
4. Copy the token.
5. In Cortex XSOAR, create a Binalyze AIR Extended integration instance.
6. Enter the Binalyze AIR Server URL and API Key.
7. Click **Test** to verify connectivity.
