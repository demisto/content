#### Integrations

##### PagerDuty v2

- Fixed an issue where ***fetch-incidents*** silently dropped incidents.
- Fixed an issue where short-lived incidents that transitioned to `resolved` between two fetch cycles were never ingested.
- Added the *Maximum number of incidents per fetch* integration parameter (default: **50**, hard cap: **200**).
- Added the *Incident statuses to fetch* integration parameter (multi-select: **triggered**, **acknowledged**, **resolved**).
- Updated the Docker image to: *demisto/python3:3.12.13.10116658*.
