#### Breaking Changes
- The **Inception** integration is now deprecated. A new **Stairwell** integration with ID **Stairwell** replaces it. The original **Inception** integration (ID: **Inception**) remains available as a deprecated stub for backward compatibility.
- All command names now use the ***stairwell-*** prefix instead of ***inception-***. For example, ***inception-file-enrichment*** is now ***stairwell-file-enrichment***.
- All context paths have changed from **Inception.\*** to **Stairwell.\***. Update existing playbooks and automations to use the new integration, command names, and context paths.

#### New Features
- Added the ***stairwell-intake-upload*** command to upload files for analysis from XSOAR War Room entries, HTTP/HTTPS URLs, or direct file paths. Includes automatic SHA256 calculation, preflight checks, and retry logic.
- Added the ***stairwell-ai-triage-summarize*** command to get AI-generated threat analysis summaries with malicious likelihood scoring and IOC extraction.
- Added the ***stairwell-object-sightings*** command to list all sightings for a file across your organization's assets.
- Added the ***stairwell-object-detonation-trigger*** and ***stairwell-object-detonation-get*** commands for sandbox detonation analysis.
- Added the ***stairwell-object-opinions*** command to retrieve threat intelligence opinions for objects.
- Added the ***stairwell-run-to-ground-generate*** command for comprehensive multi-object tracing and relationship mapping.
- Added network intelligence commands: ***stairwell-asn-get-whois***, ***stairwell-hostname-get***, ***stairwell-hostname-get-resolutions***, ***stairwell-hostname-batch-get-resolutions***, ***stairwell-ipaddress-get***, ***stairwell-ipaddress-lookup-cloud-provider***, ***stairwell-ipaddress-get-hostnames-resolving-to-ip***, ***stairwell-ipaddress-get-whois***.
- Added YARA rule management commands: ***stairwell-yara-create-rule***, ***stairwell-yara-get-rule***, ***stairwell-yara-query-matches***.
- Added asset management commands: ***stairwell-asset-list***, ***stairwell-asset-create***, ***stairwell-asset-get***.
- Added utility commands for URL/hostname canonicalization and cloud IP range lookups.

#### Improvements
- Improved error messages across all commands with actionable troubleshooting guidance for HTTP 401, 403, 404, 429, and 5xx errors.
- Added automatic retry logic with exponential backoff for transient API failures.
- Added HTTP timeout management (120 seconds for API calls, 600 seconds for uploads).
- Added streaming download support for large files in intake upload.
- Added input validation for numeric parameters (***page_size***, ***origin_zone_id***).

#### Bug Fixes
- Fixed crashes from timeout exceptions in file upload and download operations.
- Fixed crashes from invalid ***page_size*** parameters in YARA and Asset commands.
- Fixed crashes from invalid ***origin_zone_id*** parameter in intake upload.
- Fixed missing error handling in object sightings, detonation, and opinions commands.
- Fixed JSON parsing errors in intake preflight response handling.
