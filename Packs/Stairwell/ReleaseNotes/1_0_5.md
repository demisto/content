#### Integrations
##### Stairwell
- **Brand Refresh**: Removed all "Inception" references and rebranded to "Stairwell" throughout the integration
  - Updated command names from `inception-*` to `stairwell-*` (e.g., `inception-file-enrichment` → `stairwell-file-enrichment`)
  - Updated output context paths from `Inception.*` to `Stairwell.*`
  - Updated data structure references to match new API response format
- **New Command - File Intake Upload**: Added `stairwell-intake-upload` command to upload files to Stairwell with preflight checks and automatic retry logic
- **New Command - AI Triage Summarize**: Added `stairwell-ai-triage-summarize` command to get AI-generated file summaries including threat analysis, IOCs, and risk assessment
- **New Commands - Object Operations**:
  - `stairwell-object-sightings`: list sightings for an object
  - `stairwell-object-detonation-trigger`: trigger a detonation for an object
  - `stairwell-object-detonation-get`: get detonation details
  - `stairwell-object-opinions`: list opinions for an object
  - `stairwell-run-to-ground-generate`: generate Run-To-Ground analysis for objects
- **New Commands - Network Intelligence APIs**:
  - **ASN Operations**:
    - `stairwell-asn-get-whois`: get ASN WHOIS information
  - **Hostname Operations**:
    - `stairwell-hostname-get`: get hostname entity with DNS resolution data
    - `stairwell-hostname-get-resolutions`: get all addresses resolved to by a hostname over a time range
    - `stairwell-hostname-batch-get-resolutions`: get resolution summaries for multiple hostnames
  - **IP Address Operations**:
    - `stairwell-ipaddress-get`: get IP address entity with enrichment data
    - `stairwell-ipaddress-lookup-cloud-provider`: check if an IP address belongs to a known cloud provider
    - `stairwell-ipaddress-get-hostnames-resolving-to-ip`: get all hostnames resolved to by an IP address over a time interval
    - `stairwell-ipaddress-get-whois`: get WHOIS information for an IP address
  - **Utility Operations**:
    - `stairwell-utilities-get-cloud-ip-ranges`: get IP ranges for known cloud providers
    - `stairwell-utilities-batch-canonicalize-hostnames`: canonicalize multiple hostnames in bulk
    - `stairwell-utilities-batch-compute-etld-plus-one`: compute effective top-level domain plus one for multiple domains
    - `stairwell-utilities-canonicalize-hostname`: canonicalize a single hostname
    - `stairwell-utilities-compute-etld-plus-one`: compute effective top-level domain plus one for a single domain
    - `stairwell-utilities-batch-canonicalize-urls`: canonicalize multiple URLs in bulk
    - `stairwell-utilities-canonicalize-url`: canonicalize a single URL
- **Enhanced File Enrichment**: Improved `stairwell-file-enrichment` command with better error handling, safer response parsing, and updated output structure
- **Code Improvements**: Added helper functions, improved error handling, and enhanced response parsing with safe accessors throughout the integration
