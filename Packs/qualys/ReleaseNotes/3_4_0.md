
#### Integrations

##### Qualys VMDR

- Migrated deprecated Qualys API v2.0 endpoints to their latest supported versions to prevent service disruption before the End-of-Life deadlines:
  - ***qualys-host-list***, ***qualys-host-update***, and ***qualys-purge-scan-host-data*** commands migrated from v2.0 to v5.0.
  - ***qualys-host-list-detection*** command and the *fetch-assets* (assets stage) migrated from v2.0 to v5.0.
  - ***qualys-vulnerability-list***, ***qualys-get-quid-by-cve*** commands and the *fetch-assets* (vulnerabilities stage) migrated from v2.0 to v4.0.
  - ***qualys-vm-scan-list*** command migrated from v2.0 to v3.0.
  - ***qualys-report-list*** command migrated from v2.0 to v3.0.
- Added the *cloud_agent_scan_type* argument to the ***qualys-vulnerability-list*** and ***qualys-get-quid-by-cve*** commands.
