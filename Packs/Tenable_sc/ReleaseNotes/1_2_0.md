
#### Integrations

##### Tenable.sc

- Added the ***fetch-assets*** command to import host/asset data into Cortex XSIAM using the `/hosts/search` API endpoint with pagination support.  
- Added Assets Modeling Rules for Cortex XSIAM asset ingestion.  
- Added the *Assets fetch interval* parameter to configure the interval in minutes between asset fetches.  
- Added the **vulnerability fetching** capability to the ***fetch-assets*** command to send cumulative vulnerability data to Cortex XSIAM via the `/analysis` API endpoint with pagination support.  
  - Vulnerabilities are fetched using `sourceType=cumulative` after the asset fetch phase completes, mirroring the Tenable.io integration pattern.  
  - Vulnerability data is sent to Cortex XSIAM with product `sc_vulnerabilities`.
