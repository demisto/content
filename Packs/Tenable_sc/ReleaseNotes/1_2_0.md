
#### Integrations

##### Tenable.sc
- Added **fetch-assets** capability to import host/asset data into XSIAM using the `/hosts/search` API endpoint with pagination support.
- Added Assets Modeling Rules for XSIAM asset ingestion.
- Added configurable **Assets fetch interval** parameter (default: 720 minutes, minimum: 60 minutes).
- Added **vulnerability fetching** capability to the `fetch-assets` command, sending cumulative vulnerability data to XSIAM via the `/analysis` API endpoint with pagination support.
- Vulnerabilities are fetched using `sourceType=cumulative` after the assets fetch phase completes, mirroring the Tenable.io integration pattern.
- Vulnerability data is sent to XSIAM with product `sc_vulnerabilities`.
