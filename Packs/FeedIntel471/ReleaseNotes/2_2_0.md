
#### Indicator Fields

##### Intel471 Info Stealer Version

- New: **Intel471 Info Stealer Version** - Stores the version of the info stealer malware.

##### Intel471 Info Stealer Screenshot Path

- New: **Intel471 Info Stealer Screenshot Path** - Stores the path to the screenshot captured by the info stealer on the infected machine.

##### Intel471 Info Stealer Computer Username

- New: **Intel471 Info Stealer Computer Username** - Stores the username logged in on the infected machine.

##### Intel471 Info Stealer PC Name

- New: **Intel471 Info Stealer PC Name** - Stores the PC name of the infected machine.

##### Intel471 Info Stealer Antivirus Software

- New: **Intel471 Info Stealer Antivirus Software** - Stores the antivirus software detected on the infected machine.

##### Intel471 Info Stealer IP

- New: **Intel471 Info Stealer IP** - Stores the IP address of the infected machine.

##### Intel471 Info Stealer Infection Timestamp

- New: **Intel471 Info Stealer Infection Timestamp** - Stores the timestamp when the info stealer infection occurred.

##### Intel471 Info Stealer Machine Id

- New: **Intel471 Info Stealer Machine Id** - Stores the unique machine identifier of the infected host.

##### Intel471 Info Stealer Malware Install Path

- New: **Intel471 Info Stealer Malware Install Path** - Stores the filesystem path where the info stealer malware was installed on the infected machine.

##### Intel471 Info Stealer ISP

- New: **Intel471 Info Stealer ISP** - Stores the internet service provider of the infected machine.

##### Intel471 Info Stealer Malware Family

- New: **Intel471 Info Stealer Malware Family** - Stores the info stealer malware family responsible for the credential theft.

##### Intel471 Info Stealer OS

- New: **Intel471 Info Stealer OS** - Stores the operating system of the infected machine.


#### Integrations

##### Intel 471 Credentials

- New: Added the **Intel 471 Credentials** integration, which fetches leaked credentials from the Intel 471 Credentials API and produces an indicator per credential. While building each indicator, the integration also creates an associated incident.
 <~XSOAR>(Available from Cortex XSOAR 6.8.0).</~XSOAR>

- Added the following commands:
	- ***intel471-credentials-get-indicators***
