
#### Integrations

##### Azure Web Application Firewall
- Updated the Docker image to: *demisto/crypto:1.0.0.6308375*.

- Added support for the **azure-waf-front-door-policies-list-all-in-subscription** command that lists all of the Front Door protection policies within a subscription.
- Added support for the**azure-waf-front-door-policy-update-or-create** command that creates or updates a front door policy with a specified rule set name within a resource group.
- Added support for the **azure-waf-front-door-policies-list** command that lists all of the front door protection policies within a resource group.
- Added support for the **azure-waf-front-door-policy-delete** command that deletes a front door policy.
