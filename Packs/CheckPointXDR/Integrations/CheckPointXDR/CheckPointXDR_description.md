# CheckPointXDR

To set up a CheckPointXDR instance, please provide the Check Point XDR API URL and your credentials.

* Contact [Check Point XDR team](xdr@checkpoint.com) to obtain your Client ID and Client Secret.
* Ensure your credentials are kept secure, as they provide access to your data in Check Point Systems.

This integration allows you to fetch and manage incidents from Check Point XDR, enabling seamless integration with your security operations workflows.
