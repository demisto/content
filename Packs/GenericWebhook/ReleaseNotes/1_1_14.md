
#### Integrations

##### Generic Webhook

- Added support for **Mirroring Direction** parameter that specifies the direction of incident mirroring between XSOAR and the external system.
- Added support for **Mirror Instance** parameter that specifies integration instance name to use for mirroring.
- Added support for **Mirror Tag for Notes** parameter that specifies tag value used to mirror XSOAR notes back to external system.
- Added support for **Duplication Key** parameter that specifies key to use for deduplicating incidents. Multiple values can be provided as a comma-separated list.
