
#### Integrations

##### Elasticsearch v2

- Added **Incident Mirroring** support for Elasticsearch Security Alerts and Cases.
  - New **Incident Mirroring Direction** parameter to control mirroring direction (None, Incoming, Outgoing, Incoming And Outgoing).
  - New **Fetch by Severity** parameter to filter Elasticsearch Cases by severity (critical, high, medium, low).
  - New **Fetch by Status** parameter to filter Elasticsearch Cases by status (open, acknowledged, in-progress, closed).
  - New **Fetch alerts for case** parameter to fetch alerts associated with each Elasticsearch Case.
  - New **Close Mirrored XSOAR Incident** parameter: when enabled, closing an Elasticsearch alert or case is mirrored in Cortex XSOAR.
  - New **Close Mirrored Elasticsearch Incident** parameter: when enabled, closing a Cortex XSOAR incident is mirrored in Elasticsearch.

#### Incident Types

- Added **Elasticsearch Security Alert** incident type for Elasticsearch detection alerts.
- Added **Elasticsearch Case** incident type for Elasticsearch Cases.

#### Incident Fields

- Added **Elasticsearch Workflow Alert Status** field (single select: open, acknowledged, in-progress, closed) for Security Alert incidents.
- Added **Elasticsearch Workflow Alert Status Reason** field (single select: false_positive, duplicate, true_positive, benign_positive, automated_closure, other) for Security Alert incidents.
- Added **Elasticsearch Status Update Date** field for Security Alert incidents.
- Added **Elasticsearch Alert Reason** field for Security Alert incidents.
- Added **Elasticsearch Case ID** field for Case incidents.
- Added **Elasticsearch Case Status** field (single select: open, acknowledged, in-progress, closed) for Case incidents.
- Added **Elasticsearch Case Close Reason** field (single select: false_positive, duplicate, true_positive, benign_positive, automated_closure, other) for Case incidents.

#### Classifiers

- Added **Elasticsearch Security Alert - Classifier** classifier.
- Added **Elasticsearch Case - Classifier** classifier.
- Added **Elasticsearch Security Alert - Incoming Mapper** mapper.
- Added **Elasticsearch Case - Incoming Mapper** mapper.
- Added **Elasticsearch Security Alert - Outgoing Mapper** mapper.
- Added **Elasticsearch Case - Outgoing Mapper** mapper.
