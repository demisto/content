
#### Integrations

##### Elasticsearch v2

- Added the *Space ID* configuration parameter, used by the new Kibana commands.
- Added 33 new commands for interacting with the Kibana REST API:
  - **Alerting / Rules Framework**: ***es-kibana-alerting-health-get***, ***es-kibana-rule-types-list***, ***es-kibana-rule-list***, ***es-kibana-rule-enable***, ***es-kibana-rule-disable***, ***es-kibana-rule-update***, ***es-kibana-rule-alert-mute***, ***es-kibana-rule-alert-unmute***, ***es-kibana-detection-alert-status-set***.
  - **Case Management**: ***es-kibana-case-create***, ***es-kibana-case-update***, ***es-kibana-case-delete***, ***es-kibana-case-list***, ***es-kibana-case-alerts-list***, ***es-kibana-case-comment-add***, ***es-kibana-case-comment-update***, ***es-kibana-case-comment-delete***, ***es-kibana-case-file-attach***.
  - **Security Elastic Endpoint Exceptions**: ***es-kibana-endpoint-exception-list-item-create***, ***es-kibana-endpoint-exception-list-item-update***, ***es-kibana-endpoint-exception-list-item-delete***, ***es-kibana-endpoint-exception-list-item-list***.
  - **Security Exception Lists**: ***es-kibana-exception-list-list***, ***es-kibana-exception-list-create***, ***es-kibana-exception-list-update***, ***es-kibana-exception-list-delete***, ***es-kibana-exception-list-item-list***, ***es-kibana-exception-list-item-create***, ***es-kibana-exception-item-list-update***, ***es-kibana-exception-list-item-delete***.
  - **Value Lists**: ***es-kibana-value-lists-list***, ***es-kibana-value-list-item-get***, ***es-kibana-value-list-item-create***, ***es-kibana-value-list-item-update***, ***es-kibana-value-list-item-delete***, ***es-kibana-value-list-item-export***, ***es-kibana-value-list-item-import***.
