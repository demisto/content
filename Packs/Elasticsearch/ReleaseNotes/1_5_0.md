
#### Integrations

##### Elasticsearch v2

- Added support for **Space ID** parameter that the default kibana space id to use for eskibana* commands. used to derive the kibana base url from the server url. if a space_id argument is provided to a command, it overrides this value. see https://www.elastic.co/docs/deploymanage/managespaces for more information.
- Added support for **es-kibana-rule-update** command that update a rule.
- Added support for **es-kibana-case-comment-delete** command that deletes all comments and alerts from a case.
- Added support for **es-kibana-case-list** command that retrieves the details of kibana cases.
- Added support for **es-kibana-endpoint-exception-list-item-list** command that retrieves elastic endpoint exception list items.
- Added support for **es-kibana-exception-list-delete** command that delete an exception list using the id or list_id field.
- Added support for **es-kibana-value-list-item-delete** command that deletes a value list item.
- Added support for **es-kibana-rule-alert-mute** command that mute an alert, or mute all alerts for a rule.
- Added support for **es-kibana-rule-disable** command that disable a rule.
- Added support for **es-kibana-exception-item-list-update** command that updates an existing exception list item.
- Added support for **es-kibana-case-delete** command that deletes one or more cases by id.
- Added support for **es-kibana-exception-list-create** command that create an exception list.
- Added support for **es-kibana-rule-enable** command that enable a rule.
- Added support for **es-kibana-case-file-attach** command that attach a file to a case.
- Added support for **es-kibana-case-comment-add** command that add a case comment or alert.
- Added support for **es-kibana-value-list-item-get** command that retrieves value list items.
- Added support for **es-kibana-exception-list-list** command that get a list of all exception list containers.
- Added support for **es-kibana-value-list-item-export** command that exports all items of a value list as a file (returned to the war room).
- Added support for **es-kibana-rule-types-list** command that retrieves all rule types available in kibana.
- Added support for **es-kibana-endpoint-exception-list-item-update** command that update an elastic endpoint exception list item.
- Added support for **es-kibana-detection-alert-status-set** command that set the status of one or more detection alerts.
- Added support for **es-kibana-value-lists-list** command that retrieves details of a value list (the list container).
- Added support for **es-kibana-rule-alert-unmute** command that unmute an alert, or unmute all alerts for a rule.
- Added support for **es-kibana-endpoint-exception-list-item-delete** command that delete an elastic endpoint exception list item.
- Added support for **es-kibana-endpoint-exception-list-item-create** command that create an elastic endpoint exception list item, and associate it with the elastic endpoint exception list.
- Added support for **es-kibana-case-alerts-list** command that retrieves all alerts for a case.
- Added support for **es-kibana-exception-list-item-list** command that get a list of all exception list items in the specified list.
- Added support for **es-kibana-alerting-health-get** command that retrieves the health of the kibana alerting framework.
- Added support for **es-kibana-rule-list** command that retrieves information about rules.
- Added support for **es-kibana-value-list-item-update** command that updates an existing value list item.
- Added support for **es-kibana-value-list-item-import** command that import value list items from a txt or csv file.
- Added support for **es-kibana-exception-list-update** command that updates an existing exception list.
- Added support for **es-kibana-case-update** command that update cases in kibana.
- Added support for **es-kibana-value-list-item-create** command that adds a new item to a value list.
- Added support for **es-kibana-exception-list-item-create** command that create an exception item and associate it with the specified exception list.
- Added support for **es-kibana-exception-list-item-delete** command that deletes an exception list item.
- Added support for **es-kibana-case-comment-update** command that update a case comment or alert.
- Added support for **es-kibana-case-create** command that creates a new case in kibana.
