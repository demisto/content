
#### Integrations

##### New: Microsoft Graph Mail Single User
- Added 3 new commands:
   - ***msgraph-mail-list-folders***
   - ***msgraph-mail-create-folder***
   - ***msgraph-mail-move-email***
- Microsoft Graph grants Cortex XSOAR authorized access to a user's Microsoft Outlook mail data in a personal account or organization account. (Available from Cortex XSOAR 6.8.0).
