
#### Integrations

##### Armis Event Collector
- Updated the Docker image to: *demisto/python3:3.12.13.10325753*.

- Improved implementation to reduce memory usage and prevent out-of-memory failures on high-volume tenants.

- Improved implementation to handle Armis API 500/JSON decoder errors and reduce error rates.

- Improved implementation of the alert enrichment mechanism to use bulk requests, allowing all fetched alerts to be enriched reliably and efficiently.

- Updated the integration to always enable concurrent event fetching. The **Enable Concurrent Event Fetching** parameter has been removed.
