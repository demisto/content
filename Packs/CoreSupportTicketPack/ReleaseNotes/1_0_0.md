
#### Scripts

##### FillSupportTicket

- New: Returns fields for the completion of the support ticket.

##### GetSupportTicketTaxonomy

- New: Retrieves the complete support ticket taxonomy including all issue categories and their associated problem concentrations.

##### GetSupportTicketTaxonomyWrapper

- New: Wrapper for GetSupportTicketTaxonomy. Calls the inner script via executeCommand so it is dispatched through the XSOAR server where the exclusion list is honored.

##### SupportTicketCategoryParser

- New: Parses a delimited support ticket classification string into separate fields and validates both values against a provided taxonomy.

##### SupportTicketClassification

- New: LLM-based script that classifies support ticket details to the exact issue category and problem concentration from the provided taxonomy.

##### VerifySupportTicketPermission

- New: Verifies whether the current user has the required permissions to manage support tickets. Checks both user CSP permission and tenant entitlement.

##### VerifySupportTicketPermissionWrapper

- New: Wrapper for VerifySupportTicketPermission. Calls the inner script via executeCommand so it is dispatched through the XSOAR server where the exclusion list is honored.

#### Playbooks

##### Cortex - Fill Support Ticket

- New: Fills a Cortex support ticket by classifying the issue, parsing the taxonomy, and collecting required fields.

#### Agentix Actions

##### CortexFillSupportTicket

- New: Populates a Cortex support ticket template using conversation history.
