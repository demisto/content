## GetSupportTicketTaxonomy

Retrieves the complete support ticket taxonomy including all issue categories and their associated problem concentrations.

### Outputs

| **Path** | **Description** | **Type** |
| --- | --- | --- |
| Core.SupportTicketTaxonomy | A mapping of support issue categories to their specific problem concentrations. Each key represents an issue category, and its associated list represents the valid problem concentrations for that category. | String |
