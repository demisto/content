
#### Integrations

##### Halcyon

- New: **Halcyon** event collector integration.
  - Fetches alerts and events from the Halcyon device management platform.
  - Supports username/password authentication with automatic token refresh.
  - Uses ContentClient for enhanced HTTP request handling with retry logic, circuit breaker, and rate limiting.
  - Configurable log types (Alerts, Events) and fetch limits.
  - Data is stored in the `halcyon_halcyon_raw` dataset.
