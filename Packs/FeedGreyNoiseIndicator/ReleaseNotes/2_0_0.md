
#### Integrations

##### GreyNoise Indicator Feed

- Updated the integration to support GreyNoise v3 APIs and SDK version 3.0.1.
- Added Suspicious option to *Indicator Reputation* parameter to support Suspicious IP Classification.
- Added *Suspicious*, *Malicious + Suspicious* and *Benign + Malicious + Suspicious* options for the GreyNoise Feed Type parameter.
- Added *geolocation* indicator field.
- Updated the Docker image to: *demisto/greynoise:1.0.0.3876899*.
