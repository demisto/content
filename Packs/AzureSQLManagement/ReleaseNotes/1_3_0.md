
#### Azure SQL Management

##### New: Azure SQL Management

- Added support for the ***azure-sql-firewall-rule-list*** command to list firewall rules for a server.
- Added support for the ***azure-sql-firewall-rule-create-update*** command to create or update a firewall rule.
- Added support for the ***azure-sql-firewall-rule-delete*** command to delete a firewall rule.
- Added support for the ***azure-sql-firewall-rule-replace*** command to replace all firewall rules on a server.
