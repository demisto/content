
#### Scripts

##### MicrosoftApiModule

- Added UCP (Unified Connection Platform) authentication support. When UCP auth is enabled, the module skips traditional oproxy/self-deployed credential setup and defers token injection to the platform.

##### GSuiteApiModule

- Added UCP (Unified Connection Platform) authentication support. When UCP auth is enabled, the module fetches OAuth2 access tokens from the UCP service, supporting user impersonation via domain-wide delegation.
