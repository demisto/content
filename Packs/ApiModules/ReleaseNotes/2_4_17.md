#### Scripts

##### COOCApiModule

- Fixed an issue in the `is_gov_account` function where the `cloud_partition` check block was missing, causing it to return `None` instead of `True`/`False`.
- Added `get_timeout` utility function that parses the `timeout` integration parameter into `(read_timeout, connect_timeout)` seconds. Accepts either `"<read>"` (connect defaults to 10 s) or `"<read>,<connect>"`. Intended for use by COOC integrations that support the XSOAR / XSIAM marketplace path.
