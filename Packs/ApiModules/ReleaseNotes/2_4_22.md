#### Scripts

##### COOCApiModule

- Fixed a bug in `is_gov_account` where the `cloud_partition` check block was missing, causing the function to return `None` instead of `True`/`False`.
- Added the `get_timeout` utility function that parses the `timeout` integration parameter into `(read_timeout, connect_timeout)` seconds. Accepts either `"<read>"` (connect defaults to 10 s) or `"<read>,<connect>"`. Intended for use by COOC integrations that support the **Cortex XSOAR** and **Cortex XSIAM** marketplace paths.
