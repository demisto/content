### Basic Information
Supports AppSec 3.0.0 and above.

When using Cortex XSOAR credentials, insert your API key as the credential password. Otherwise, insert your API key into the relevant required parameters.
### Base URL
Fill the {region-code} by your region.
- United States: use region-code=us or us2 or us3.
- Europe: use region-code=eu.
- Canada: use region-code=ca.
- Australia: use region-code=au.
- Japan: use region-code=ap.

### Get a Rapid7 AppSec API Key
1. Create an account in Rapid7.
2. Click **Settings** > **API Keys**.
3. Generate your API key.
